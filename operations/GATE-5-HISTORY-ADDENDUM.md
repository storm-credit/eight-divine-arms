# 《팔대신병록》 Gate 5 역사 지명 교정 부록

상태: `resolved-provisional / superseding-terminology`
결정자: PM 오케스트레이터
근거: 태주 지방지·해방사 재검증

이 문서는 지명에 한해 `design/SETTING-BIBLE-v1.0.md`와 이전 Gate 문서의 작업명을 우선 교정한다.

## 1. 발견

태주 연해에는 명 홍무 20년인 1387년에 설치된 실제 **해문위(海門衛)**가 존재했다.

1555년 현재에도 해문은 태주 연해 방어와 군사·항구 맥락을 가진 실제 지명이었다.

따라서 허구의 `해문도`를 같은 태주 해역에 둘 경우:

- 실제 해문위와 허구 섬이 혼동되고
- 1368년 원사건이 후대 해문위와 직접 연결된 실제 사건처럼 보이며
- 최종부의 군사 관할이 불필요하게 복잡해진다.

## 2. 교정

### 원사건 장소

- 기존 작업명: 해문도(海門島)
- 새 작업명: **쇄조도(鎖潮島)**

### 원사건

- 기존: 해문도 이관 참변
- 새: **쇄조도 이관 참변**

### 최종 결말·합증

- 기존 작업명: 해문합증
- 새 작업명: **쇄조합증(鎖潮合證)**

### 명부·밀약

- 해문 밀약 → 쇄조 밀약
- 해문 생존자·후손 → 쇄조도 생존자·후손
- 해문합증록 → 쇄조합증록

## 3. 이름의 작품 내 의미

`쇄조도`는 조수를 막는 좁은 수로·암초·임시 방책 때문에 붙은 지방 속칭이다.

- 공식 대지도에는 다른 행정명 또는 무명 외도로 기록될 수 있음
- 선원·조선공·밀무자·지역민이 쇄조도라고 부름
- 잠긴 수문·봉쇄·늦은 구조라는 원사건의 물리적 특징과 연결

실제 지명처럼 사료에 존재했다고 주장하지 않는다.

## 4. 실제 해문위의 사용

실제 해문위는 배경에서 별도로 사용할 수 있다.

- 관군·군기·해방 문서
- 해상 검문·수군·역포
- 금의위 차견과 지방 군무의 관할 충돌

그러나:

- 쇄조도 원사건의 비밀 주체로 자동 연결하지 않음
- 실제 해문위 장병 전체를 은폐 공범으로 만들지 않음
- 최종 쇄조합증 장소와 해문위 사이에는 물리적 거리를 둠

## 5. 문서 적용 규칙

이전 문서의 다음 표현은 모두 레거시 작업명이다.

- 해문도
- 해문도 참변
- 해문 밀약
- 해문합증
- 해문합증록

새 장별 개요와 본문에서는 쇄조도·쇄조 밀약·쇄조합증을 사용한다.

기존 문서는 결정 과정 보존을 위해 즉시 모두 덮어쓰지 않으며, 이 부록이 명칭에 한해 우선한다.

## 6. 영향

변경되는 것:

- 고유 지명과 결말 작업명

변경되지 않는 것:

- 1368년 원사건의 인과
- 최초 여덟
- 팔대신병
- 주인공과 인물 관계
- 6부 플롯
- 분산 합증 결말 구조
- 전체 기간과 권역
