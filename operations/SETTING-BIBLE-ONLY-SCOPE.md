# 《팔대신병록》 작업 범위 교정 — 설정집 우선

상태: `superseding-decision / active`
결정자: 사용자
총괄: PM 오케스트레이터

## 1. 사용자 의도

현재 단계의 목적은 **소설 원고·장면·장별 전개를 쓰는 것이 아니라, 이후 어떤 이야기든 안정적으로 올릴 수 있을 만큼 설정집을 완성하는 것**이다.

따라서:

- 세계의 법칙·제도·세력·생활·무공·기환·병기·역사·지역·용어·장부는 지금 세밀하게 설계한다.
- 개별 장면의 대사·연출·감정 호흡·전투 진행·문장 표현은 나중에 붙인다.
- 특정 플롯에만 필요한 세부 인물·사건·장면은 설정집 정본을 흔들지 않는 참고안으로만 둔다.

## 2. 즉시 중단·잠금

다음 작업을 중단한다.

- 장편 초고
- 장면 카드 추가 상세화
- 60장 개요의 장면 단위 확장
- 대사·문체·연출 작성
- 장별 전투 동작 세부 설계

`manuscript/CHAPTER-OUTLINE-v0.1.md`와 `manuscript/SCENE-CARDS/`는 **premature-detail / deferred-reference**로 강등한다.

- 삭제하지 않고 나중의 원고 설계 참고물로 보존한다.
- 설정집의 정본 근거로 사용하지 않는다.
- 현재 PM 진행률 계산에서 원고 성과로 간주하지 않는다.

실제 초고 작업은 삭제·중단한다.

## 3. 현재 최우선 산출물

1. 설정집 완성도 재감사
2. D01~D27 분야별 `완성 / 보강 필요 / 뼈대만 / 의도적 미상` 판정
3. 보강 필요 분야의 세부 설정 모듈 작성
4. 세력·지역·제도·무공·기환·생활·경제의 교차 장부
5. 용어 사전·관계도·권리표·상한표
6. 설정집 v2.0 통합
7. 전문가 합동 최종 누락 감사
8. 사용자 Human Edit 뒤 `canon` 승격

## 4. 설정집 완성 기준

설정 분야마다 다음이 있어야 한다.

- 정의와 범위
- 세계 안에서 실제 작동하는 절차
- 누가 이익·손해를 보는지
- 지역·신분·세력에 따른 차이
- 예외와 실패 조건
- 다른 설정과의 연결
- 본편에서 사용하지 않아도 세계가 굴러가는 최소 현실
- 의도적으로 정하지 않은 영역과 그 이유

## 5. 상세화와 살 붙이기의 경계

### 지금 하는 상세화

- 표국의 수입·계약·배상·신원 보증 방식
- 문파의 재정·입문·파문·계승·장례
- 병장기별 전술 역할과 수리·운송·감정
- 무공 수련 시간·부상·경지 평가 언어
- 영초의 분류·채집·가공·거래 상한
- 강시·시귀 판별 절차
- 명성록·대회·후기지수 제도의 운영자와 조작 가능성
- 여성·하층민·장인·뱃사람·군호의 생활 차이
- 지역별 기후·교통·생업·강호 문화

### 나중에 붙이는 살

- 특정 장면의 대사
- 주인공이 어느 골목에서 누구와 싸우는지
- 각 장의 클리프행어
- 초식명·별호·개별 영초 이름의 대량 목록
- 실제 대회 우승자와 고수 순위
- 사건별 단역 명단
- 장면 묘사와 문체

## 6. 자동 진행 규칙

- PM은 설정집 분야를 순서대로 자동 진행한다.
- 새 설정이 필요하면 기존 D01~D27에 편입하고 교차 감사를 수행한다.
- 상위 작품 헌법을 바꾸는 결정만 사용자에게 요청한다.
- 원고·장면 작업으로 넘어가지 않는다.
- 각 묶음 종료 때 현재 위치·완료 항목·추가된 설정·남은 분야를 보고한다.
