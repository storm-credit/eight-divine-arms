## 1. 목적

5막·8구간 플롯을 **집필 가능한 장편 구조**로 펼친다.

Outline은 사건의 순서·목표·전환·결산을 설계하지만 소설 문장과 장면별 대사는 쓰지 않는다.

## 2. 포함 범위

1. 장 수 후보 비교
2. 막·구간별 장 배분
3. 각 구간의 시작 상태·목표·압력·비가역 변화
4. 핵심 인물 주도권과 관계 변화
5. 세력·신병·증거의 진입·퇴장
6. 이동·계절·체류·회복 시간
7. 부상·치료·수배·신용·권리 상태
8. 단서 공개·오인 가설·재해석
9. 각 장의 기능 단위 개요
10. Outline 전체 교차감사

## 3. 제외 범위

- 소설 문장
- 대사
- 장면별 동작과 감정 연출
- 전투 합의 세부 수순
- 모든 객잔·시장·음식·의복 설정
- 등장하지 않는 문파·세력의 도감
- 모든 초식·영초·부적 이름
- 기존 60장안과 장면 카드 자동 복원

## 4. 국소 상세화 허용

다음에 직접 영향을 주는 경우만 새 설정을 만든다.

- 장의 목표·전환
- 단서와 오인
- 인물의 선택
- 전투·호송·치료·협상의 결과
- 이동·가격·권리의 가능성
- 반복 등장 장소·세력

## 5. Outline 카드 수준

각 장은 다음까지만 기록한다.

- 장 번호·작업 제목
- 시점 인물
- 장소·시간 범위
- 장의 목표
- 압력·충돌
- 공개 정보·숨은 정보
- 인물 선택
- 상태 변화
- 다음 장 연결

대사·묘사·세부 장면 분할은 금지한다.

## 6. 비교할 장편 구조

### O-36 압축형

- 총 34~38장
- 구간당 4~5장
- 빠른 미스터리와 사건 중심

### O-48 균형형

- 총 44~52장
- 구간당 5~7장
- 인물·지역·증거·회복 시간을 균형 있게 확보

### O-60 확장형

- 총 56~64장
- 구간당 7~9장
- 권역과 조연을 넓게 활용

## 7. 완료 조건

- 5막·8구간이 각각 독립 목표와 비가역 변화를 가짐
- 모든 핵심 인물이 최소 한 구간을 주도
- 여덟 신병이 수집품처럼 순차 등장하지 않음
- 중반 정보 구간에도 행동 압력이 있음
- 이동·부상·치료 시간이 모순 없음
- 결말이 회의 한 장면에 집중되지 않음
- 장 수가 설정 설명 때문에 부풀지 않음
- 원고 없이도 작가가 각 장의 기능을 이해할 수 있음
