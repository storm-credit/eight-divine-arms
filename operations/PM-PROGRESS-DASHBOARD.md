# 4. Gate 4 고정 원칙

- 여덟 신병을 하나씩 획득하는 수집 퀘스트로 만들지 않는다.
- 중심 신병은 4~5개만 직접 다루고 나머지는 증거·권리·외교로 연결한다.
- 사문경은 전지적 흑막이 아니며, 여러 세력의 이해를 이용하는 전략가다.
- 현재 범죄와 원사건은 닮지만 같은 사건의 복제는 아니다.
- 결말은 모든 기록의 무제한 공개·완전 은폐·전면 파괴 중 하나로 단순화하지 않는다.
- 주인공의 성공은 최강자가 되는 것이 아니라 책임 있는 인계·증언 구조를 만드는 것이다.
- 여운을 위해 대량 사망과 무조건적 비극에 의존하지 않는다.
- 주요 인물은 생존 여부보다 자신이 믿던 책임 기준을 수정하거나 끝까지 거부하는 선택으로 결산한다.

# 6. 새 아이디어 처리

기존 패킷의 하위 질문은 현재 작업에 편입한다.

개별 별호·초식·대회 회차·영초 도감·미디어 시안은 백로그에 보존하고 Gate 순서를 바꾸지 않는다.

시대·주 무대·팔대신병 정의·기환 강도를 바꾸는 항목만 관련 Gate를 재개방한다.
