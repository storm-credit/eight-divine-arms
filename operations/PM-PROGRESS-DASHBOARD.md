# 4. 설계 핵심

## 시대·원사건

- 현재: 1555~1556년 명 가정 연간
- 시작: 절강 태주부 연해 청계선창
- 원사건: 1368년 늦봄 쇄조도 이관 참변

## 주인공

- 심소연, 진원표국 검표수·인계표사
- 혈통 선택자·천재 고수가 아님
- D3 철조구 제한 적합
- 과거 기록 조작을 공개하고 징계·배상을 수용

## 결말

- 쇄조합증
- 사건 골격·조직 책임 공개
- 현재 신원·환자·피난처 제한
- 신병·기록 분산 보관
- 새 중앙 초법기관 없음

## 전체 구조

1. 청계의 불 — 8장
2. 절류의 항로 — 9장
3. 금릉의 저울 — 10장
4. 봉서의 재심 — 10장
5. 청맥의 길 — 10장
6. 쇄조합증 — 13장

총 60장 작업 구조이며 초고 리듬 감사에서 54~66장 범위 조정 가능.

# 6. 자동 진행 통제

- 새 상위 설정은 추가하지 않는다.
- 초고에 필요한 지역 인물·소품·문서 형식만 장면 기능에 따라 추가한다.
- 시대·원사건·신병 정의·주인공·결말 변경은 관련 Gate를 재개방한다.
- 최신 사용자 결정은 과거 감사안보다 우선한다.
- 초고는 부별로 작성하고 각 부 뒤 자체 감사를 거친다.
