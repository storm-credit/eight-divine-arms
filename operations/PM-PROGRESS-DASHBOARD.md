## 1. 우리가 만드는 것

**장편소설 《팔대신병록》의 완성형 작품 설계 패키지.**

설정 백과사전도, 현재 원고도 아니다.

필수 산출물:

1. 설정·세계 시스템
2. 인물·관계
3. 결말·전체 플롯
4. Outline
5. 연속성·누락·과잉 최종 감사
6. 집필 준비 판정

## 2. 현재 위치

| 영역 | 상태 | 진행 판단 |
|---|---|---:|
| 작품 헌법·세계·시스템 | `canon` | 100% |
| 인물·관계 | `canon` | 100% |
| 결말·전체 플롯 | 설계·감사 완료, 작업 기준 존재 | 85~90% |
| Outline | 다음 본선 | 0~5% |
| 최종 설계 감사 | 대기 | 0% |
| 장면·원고 | 잠금 | 0% |

**작품 설계 패키지 전체 진행률: 약 65~72%.**

원고까지 포함한 소설 제작률이 아니라, 집필 전에 필요한 설계 패키지 기준이다.

## 3. 완료된 것

### 설정

- 명대 역사·권역·강호·국가·상업 작동 원리
- 무공·병장기·비주류 무기·의술·독·기환 상한
- 세력·조직·기록·명성·가격·권리·보관 규칙
- 팔대신병 여덟 개별 핵심
- 쇄조도 원사건의 세계사적 골격

### 인물

- 심소연 중심 주인공 구조
- 상시 핵심 5인 구조
- 반복 실무자와 적대 압력 삼각 구조
- 과오·전문성·무지·가족·관계 원칙

### 플롯

- 결말 후보 비교
- 5막·8구간 권고 구조
- 조선소 붕괴·이중 인계 첫 사건
- 미스터리·증거 층위
- 적대 압력·책임 분리
- 상태·연속성 장부
- 전문가 교차감사

## 4. 지금 하지 않는 것

- 구파일방·오대세가 전원 백과사전
- 모든 문파 무공·초식 목록
- 도시·객잔·음식·의복 도감
- 영초·영수·부적·보패 대량 도감
- 역대 고수·대회 우승자 전체 명단
- 소설 문장·대사·장면 카드

필요한 항목은 Outline에서 실제 사용될 때만 국소 상세화한다.

## 5. 남은 본선 작업

### A. 플롯 작업 기준 마감

- 5막·8구간을 Outline 작업 기준으로 고정
- 피해자·보호 대상·로맨스의 공개 범위 유지
- 기존 6부·60장안 자동 복원 금지

### B. Outline Gate

1. 각 막·구간의 목표
2. 시작 상태·압력·전환·결산
3. 핵심 인물 주도권
4. 세력·신병·증거 배치
5. 권역·계절·이동
6. 부상·치료·자원·수배·권리
7. 장 수·시점 후보
8. 단서·오인 가설·반전 순서

### C. 최종 설계 감사

- 설정→인물→플롯→Outline 인과
- 중복·누락·과밀
- 역사·거리·가격·부상·권리
- 인물 주도권과 감정 결산
- 집필 가능성 판정

## 6. 백과사전형 상세화 처리

`S8~S12 상세 설정집`은 본선 필수 단계에서 철회한다.

- 유용한 아이디어는 백로그로 보존
- 실제 Outline에서 필요한 세력·장소·무공·기환만 상세화
- 설정 핵심 정본은 유지

## 7. 현재 다음 작업

**Outline Gate 범위 동결과 구조 후보 비교.**

원고는 Outline과 최종 설계 감사 전까지 잠근다.
