# 3. 설계 결과 요약

## 시대·원사건

- 현재: 1555~1556년 명 가정 연간
- 시작: 절강 태주부 연해 청계선창
- 원사건: 1368년 늦봄 **쇄조도 이관 참변**
- `해문도`는 실제 태주 해문위와 혼동되어 레거시 작업명으로 폐기

## 팔대신병

- 절대병기 조각이나 부활 장치가 아님
- 여덟 핵심 당사자의 불완전한 증언·책임 병기
- 잔흔은 제한적 감각·동작·감정 파편
- 집합은 힘 증폭이 아니라 합증·계승·책임 재심

## 주인공

- 심소연, 진원행 검화수·인계호송인
- 혈통 선택자·천재 고수 아님
- D3 철조구 제한 적합
- 과거 기록 조작과 책임 전가를 결말에서 공개·배상

## 결말

- **쇄조합증**
- 사건 골격과 조직 책임 공개
- 현재 신원·환자·피난처 제한
- 신병과 기록 분산 보관
- 새 중앙 초법기관 없음

## 전체 구조

1. 청계의 불
2. 절류의 항로
3. 금릉의 저울
4. 봉서의 재심
5. 청맥의 길
6. 쇄조합증

작업 장 수:

- 총 60장
- 리듬 감사에서 54~66장 범위 조정 가능

# 4. 역사·용어 교정

즉시 적용:

- 진원표국 → 진원행
- 검표수 → 검화수
- 표사 → 호송인·압운인
- 표두 → 호송두·행관
- 남방 회수별조 → 남방 회수차견·금의위 차견 일행
- 해문도 → 쇄조도
- 해문합증 → 쇄조합증

이전 문서는 결정 과정·검색 호환용 레거시 작업명을 유지할 수 있으나 새 원고 설계는 교정 용어를 쓴다.

## O2 실제 지리·계절 검증

- 태주·금릉·중원 경계·서남 경로
- 수운·항해·우기·겨울 이동
- 금의위 차견 직급·문서

를 순차 검증한다.

- 먼저 제1부 8장부터 묶음 검증

## O4 장편 초고

# 6. 자동 진행 통제

현재부터 새 상위 설정을 추가하지 않는다.

- 장별 개요에 필요한 세부 인물·지명만 편입
- 나중에 쓸 세부 설정은 백로그
- 시대·원사건·신병 정의·주인공·결말 변경은 영향 분석 후 Gate 재개방

정식 설정집 `canon` 승격과 장면 카드 대량 작성은 Human Edit 체크포인트 뒤 진행한다.
