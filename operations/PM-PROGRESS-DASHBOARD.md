## 1. 현재 목표

**설정집 완성.** 원고·장면·대사·연출은 잠금 상태다.

- 기존 시놉시스·60장 개요·장면 카드: `premature-detail / deferred-reference`
- 실제 초고: 중단·삭제
- 현재 작업 폴더: `design/`, `audit/`

## 2. 진행 현황

| 묶음 | 상태 | 핵심 산출물 |
|---|---|---|
| 상위 설정 골격 D01~D27 | 완료 | 범위 지도·최초 누락 감사 |
| 설정집 완성도 재감사 | 완료 | 실제 완성도 55~65%로 재산정 |
| S1 세계 운영 현실 | 완료 | 시간·국가법·경제·교통정보·생활·범죄구휼 |
| S2 강호 제도·조직 운영 | 임시 통과 | 조직·입문계승·무림맹·표국상업·지하조직 |
| S3 무공·병장기·의술 | 임시 통과 | 수련·경지·병장기·갑주화기·의술검시 |
| S4 기환·종교·생태 | 진행 중 | 부적·법구·보패부터 시작 |
| S5 권역·외부 무림 | 대기 | 지역별 기후·생업·문화·세력 배치 |
| S6 기록·명성·용어·장부 | 대기 | 서열록·대회·교육·용어·가격·권리 장부 |
| S7 팔대신병 개별 설정집 | 대기 | 여덟 개별 카드·보관·감정·수리 |

## 3. 진행률

- 상위 설정 골격: 약 90%
- 설정집 실제 완성도: **약 76~80%**
- 원고: 현재 범위에서 제외

S1~S3 세부 모듈과 교차감사를 기준으로 재산정했다.

## 4. 완료된 S1

- 역법·시간·연령·연표
- 국가·행정·법·군사
- 경제·화폐·계약·배상
- 교통·통신·정보 유통
- 신분·가족·성별·일상·구휼
- 범죄·암시장·지하질서·피해 회복

## 5. 완료된 S2

- 강호 조직 공통 운영
- 입문·교육·계승·징계·은원
- 무림맹·구파일방·오대세가 의석·재심
- 진원표국·상단·전장·객잔 운영
- 개방·하오문·녹림·수로채·살수 운영
- S2 교차감사 통과

## 6. 완료된 S3

- 무공 수련 생애주기
- 경지 언어·숙련·실전 프로필
- 병장기별 무공·비주류 무기·상성
- 갑주·기관·암기·화기·군진
- 의술·독·검시·약재·재활
- S3 교차감사 통과

D09·D10·D11은 설정집 기준 `A-`로 승격했다.

## 7. 현재 S4 작업 큐

1. 부적·법구·보패의 정의·제작·사용·실패
2. 진법·풍수·점복의 실제·오인·사기
3. 영물·영수·영초·사방신의 발생·생태·거래
4. 죽음·장례·강령·강시·시귀 판별
5. 불문·도문·민간신앙·사교 판정
6. 기환 판별·사기·오인 교차감사

## 8. 최신 사용자 결정

정식 작업 용어:

- 진원표국
- 검표수
- 인계표사
- 표사
- 표두
- 금의위 남방 회수별조

지명 교정:

- 쇄조도
- 쇄조합증

## 9. 자동 진행 규칙

- 원고로 넘어가지 않는다.
- 새 항목은 D01~D27에 편입한다.
- 분야마다 정의·절차·자원·지역차·예외·실패·교차 연결을 작성한다.
- 각 S 묶음 뒤 전문가 교차감사를 거친다.
- 설정집 v2.0과 최종 누락 감사 뒤 사용자 승인 지점을 연다.
