---

# 1. 범위 교정

현재 목표는 **설정집 완성**이다.

- 장편 초고: 중단
- 장면 카드 추가 상세화: 잠금
- 60장 개요·기존 장면 카드: `premature-detail / deferred-reference`
- 대사·문체·장면 연출: 후속 단계

방금 시작한 실제 초고 작업은 삭제했다.

---

# 2. 전체 단계 현황

| 구간 | 상태 | PM 판정 |
|---|---|---|
| 운영 하네스·상위 분야 발견 | 완료 | D01~D27 확보 |
| 세계관 상위 골격 | 완료 | provisional-canon |
| 팔대신병·원사건·핵심 인물 후보 | 완료 | 설정집 통합 시 재감사 |
| 설정집 v1.0 | 상위 골격 완료 | 세부 완성도 재감사 필요 |
| 설정집 완성도 재감사 | 진행 중 | 분야별 빈칸 재분류 |
| 세부 설정 모듈 | 진행 예정 | 제도·지역·세력·생활·무공·기환 |
| 설정집 v2.0 통합 | 대기 | 세부 모듈 완료 후 |
| 최종 전문가 누락 감사 | 대기 | v2.0 기준 |
| 정식 원고 | 잠금 | 설정집 승인 뒤 |

## 진행률 구분

- **상위 설정 골격:** 약 90%
- **설정집 실제 완성도:** 약 55~65%
- **원고:** 현재 범위에서 제외

과거의 `설정 설계 100%` 판정은 상위 구조와 플롯 의존 설정을 기준으로 한 과대평가였다. 생활·지역·제도·세력 운영·무공 수련·기환 하위 규칙 등 설정집 자체의 세부 완성도를 다시 계산한다.

# 5. 최신 사용자 결정

정식 작업 용어:

- 진원표국
- 검표수
- 인계표사
- 표사
- 표두
- 금의위 남방 회수별조

폐기된 검토안은 사용하지 않는다.

지명 교정 `쇄조도·쇄조합증`은 유지한다.

---

# 6. 자동 진행 규칙

- 새 상위 설정을 무제한 추가하지 않는다.
- 필요한 새 항목은 D01~D27에 편입한다.
- 분야마다 정의·절차·자원·지역차·예외·실패·교차 연결을 작성한다.
- 원고·장면·대사로 넘어가지 않는다.
- 각 S 묶음마다 전문가 독립 검토와 PM 통합을 수행한다.
- 설정집 v2.0과 최종 누락 감사가 끝난 뒤에만 원고 단계 재개 여부를 판단한다.
