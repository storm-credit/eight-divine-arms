# WP-304 — 현재 팔대신병 여덟 지위

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-301~303, WP-207~209 완료
- 종료 조건: 현재 여덟 신병이 형태·기억 채널·실용 기능·기환 효과·조건·대가·사각지대·원본/계승 상태에서 구별될 것

## 2. 목적

팔대신병을 여덟 속성 무기나 강한 순서의 보물로 만들지 않고, 최초 당사자의 삶과 해문도 행동에서 나온 서로 다른 증언 병기로 개별화한다.

## 3. 범위

- 신병 코드와 작업명
- 최초·현재 형태
- 원본·수리 원본·계승품·분쟁 상태
- 강한 잔흔 채널 최대 두 개
- 실전 기능과 기환 보조
- 촉발 조건과 사각지대
- 사용 대가와 침식
- 다른 신병과의 강한 교차 반응
- 후대 전설과 오해

## 4. 구조 후보

### A. 여덟 속성 무기형

화·수·풍·지 등 속성 능력 배정.

### B. 여덟 전투 역할형

공격·방어·치유·봉인 등 기능 슬롯 배정.

### C. 최초 당사자의 병기 보존형

최초 형태와 기억을 거의 그대로 유지.

### D. 생업 병기 + 후대 역사 변형형

최초 당사자의 직업·환경·선택에서 형태가 나오고, 수리·계승·오해로 현재 기능과 외형이 변형됨.
