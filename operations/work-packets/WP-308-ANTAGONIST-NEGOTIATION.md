# WP-308 — 적대자·반대자·협상 구조

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-306~307 완료
- 종료 조건: 주요 적대 층·욕망·수단·도덕 논리와 주인공을 즉시 제거하지 못하는 이유가 확정될 것

## 2. 목적

모든 사건의 배후인 절대 악역이나 고대 마두를 피하고, 진실·질서·생존·권리를 서로 다르게 우선하는 현재 인물들이 주인공을 이용·보호·협박·협상하도록 설계한다.

## 3. 범위

- 전략적 주적
- 국가·세력의 제도적 반대자
- 폭력적 반대자와 일시 동맹 가능성
- 공개 목표·실제 목표·개인 상처
- 주인공과 핵심 인물의 약점 이용
- 살해보다 생포·고용·증언·서명이 필요한 이유
- 단계별 목표 변화
- 패배·타협·전환 가능성

## 4. 구조 후보

### A. 혈교 최종 흑막형

### B. 황실 비밀기관 단일 악역형

### C. 최초 여덟 후손의 복수형

### D. 전략적 주적 + 제도 압력 + 폭력적 반대자형
