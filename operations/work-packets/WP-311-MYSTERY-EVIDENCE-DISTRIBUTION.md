# WP-311 — 미스터리 층위·증거 배분

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-301~310 완료
- 종료 조건: 현재 사건·원사건·신병·인물·적대자에 관한 지식과 오해가 인물별로 분배되고 Gate 4가 공개 순서를 설계할 수 있을 것

## 2. 목적

주인공과 독자가 같은 속도로 모든 정보를 얻거나, 적대자만 완전한 진실을 알고 있는 구조를 피한다. 각 인물은 자기 전문 영역에서 사실을 알지만 다른 영역에서는 편향과 오해를 가진다.

## 3. 범위

- 핵심 미스터리 질문
- 현재 사건과 원사건의 구분
- 인물·세력별 초기 지식
- 사실·유력·오해·거짓말
- 핵심 증거 계열의 보유자
- 가짜 신병·오보·사설 전설의 역할
- 부별 공개 이전의 논리적 선행 관계
- 각 단계의 최소 결산
