# WP-213 — 기환 하위 현상 경계

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-203~205 완료
- 종료 조건: 부적·법구·보패·진법·풍수·점복·영물·영수·영초·신수·사방신·강령·강시·시귀가 공통 상한법 아래 배치될 것

## 2. 목적

사용자가 제시한 기환 요소를 누락하지 않되 각각 독립 마법 체계로 팽창하지 않도록 용어·희귀도·효과·금지선을 정한다.

## 3. 구조 후보

### A. 독립 술법 학파형

각 요소가 별도 마법 계통.

### B. 전부 오인·사기형

실제 기환 없이 의학·기관·심리로만 설명.

### C. 제한적 개별 실재형

일부는 실제지만 서로 다른 원리를 가짐.

### D. 공통법 하위 유형형

WP-204의 흔적·매개·공명 규칙 아래 기능과 매개체가 다른 하위 현상으로 배치.
