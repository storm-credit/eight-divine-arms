# WP-305 — 현재 소유·보관·계승 지도

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-304 완료
- 종료 조건: 본편 시작 시점의 여덟 신병과 핵심 증거의 위치·공개 상태·권리 주장·이동 가능성이 확정될 것

## 2. 목적

팔대신병을 이미 한 세력에 모아 두거나 모두 완전 실종시키지 않고, 지역·세력·권리·정보 수준이 다른 분산 상태를 설계한다.

## 3. 범위

- 물리 위치와 보관자
- 공개적으로 알려진 지위와 실제 상태
- 소유·보관·사용·계승·감정·공개 주장
- 관련 증거의 별도 보관 경로
- 이동·호송·압수·도난 가능성
- 1555년 첫 사건과 연결 가능한 불안정 지점
- 세력별 직접 추적 상한

## 4. 구조 후보

### A. 전부 실종형

여덟 모두 행방불명이고 주인공이 하나씩 발견.

### B. 대세력 분할 보유형

여덟 대세력이 하나씩 안정 보유.

### C. 공개·비공개 혼합형

알려진 보유품·숨은 보관품·오인된 유물·이중 계승품이 섞임.

### D. 이동 중인 여덟 화물형

현재 모두 거래·호송·압수 과정에 들어가 사건이 시작.
