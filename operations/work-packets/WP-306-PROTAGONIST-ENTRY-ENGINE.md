# WP-306 — 주인공 진입 엔진

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-301~305 완료
- 종료 조건: 주인공이 혈통 선택 없이 사건 중심에 들어가고, 여러 세력이 죽이지 않고 필요로 하는 이유와 장기적 욕망·오해가 확정될 것

## 2. 목적

주인공을 팔대신병이 선택한 유일한 계승자가 아니라, 직업·과거 선택·신체 경험·증거 책임 때문에 사건을 피할 수 없는 인물로 설계한다.

## 3. 범위

- 출신·신분·직업
- 무공과 실무 능력
- 부상·약점
- 욕망·두려움·잘못 믿는 것
- 숨긴 과오
- D3 첫 접촉 이유
- 여러 세력이 살려둘 이유
- 장기 성장과 주제 연결
- 모든 신병에 대한 적합성 제한
