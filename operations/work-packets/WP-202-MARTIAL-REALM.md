# WP-202 — 무공 존재론·경지 언어·전투 상한

상태: `research-active`
게이트: Gate 2
총괄: PM 오케스트레이터

## 1. 작업 교정

사용자가 예로 든 `절정·화경·현경·삼화취정`을 그대로 경지표로 채택하지 않는다.

먼저 다음 네 계통을 분리 조사한다.

1. 고전 한문에서 기술의 숙련을 표현한 말
2. 도가 내단·양생의 실제 수련 용어
3. 근현대 중국 무술론의 단계어
4. 한국·중국 현대 무협 장르에서 창작된 레벨어

출처와 시대가 다른 용어를 한 줄에 임의로 이어 붙이지 않는다.

## 2. 1차 조사 결과

### A. 사회적 실력 평가어

후보군:

- 하수·숙수·고수
- 삼류·이류·일류
- 정상급·절정고수
- 명가·종사·대종사

성격:

- 전적·평판·공적·증언으로 매겨지는 사회적 평가
- 객관적 신체 변화나 초자연 단계가 아님
- 지역·정보조직·세력에 따라 평가가 달라질 수 있음

주의:

- `종사`는 원래 특정 분야의 큰 스승과 모범이 되는 인물을 뜻하는 존칭에 가깝다.
- `절정`은 최고봉·탁월함을 뜻하는 일반 평가어이므로 생리적 경지명으로 자동 사용하지 않는다.

### B. 학예·기술의 숙련 표현

후보군:

- 미득기문: 문도 찾지 못함
- 초규문경: 대강의 길을 엿봄
- 등당: 학예가 본격 궤도에 오름
- 입실: 사문의 깊은 뜻을 체득
- 노화순청: 기술이 정련되어 군더더기가 사라짐
- 출신입화·등봉조극: 극도로 뛰어난 조예를 칭찬하는 표현
- 반박귀진: 꾸밈이 사라지고 본질로 돌아간 상태

성격:

- 무공 전 분야의 절대 출력 등급이 아니라 특정 기예의 숙련도
- 검술은 입실했지만 내공은 미숙한 인물처럼 비대칭 설계 가능

### C. 도가 내단·양생 용어

후보군:

- 연정화기
- 연기화신
- 연신환허
- 삼화취정
- 오기조원
- 주천·하거·황정·현관 등

성격:

- 호흡·정기신·오장·심신 통합을 설명하는 종문 내부 수련 언어
- 강호 전체가 공유하는 전투 레벨표가 아님
- 도문 무공이나 장생·정신 수련에서만 제한적으로 사용

주의:

- `삼화취정`과 `오기조원`은 도가 내단 용어이므로 `화경 다음 단계` 같은 식으로 배열하지 않는다.
- 작품의 저강도 기환 원칙상 불사·비승·금선 같은 결과는 채택하지 않는다.

### D. 근현대 중국 무술론 용어

후보군:

- 명경·암경·화경 또는 명劲·암劲·화劲 계열
- 형태·발력·의념과 힘의 운용을 설명하는 단계

성격:

- 특정 근현대 내가권 이론과 밀접
- 명나라 가정 연간의 전국 공통 용어로 사용하면 시대착오 위험

처리:

- 작품 내 공용 경지명에서는 우선 제외
- 후대 편집자의 주석이나 특정 계보의 독자 용어로도 역사 검증 전 사용 금지

### E. 현대 한국식 판타지 무협 레벨어

자주 보이는 후보군:

- 삼류 → 이류 → 일류 → 절정 → 초절정 → 화경 → 현경 → 생사경
- 검기 → 검사 → 검강 → 검환 → 이기어검 → 심검 등

성격:

- 현대 한국 웹소설·웹툰에서 독자 편의를 위해 표준처럼 널리 재사용되는 창작 관습
- 작품마다 순서·능력·상한이 크게 다름
- 중국 고전이나 모든 전통 무협에 공통된 역사적 체계가 아님

처리:

- 독자 친숙성 참고 자료로 보존
- 《팔대신병록》의 기본 체계로 자동 채택하지 않음
- `현경·생사경·자연경` 등은 고도 판타지화를 부를 수 있어 우선 제외 후보

## 3. 설계에서 비교할 구조안

### 안 1 — 현대식 단일 경지표

모든 무인을 한 줄에 배열한다.

장점: 이해가 가장 쉬움.
위험: 게임화·파워 인플레이션·시대감 붕괴.

### 안 2 — 사회적 품계 + 기술별 숙련

공적 평가는 `일류·절정·종사` 같은 느슨한 호칭을 사용하고, 실제 능력은 내공·신법·병기·감각·실전·협공으로 따로 기록한다.

장점: 전통 무협 질감과 상성 유지.
위험: 설명이 복잡할 수 있음.

### 안 3 — 사문별 성취어 + 외부 평판

외부인은 `고수·절정·종사`로 부르지만, 소림·무당·마교·군문 등은 서로 다른 수련 성취어를 사용한다.

장점: 문파 문화가 살아남.
위험: 용어 과잉.

### 안 4 — 무공 숙련 단계 + 심신 현상 분리

`입문·등당·입실·화경`은 기술 숙련을 말하고, `삼화취정·오기조원`은 도가 내공의 특수 현상으로 분리한다. 전투력은 별도 평가한다.

장점: 용어 출처를 존중하면서 깊이를 확보.
위험: 독자가 화경을 절대 레벨로 오해할 수 있음.

## 4. PM 예비 방향

현재 가장 유력한 구조는 안 2와 안 4의 결합이다.

- 강호의 외부 평판: 이류·일류·절정급·종사급 같은 느슨한 표현
- 특정 무학 숙련: 초규·등당·입실·화경
- 도문 내공 현상: 주천·삼화취정·오기조원
- 실제 전투 프로필: 내력·발경·신법·감각·병기·지구력·실전·협공으로 분리

아직 정본이 아니다.
