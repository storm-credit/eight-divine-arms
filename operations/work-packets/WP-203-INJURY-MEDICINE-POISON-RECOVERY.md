# WP-203 — 부상·의술·독·회복

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-202 완료
- 종료 조건: 전투와 신병 사용의 비용이 진단·치료·회복·후유증 규칙으로 연결될 것

## 2. 목적

무협식 치료가 모든 부상을 즉시 되돌리는 장치가 되지 않도록, 외상·내상·중독·감염·주화입마·정신 침식의 차이와 회복 한계를 확정한다.

## 3. 범위

- 상처와 출혈
- 골절·탈구·관절·근육 손상
- 장기 손상과 내출혈
- 무공 운용으로 생긴 내상과 기혈 장애
- 독의 투입 경로·용량·시간
- 감염·발열·부패
- 주화입마·공황·기억 침식
- 응급처치·수술·침구·탕약·금창약·해독
- 회복 단계·재활·후유증·재발
- 의원·의가·독문·검시관·기환 전문가의 역할 경계

## 4. 범위 밖

- 개별 약재·독약 도감
- 실제 역사 처방의 재현
- 특정 인물의 부상 이력
- 팔대신병별 고유 대가
- 강시·시귀의 최종 분류

## 5. 구조 후보

### A. 장르적 급속 회복형

침과 영약으로 대부분 며칠 내 복귀.

### B. 현실 외상 중심형

무공 부상도 일반 외상과 동일하게 수개월 회복.

### C. 내상 독립 질환형

내상과 경맥 손상을 별도 초자연 질환처럼 취급.

### D. 다층 혼합형

해부학적 손상·기능 장애·기혈 운용 장애·중독·감염·정신 손상을 구분하고 서로 겹칠 수 있게 한다.

## 6. 투입 에이전트

- 의술·독·검시 에이전트
- 무공 시스템 에이전트

- 생활·생계 에이전트
- 기환 경계 에이전트

- 정합성 감사
