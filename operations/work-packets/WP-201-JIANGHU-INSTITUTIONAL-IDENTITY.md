# WP-201 — 강호의 제도적 정체

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 종료 조건: 강호·사문·문파·세가와 국가 법의 관계가 구조적으로 설명되고 후속 세력 설계에 사용 가능할 것

## 2. 목적

강호가 단순히 ‘관부 밖의 무인 세계’라는 분위기 설명을 넘어, 명나라 사회 안에서 어떤 비공식 제도와 보증망으로 실제 존재하는지 확정한다.

## 3. 범위

이번 패킷에서 반드시 다룬다.

- 강호인의 법적·사회적 위치
- 사문 보증과 신원 확인
- 문파·세가·무관·산장·방회·단맥의 차이
- 비무·복수·파문·폐공·중재의 관습
- 무림맹 판정의 효력
- 관부 사건으로 전환되는 기준
- 무기 휴대·사병·호송·표행의 묵인 조건
- 외지 무인의 숙박·통행·고용

## 4. 범위 밖

- 구파일방 최종 명단
- 오대세가 최종 명단
- 개별 문파의 규율
- 주인공의 출신 사문
- 실제 무림대회와 서열 명단
- 무공 경지의 상세

범위 밖 아이디어는 `audit/SETTING-BACKLOG.md`로 이동한다.

## 6. 해결 질문

### 주 질문

강호는 국가 법과 별개인 자치 세계인가, 국가 사회 안의 비공식 보증망인가, 혹은 지역별로 다른 혼합 체계인가?

### 보조 질문

1. 강호인은 자신의 신원과 사문을 어떻게 증명하는가?
2. 무림맹의 판정은 누구에게 어떤 효력이 있는가?
3. 사적 복수와 비무는 어디까지 묵인되는가?
4. 관부가 개입하는 명확한 전환 조건은 무엇인가?

## 8. 구조적 후보

최소 네 안을 독립 생성한다.

### 후보 A — 사실상 독립 자치강호

문파와 무림맹이 자체 법·재판·징계를 갖고 관부와 영역을 나눈다.

검토 초점:

- 장르 직관성
- 국가 안의 국가가 되는 역사적 위험

### 후보 B — 사문 보증망

강호는 법적 신분이 아니라 사문·가문·표국·상단이 사람의 신원과 채무를 보증하는 비공식 사회망이다.

검토 초점:

- 역사적 현실성
- 강호의 낭만과 독립성이 약해질 위험

### 후보 C — 지역별 혼합 체계

중원·강남·해안·변경마다 관부와 무림의 경계가 다르며 무림맹은 이를 느슨하게 연결한다.

검토 초점:

- 지역 다양성
- 독자 설명 부담과 복잡도

### 후보 D — 공적·계약 기반 특권 체계

호송·토벌·구휼·치안 공적을 쌓은 문파와 세력만 지방관·향신에게 제한적 무력 사용과 자치 중재를 묵인받는다.

검토 초점:

- 표국·상단·관부와의 연결
- 강호가 관부의 하청조직처럼 보일 위험

각 후보는 독립안으로 평가하고, 혼합안은 비교 뒤에만 작성한다.
