# WP-303 — 공식 기록·전설·실제 진실

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-301~302 완료
- 종료 조건: 해문도 사건의 공식 기록·강호 전설·세력별 기록·민간 기억·실제 진실과 증거가 분리되어 후속 미스터리에 사용 가능할 것

## 2. 목적

단일 비급·고백·신병 기억으로 원사건을 해설하지 않고, 서로 다른 목적과 보관 경로를 가진 기록들이 사실·오해·은폐를 어떻게 나누는지 설계한다.

## 3. 범위

- 명 측 공식 기록
- 지방 관부와 군무 실무 기록
- 강호 전설과 병기록
- 상단·선단·전장 기록
- 사찰·의가·지역 공동체 기억
- 다언어·외부 해상 기록
- 최초 여덟의 개인 기록과 병기 잔흔
- 실제 사건의 확정 사실·논쟁·미상
- 증거 조각과 1555년 보존 가능성

## 4. 구조 후보

### A. 공식 기록은 거짓, 비밀 기록은 진실

명확한 이분법.

### B. 여덟 개의 동등한 상대주의

모든 기록이 각자의 진실이며 객관적 사실을 확정하지 않음.

### C. 숨은 단일 진실 문서형

완전한 원본 보고서 하나가 모든 사건을 설명.

### D. 층위별 부분 사실·교차 증거형

각 기록은 목적에 따라 사실 일부를 정확히 남기고 원인·책임·사람을 왜곡하며, 실제 진실은 기록·물증·신병 잔흔의 교차로 복원.
