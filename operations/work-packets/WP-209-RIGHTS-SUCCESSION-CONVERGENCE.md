# WP-209 — 신병 권리·계승·집합 현상

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-205~208 완료
- 종료 조건: 신병의 점유·소유·보관·사용·계승·공개·파기 권리와 복수 신병의 상호작용이 확정될 것

## 2. 목적

병기를 손에 넣은 자가 모든 권리를 자동 획득하는 구조를 막고, 신병 쟁탈을 재산·계승·책임·증거·사용 적합성의 복합 갈등으로 만든다.

## 3. 범위

- 점유·소유·보관·사용·계승 권리
- 감정·열람·공개·이관·파기 권한
- 상속·매매·담보·압수·전리·발견
- 분실·도난·위탁·표행
- 권리 증명과 분쟁 포럼
- 다른 신병과의 교차 반응
- 여러 신병이 모일 때의 효과·조건·위험
- 완전체·부활·절대무공 금지

## 4. 집합 현상 후보

### A. 단순 위력 증폭형

모일수록 사용자가 강해짐.

### B. 완전 기억 복원형

여덟이 모이면 원사건을 객관적으로 재생.

### C. 교차 인증형

진위·계승·공통 기억을 서로 검증.

### D. 합증·결정 자격형

교차 인증과 기억 비교를 통해 과거 협약의 일부 책임을 실행하거나 재정의할 자격이 성립.
