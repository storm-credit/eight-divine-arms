# WP-206 — 기록·증거·감정·위조 체계

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-205 완료
- 종료 조건: 관부·무림맹·문파·상단이 진실·진품·소유권을 판단하는 증거 기준과 위조 대응이 확정될 것

## 2. 목적

미스터리의 진실이 단일 비급·고백·신병 기억으로 즉시 확정되지 않도록, 증거의 종류·출처·보관·동기·검증 절차를 설계한다.

## 3. 범위

- 공문·판결·수배·압수 기록
- 족보·사승록·공적록·파문 기록
- 상단·표국·전장·창고·선박 장부
- 감정서·도록·비급·서신·암호
- 증언·소문·자백
- 병기·시신·독물·현장 물증
- 신병 기억·기환 체험
- 위조·검열·분실·대체·매수
- 보관·인계 연속성
- 판단 기관별 증거 우선순위

## 4. 구조 후보

### A. 문서 우선형

봉인된 문서와 공문이 가장 강한 증거다.

### B. 증인·명망 우선형

사람의 신분과 명예가 증거력을 결정한다.

### C. 물증·감정 우선형

병기·시신·재료·현장 흔적을 중심으로 판단한다.

### D. 다중 포럼·교차 증거형

관부·무림맹·상단·문파마다 기준이 다르며 중대한 결론은 서로 다른 종류의 증거를 교차한다.
