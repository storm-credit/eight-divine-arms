# WP-307 — 핵심 동료·경쟁자·관계 구조

## 1. 상태

- 상태: `active`
- 게이트: Gate 3
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-306 완료
- 종료 조건: 주인공의 부족한 전문성과 오해를 보완·반박하는 핵심 인물군과 관계 변화가 확정될 것

## 2. 목적

주인공 한 명에게 고급 무공·의술·정보·국가 수사·해상 역사·신병 감정을 몰아주지 않고, 서로 다른 기록과 권리를 대표하는 인물들이 협력과 충돌을 반복하도록 설계한다.

## 3. 범위

- 상시 핵심 동료 수
- 반복 경쟁자·조건부 동맹
- 각 인물의 전문 기능과 한계
- 욕망·오해·숨긴 과오
- 주인공과의 관계 변화
- 신병 접근 적합성
- 세력·신분·생계
- 감정적 대척과 신뢰 비용

## 4. 구조 후보

### A. 전통 사인 파티형

주인공·검객·의원·정보가가 처음부터 고정 동료.

### B. 대세력 대표단형

각 핵심 세력의 젊은 대표가 동행.

### C. 사건별 교체형

지역마다 동료를 전부 교체.

### D. 소규모 핵심 + 순환 동맹형

주인공과 3명의 상시 핵심, 2명의 반복 경쟁·조건부 동맹을 둔다.
