# WP-205 — 병기·법구·보패·신병 분류

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-202~204 완료
- 종료 조건: 희귀 물건의 명칭·격·감정·수리·위조·운송 차이가 후속 팔대신병 설계에 사용 가능할 것

## 2. 목적

모든 희귀 물건을 강한 순서로 한 줄에 놓지 않고, 제작·기능·기환성·재현성·역사·권리의 차이로 분류한다.

## 3. 범위

- 일반 병기·정병·명병·보병
- 법구·보패
- 신병
- 재료·제작·수리·파손·감정·위조
- 휴대·보관·운송·압수
- 전투 가치와 역사·의례 가치의 차이

## 4. 구조 후보

### A. 단일 위력 등급

범병 → 명병 → 보병 → 신병.

### B. 기환 유무 이분법

일반 물건과 기환 물건으로만 구분.

### C. 용도별 분류

병기·의료·의례·기록·운송 도구로 분류.

### D. 다축 분류

제작 품질·기능·재현성·기환 흔적·역사·권리를 별도 평가하고 통칭을 부여.
