# WP-208 — 신병의 기억·감정·동작 흔적

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-204, WP-206, WP-207 완료
- 종료 조건: 신병 흔적의 생성·내용·접근·왜곡·소실·대가가 확정될 것

## 2. 목적

신병 기억을 완전한 영상 기록이나 죽은 인격의 대화로 만들지 않고, 미스터리·무공·감정에 사용할 수 있는 제한된 흔적 체계로 설계한다.

## 3. 범위

- 흔적이 생성되는 조건
- 감각·동작·감정·언어·의도의 보존 비율
- 자발적·비자발적 접근
- 사용자 적합성과 해석
- 여러 시대 흔적의 중첩
- 파손·수리·계승품의 영향
- 기억 왜곡·소실·오인
- 무공 학습과 복제의 한계
- 침식·중독·의존

## 4. 구조 후보

### A. 완전 기록형

과거 장면을 영상처럼 재생.

### B. 감정 잔향형

강한 감정만 보존.

### C. 동작 기억형

손·호흡·궤적·타격 감각을 보존.

### D. 다층 잔흔형

감각·동작·감정·언어 파편이 서로 다른 비율로 중첩되고 현재 사용자가 재구성.
