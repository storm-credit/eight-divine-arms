# WP-212 — 세력–신병 관계 행렬

## 1. 상태

- 상태: `active`
- 게이트: Gate 2
- PM 담당: PM 오케스트레이터
- 선행 패킷: WP-209~211 완료
- 종료 조건: 핵심 세력이 신병·기록·증인·권리·파기 중 무엇을 왜 원하는지 구분되고 직접 충돌 수가 통제될 것

## 2. 목적

모든 세력이 같은 이유로 팔대신병을 쫓는 반복 구조를 막고, 세력별 목표·권리 주장·동맹·충돌을 기능적으로 분리한다.

## 3. 범위

- 물리적 소유·보관·사용·감정·계승
- 기록·증인·공개·은폐·파기
- 민간 피해와 배상
- 세력별 공개 목표와 실제 목표
- 자연 동맹과 구조적 충돌
- 신병 하나당 직접 추적 세력 상한
- A등급 14개 슬롯의 기본 입장

## 4. 관계 유형

- 확보: 물리적 점유·소유를 원함
- 보관: 사용보다 안전한 관리가 목적
- 사용: 특정 기능과 전투 운용이 목적
- 검증: 진위·기억·계승을 확인
- 기록: 문서·증언·원사건 정보를 추적
- 보호: 증인·유족·지역 피해를 우선
- 공개: 진실 또는 명예 회복
- 은폐: 위험·체면·정치 때문에 제한
- 무력화: 기능 봉인·분산·파기
- 중개: 거래·호송·협상·배상

- [ ] 신병별 최대 개입 세력 규칙
- [ ] 악마의 변호인
