# Gate 0 결정 기록 — 프로젝트 경계

상태: `resolved-provisional`
결정권자: 사용자 진행 승인 / PM 오케스트레이터

## 1. 판정

`Gate 0 — 프로젝트 경계`를 임시 통과한다.

프로젝트 헌법 v0.1은 후속 설계의 작업 기준으로 사용할 수 있다. 다만 사용자 최종 승인 전까지 `canon`이 아니라 `provisional-canon` 효력을 가진다.

## 2. 작업 기준으로 채택한 사항

- 장르 중심은 전통 무협이다.
- 보물 추적 모험과 미스터리가 주요 서사 엔진이다.
- 명나라 실제 역사는 결과를 바꾸지 않는 외부 골격으로 사용한다.
- 초자연은 팔대신병과 일부 유물에 집중된 저강도 기환 요소다.
- 죽은 인격의 완전하고 안정적인 부활은 기본적으로 금지한다.
- 기억·감정·동작의 잔향은 불완전하고 주관적인 정보로 취급한다.
- 정파·사파·천마신교·황실은 단순 선악으로 나누지 않는다.
- 세력은 무력뿐 아니라 경제·정보·물류·지역 기반으로 움직인다.
- 결말은 만족과 희망을 중심으로 하며, 일부 대가와 미지의 여운을 남긴다.
- 중요한 반전은 새 설정 투척보다 기존 사실의 재해석으로 만든다.

## 3. 반려한 기본안

다음 조합은 특별한 재설계 없이는 기본안으로 채택하지 않는다.

- 여덟 신병을 모으면 봉인된 절대고수가 그대로 부활한다.
- 주인공만 혈통 때문에 모든 신병을 다룰 수 있다.
- 하나의 최종 악역이 모든 사건을 완벽히 조종했다.
- 혈교를 제거하면 세계의 모든 갈등이 해결된다.
- 주인공이 모두에게 잊히거나 핵심 동료 대부분이 죽어야만 여운이 생긴다.

## 5. Gate 1 통과 조건

다음 항목이 검토되어야 한다.

- 기준 연도와 시작 지역
- 주요 권역과 이동 구조
- 황실·관부·무림의 공존 방식
- 문파·세가·상단·표국의 생존 기반
- 개인 고수와 국가 강제력의 균형
- 정보와 물류의 흐름
- 이후 세력 설계를 수용할 수 있는 슬롯 구조

## 6. 재검토 조건

아래 문제가 발생하면 Gate 0을 다시 연다.

- 역사 배경이 주제와 결말을 지나치게 제한할 때
- 팔대신병의 기능이 저강도 기환 범위를 유지할 수 없을 때
- 주제인 ‘진실의 소유와 전승’이 인물 감정과 분리될 때
- 만족·희망 중심 결말이 전체 갈등의 비용과 맞지 않을 때
