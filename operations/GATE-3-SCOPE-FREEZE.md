# 《팔대신병록》 Gate 3 범위 동결 v1.0

상태: `provisional-canon`
관문: `Gate 3 — 과거 진실·인물·핵심 갈등`

총괄: PM 오케스트레이터

## 1. Gate 3 목적

Gate 2에서 확정한 세계 규칙을 실제 역사·신병·인물·세력에 적용해 장편을 움직이는 서사 엔진을 만든다.

완료 시 다음이 성립해야 한다.

- 원사건의 실제 구조와 역사적 연대
- 최초 여덟 당사자가 살아남고 책임을 나눈 인과
- 공식 기록·강호 전설·각 세력의 오해·실제 진실
- 현재 팔대신병 여덟 지위의 개별 설계
- 혈통 선택 없이 사건 중심에 들어오는 주인공
- 동료·경쟁자·적대자·세력 대표의 욕망과 관계
- 적대자가 주인공을 즉시 제거하지 못하는 구조
- 현재 사건을 시작시키는 핵심 갈등과 증거 배분

## 2. 이번 Gate에서 완전 설계할 것

### T1. 원사건

- 사건 유형
- 역사적 시점과 장소
- 국가·강호·민간에 미친 영향
- 왜 은폐·왜곡되었는가
- 왜 여덟 당사자만 핵심 책임을 나누게 되었는가

### T2. 최초 여덟 당사자

- 사회적 위치와 소속
- 원사건에서의 실제 행동
- 서로의 관계와 불신
- 병기와 결속한 이유
- 생존·희생·책임 분산의 인과

### T3. 진실 층위

- 관부·왕조의 공식 기록
- 강호의 대표 전설
- 세력별 보존 기록과 오해
- 실제 사건
- 확인 가능한 증거
- 의도적 은폐와 우연한 소실

### T4. 현재 팔대신병

- 여덟 지위의 기능적 개성
- 형태 계통과 현재 상태
- 강한 기억 채널과 사각지대
- 기능·조건·대가
- 원본·수리 원본·계승품 여부
- 현재 보관·소유·사용·계승 분쟁

### T5. 핵심 인물

- 주인공
- 동료·경쟁자·감정적 대척점
- 주요 적대자와 반대자
- 세력 대표와 내부 반대자
- 현재 신병 보관자·사용자·증인

### T6. 핵심 갈등

- 주인공이 사건에 진입하는 직접 원인
- 적대 세력이 주인공을 이용·협상해야 하는 이유
- 첫 사건의 손실과 선택
- 각 인물이 원사건 진실과 현재 권리에서 원하는 것

## 3. 이번 Gate에서 뼈대만 설계할 것

- 시작 연월·계절과 본편 초기 동선
- 각 부에서 다룰 신병과 지역의 후보
- 실제 후기지수·은거기인·세력 대표의 명성
- 핵심 인물의 주무기·무공 계통
- 원사건과 현재 사건의 증거 공개 순서 초안

구체 부·권·장 플롯은 Gate 4에서 확정한다.

## 4. 이번 Gate 범위 밖

- 최종 결말 선택
- 여덟 신병이 모두 모이는지 여부
- 전체 부·권·장 구성
- 모든 조연과 지역 세력의 완전한 인물 바이블
- 실제 무림대회 전체 사건
- 장면·대사·전투 본문
- 세부 복식·가격·음식 도감

## 5. 작업 패킷 순서

### Truth Track

#### WP-301 — 원사건 구조와 연대

주 질문:
원말명초의 어떤 사건이 여덟 증언 병기와 책임 분산을 자연스럽게 낳았는가?

#### WP-302 — 최초 여덟 당사자와 관계

주 질문:
왜 정확히 이 여덟 사람이 살아남고 서로를 불신하면서도 협약을 맺었는가?

#### WP-303 — 공식 기록·전설·실제 진실

주 질문:
같은 원사건이 국가·강호·세력·민간에서 어떻게 다르게 기록되고 왜곡되었는가?

### Artifact Track

#### WP-304 — 현재 팔대신병 여덟 지위

주 질문:
현재 여덟 신병은 형태·기억·기능·대가·권리 갈등에서 어떻게 구별되는가?

#### WP-305 — 현재 소유·보관·계승 지도

주 질문:
본편 시작 시점에 각 신병과 관련 증거는 어디에 있고 누가 어떤 권리를 주장하는가?

### Character Track

#### WP-306 — 주인공 진입 엔진

주 질문:
주인공이 선택된 혈통 없이도 팔대신병 사건의 중심이 될 수밖에 없는 경험·직업·상처·기술은 무엇인가?

#### WP-307 — 핵심 동료·경쟁자·관계 구조

주 질문:
주인공의 오해와 선택을 바꾸는 핵심 인물들은 어떤 서로 다른 진실과 권리를 대표하는가?

#### WP-308 — 적대자·반대자·협상 구조

주 질문:
적대자는 왜 주인공을 즉시 죽이지 못하며, 무엇을 얻기 위해 이용·협상·보호·추적하는가?

### Faction and Conflict Track

#### WP-309 — 핵심 세력 최종 이름·대표 인물

주 질문:
Gate 2의 14개 기능 슬롯 중 실제 장편 전반에 필요한 12~14개는 무엇이며 누가 대표하는가?

#### WP-310 — 현재 사건의 발화점과 핵심 갈등

주 질문:
1555년의 어떤 현재 사건이 원사건·신병·세력·주인공을 동시에 움직이기 시작하는가?

#### WP-311 — 미스터리 층위·증거 배분

주 질문:
독자와 인물이 무엇을 어떤 순서로 오해하고 확인하도록 단서와 신병 기억을 배치할 것인가?

## 6. 모든 패킷의 의무 절차

- 팔대신병의 숫자·정의 변경
