# 《팔대신병록》 Gate 5 범위 동결

상태: `provisional-canon`
관문: `Gate 5 — 집필 준비·최종 감사`
총괄: PM 오케스트레이터
선행 결정: [`GATE-4-DECISION.md`](GATE-4-DECISION.md)

Gate 5는 새로운 세계관·세력·신병·주요 인물을 추가하는 단계가 아니다.

---

# 1. Gate 5 목적

Gate 0~4의 설계를 실제 본문 개요와 장면으로 안전하게 전환할 수 있도록 표현 규칙과 장부를 완성한다.

---

# 2. 작업 패킷

## WP-501 장면·서사 단위 규칙

- 장면 목표·갈등·결과
- 후속 반응·딜레마·결정
- 설정 설명을 행동으로 변환
- 인물·세력·신병 과밀 제한

## WP-502 전투 설계 규칙

- 전투 목적
- 거리·지형·병장기·상성
- 협공·군진·화기·독·기환
- 부상·피로·도주·포획
- 승패 뒤 비용과 정보 변화

## WP-503 시점·문체·정보 공개 규칙

- 기본 시점과 제한
- 신병 잔흔 표현
- 역사·제도 설명 밀도
- 대화·문서·기록 인용 방식
- 장르 어휘와 현대적 표현 제한

## WP-504 연속성 장부 동기화

- 전체 연표
- 이동·계절
- 인물 나이·부상·회복
- 신병 위치·소유·사용·침식
- 인물 지식·관계·신뢰
- 단서·복선·가짜 가설
- 사망·구금·수배·배상

## WP-505 역사·지리·용어 감사

- 1555~1556년 명나라 배경
- 원말명초 방국진 계열 배경
- 관부·금의위·군무·상업·표국 표현
- 실제 지명·수로·계절·항해
- 인명·도호·조직명·직업명·병기명
- 시대착오·청대·근현대 용어 제거

## WP-506 설정집 통합

- 최상위 설정집
- 정본·임시 정본·후속 상세화 구분
- 기존 후보 문서의 지위 정리
- 문서 인덱스와 장부 링크

## WP-507 집필 준비 최종 감사

- 선행 설정 충돌
- 플롯·인물·복선 완결성
- 장면 전환 가능성
- 미확정 항목이 본문을 막는지
- 원고 개시 승인 여부

---

# 3. Gate 5 고정 제한

- 새 A등급 세력 추가 금지
- 새 팔대신병·원사건 핵심 당사자 추가 금지
- 결말 방향 변경 금지
- 실제 장편 본문 집필 금지
- 장별 상세 개요는 집필 개시 뒤 첫 작업으로 남김
- 역사 검증 때문에 허구 핵심 구조를 불필요하게 실제 사건으로 바꾸지 않음
- 이름과 용어는 조정할 수 있으나 역할·인과를 조용히 변경하지 않음

---

# 4. 완료 조건

- 장면 규칙
- 전투 규칙
- 문체·시점·정보 공개 규칙
- 필수 장부 동기화
- 역사·용어 감사 결과
- 설정집 통합본
- 미해결 질문의 `집필 차단 / 집필 중 해결 / 후속 상세화` 분류
- 최종 악마의 변호인 감사
- 집필 준비 결정
