# 《팔대신병록》 설정집 Human Edit 검토 패킷 v1

상태: `review-ready / recommended-defaults`
총괄: PM 오케스트레이터
상위 기준: `design/SETTING-BIBLE-v2.0-MASTER.md`

## 1. 목적

설정집 구조와 운영 규칙은 완성됐다. 이 문서는 설정을 다시 넓히는 것이 아니라, 작품의 최종 체감과 취향을 결정하는 항목만 압축해 검토한다.

판정 상태:

- `resolved-user`: 사용자가 이미 결정
- `recommended-default`: PM 권고안, 사용자 반대가 없으면 후속 기준으로 사용 가능
- `open-taste`: 정본 승격 전 사용자 취향 확인이 필요한 항목

## 2. 결정 1 — 기환의 체감 강도

상태: `recommended-default / open-taste`

### 현재 기반

- 세계 대부분의 사건은 무공·독·기관·의술·사기·오인으로 설명 가능하다.
- 팔대신병과 일부 보패·잔념·영물만 설명 불가능한 경계에 접근한다.
- 완전 부활·범용 주문·신격 상시 강림·대량 생산 보패는 금지한다.

### 후보 A — 극저강도

- 설명 불가능한 현상이 거의 확정되지 않는다.
- 팔대신병 잔흔도 독·집단기억·신체 학습으로 해석 가능성을 남긴다.

장점:
- 역사 무협 질감이 가장 강함
- 공포와 미스터리의 여백이 큼

위험:
- `신병`과 기환 설정의 경이가 약해질 수 있음
- 독자가 끝까지 초자연의 실재를 의심할 수 있음

### 후보 B — 저강도 확정형

- 팔대신병 잔흔·일부 보패·특정 강시·영초 현상은 실제로 존재한다.
- 다만 작동 범위는 좁고 조건·대가·오인이 명확하다.
- 일반 강호인은 평생 진짜 기환을 한두 번 볼까 말까 한다.

장점:
- 전통 무협과 신비의 균형
- 이미 설계한 팔대신병·부적·영물·강시 시스템을 가장 자연스럽게 활용
- 진짜와 사기의 판별 갈등 유지

위험:
- 개별 사건에서 기환을 자주 쓰면 체감 강도가 빠르게 올라감

### 후보 C — 중저강도

- 문파·지역마다 실제 기환 사례가 누적되고 전문 조직이 안정적으로 다룬다.
- 영수·보패·강시·진법이 반복 사건의 주요 축이 될 수 있다.

장점:
- 시각적·사건적 다양성 증가
- 기환 전문 세력의 존재감 강화

위험:
- 팔대신병의 희소성이 감소
- 명대 역사 미스터리보다 판타지 시스템이 전면화될 위험

### PM 권고

> **후보 B — 저강도 확정형**

운영 기준:
- 일반 사건 70~80%는 현실적 설명으로 해결 가능
- 기환 의심 사건 중 실제 기환은 소수
- 한 권역·한 사건에서 실제 기환 핵심 요소는 최대 1~2개
- 팔대신병은 세계 최고 화력이 아니라 가장 깊은 역사 잔흔과 권리 문제를 가진 유물

영향 모듈:
- D12~D16
- 팔대신병 잔흔
- 사교 판정

## 3. 결정 2 — 익숙한 문파·세가와 작품 고유 세력의 비중

상태: `recommended-default / open-taste`

### 후보 A — 고전 명문 중심

- 소림·무당·개방·남궁·당문 등 익숙한 이름이 주요 갈등의 중심
- 고유 세력은 보조 기능

장점:
- 진입 장벽이 낮음
- 전통 무협의 친숙함이 강함

위험:
- 작품 고유의 해상·표국·기록·신병 권리 구조가 익숙한 문파 정치에 묻힘
- 다른 무협과 차별성이 약해질 수 있음

### 후보 B — 혼합 균형형

- 익숙한 명문은 세계의 기준점·정통 정치·독자 안내 역할
- 실제 사건을 움직이는 핵심은 진원표국·창해상련·하오문·해상 공동체·의가·군문·감정기관 같은 기능 세력

작업 비중 기준:
- 직접 등장 A급 세력 가운데 익숙한 명문 약 35~45%
- 작품 고유·기능 세력 약 55~65%
- 한 사건에 직접 개입하는 핵심 세력 최대 3~5개

장점:
- 전통 무협 친숙함과 작품 고유성 병존
- 세력의 돈·정보·호송·기록·지역 기능을 살릴 수 있음

위험:
- 고유 조직 명칭과 역할 설명을 초기에 자연스럽게 보여 줘야 함

### 후보 C — 고유 세력 중심

- 익숙한 문파는 배경·참조 수준
- 대부분의 핵심 세력은 작품 고유 조직

장점:
- 독창성 최대화
- 해상·상업·기록·지역 세계가 강하게 드러남

위험:
- 전통 무협 기대감이 약해질 수 있음
- 고유명 과부하 가능

### PM 권고

> **후보 B — 혼합 균형형**

세부 원칙:
- 소림·무당·개방·남궁·당문 등은 장르의 기준점
- 사건의 실무와 선택은 표국·상단·지역 세력·의가·기록기관이 담당
- 구파일방·오대세가는 전체 세계의 주인공이 아니라 대표 의석 정치
- 천마신교·사도련·혈교도 동일한 방식으로 기능·재정·내부 파벌을 가짐

영향 모듈:
- D08
- 권역별 세력 배치
- 후속 A/B/C 세력 승격

## 4. 결정 3 — 팔대신병 명명 방식

상태: `recommended-default / open-taste`

### 현재 작업명

- 개문창
- 절류도
- 철조구
- 묵형척
- 연생장
- 청맥인
- 해명궁
- 봉서검

### 후보 A — 현재 이름 단일 사용

장점:
- 기억하기 쉽고 기능과 주제가 분명함
- 여덟의 구분이 빠름

위험:
- 기능을 너무 직접 설명하는 이름으로 보일 수 있음
- 모두 같은 시대·같은 작명자가 지은 세트처럼 느껴질 가능성

### 후보 B — 정식명 + 지역 이형명

각 신병에 세 층을 둔다.

1. 현재 강호 공통명
2. 지역·생업권의 속칭
3. 원사건 당시 또는 보관 기록의 옛 이름

예시 구조:
- `개문창`은 강호 공통명
- 군문 무가는 다른 당호·별칭 사용
- 쇄조도 당시 장부에는 평범한 군창 또는 도구명으로 기록

장점:
- 신병이 후대에 `세트로 창조된 물건`이 아니라 역사적으로 편찬된 여덟이라는 점 강화
- 지역 문화와 선전 차이를 반영
- 현재 작업명의 가독성 유지

위험:
- 이형명을 너무 많이 만들면 독자 혼란

### 후보 C — 전면 재명명

- 각 신병을 더 고전적이고 함축적인 이름으로 변경

장점:
- 문학적·전설적 인상 강화 가능

위험:
- 새 이름 검토가 설정의 핵심 작업을 다시 흔듦
- 한자 조합이 인위적이거나 선협풍으로 갈 수 있음

### PM 권고

> **후보 B — 현재 작업명을 강호 공통명으로 유지하고, 각 신병에 지역 이형명 1개와 원기록명 1개만 추가**

규칙:
- 총 이름 수는 신병당 최대 세 개
- 본문 기본 표기는 현재 작업명
- 첫 등장 때만 지역명·옛 이름 병기
- 사방신·팔괘·속성 세트식 개명 금지
- 최종 한자·중국어 자연성은 명명 감사에서 별도 검증

영향 모듈:
- S7 개별 카드
- 용어집
- 지역 문화

## 5. 결정 4 — 쇄조도 원사건의 전면 배치 정도

상태: `recommended-default / open-taste`

### 후보 A — 공개된 대전설

- 강호 대부분이 쇄조도와 팔대신병의 기원을 안다.
- 세력은 해석과 소유권을 두고 다툰다.

장점:
- 설명이 쉽고 사건 진입이 빠름

위험:
- 역사 미스터리의 발견감 감소
- 187년 동안 은폐·왜곡된 이유가 약해짐

### 후보 B — 반쯤 알려진 재난

- 연해·상업·군문·일부 강호에는 서로 다른 전승이 남아 있다.
- `쇄조도`라는 이름과 큰 참변은 알려졌지만, 여덟 당사자·이관 목적·책임 구조는 알려지지 않았다.

장점:
- 세계에 실제 역사 흔적이 남음
- 사건의 존재 자체보다 의미와 책임을 조사하게 됨
- 지역별 전설·문서·권리 충돌 활용 가능

위험:
- 알려진 것과 숨겨진 것의 경계를 정교하게 관리해야 함

### 후보 C — 완전한 비밀사

- 극소수만 원사건을 알고 대중은 신병 전설만 안다.

장점:
- 강한 비밀 역사 미스터리

위험:
- 모든 단서를 비밀 문서·숨은 조직이 독점하게 됨
- 지역 공동체 기억과 피해자 후손의 존재가 약화

### PM 권고

> **후보 B — 반쯤 알려진 재난**

정보 층:
- 대중: 오래전 연해 화재·침몰·실종 전설
- 지역민: 봉쇄·구조·장부 소각·이주에 관한 상충 기억
- 세력 기록자: 이관 작전과 여덟 물건의 연관 가능성
- 극소수: 여덟 당사자의 책임 분담과 쇄조 밀약의 실체

원칙:
- 사건 존재보다 `누가 무엇을 숨겼고 왜 지금 공개하면 누가 다치는가`가 중심
- 지역 공동체가 단순 단서 제공자가 아니라 기억과 공개권의 주체

영향 모듈:
- D23
- 원사건·진실 계층
- 팔대신병 지역 문화

## 6. 결정 5 — 역사 고증과 장르 가독성의 균형

상태: `resolved-user`

사용자 결정:
- 진원표국·검표수·인계표사·표사·표두·금의위 남방 회수별조 유지
- 시대착오 가능성을 이유로 장르 용어를 일괄 교체하지 않음

최종 운용안:

> **역사적 골격과 사건 인과는 엄격하게, 강호의 통칭과 직업명은 장르 가독성을 우선한다.**

엄격 적용:
- 실제 역사 결과
- 연도·계절·지리
- 국가기관의 권한
- 전쟁·왜구·위소·군문·수운
- 기술·무기·의술의 가능 범위

유연 적용:
- 표국·표사류 장르 통칭
- 강호의 조직·경지·별호
- 허구 기관의 강호식 약칭
- 독자가 이해하기 위한 번역어

주의:
- 공식 공문 안에서는 필요 시 실제 기능 설명 병기
- 장르 통칭을 실제 전국 상설 제도로 소급하지 않음
- 역사적 사실을 편의상 뒤집지는 않음

## 7. PM 통합 권고안

다섯 항목의 권고 조합:

1. 기환: 저강도 확정형
2. 세력: 익숙한 명문 35~45% + 고유 기능 세력 55~65%
3. 신병 명칭: 현재 작업명 유지 + 지역 이형명·원기록명 각 1개
4. 쇄조도: 반쯤 알려진 재난
5. 역사/장르: 역사 골격 엄격 + 장르 통칭 가독성 우선

이 조합은 다음을 동시에 보존한다.
- 전통 무협의 친숙함
- 팔대신병의 신비
- 역사 미스터리의 발견감
- 해상·상업·기록·권리라는 작품 고유성
- 설정이 선협·게임·관료 절차물로 치우치는 위험 통제

## 9. 승인 방식

사용자는 다음처럼 전체 또는 일부를 수정할 수 있다.

- `PM 권고안 그대로`
- `1번만 A로`
- `2번은 고전 명문을 더 늘림`
- `신병 이름은 나중에 다시 검토`

별도 수정이 없으면 이 문서의 PM 권고안을 `recommended-default`로 유지하고, 정본 승격은 사용자 명시 승인 뒤 실행한다.
