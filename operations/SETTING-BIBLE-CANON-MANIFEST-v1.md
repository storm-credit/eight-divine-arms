# 《팔대신병록》 설정집 정본 매니페스트 v1

상태: `canon-manifest / active`
총괄: PM 오케스트레이터
승인 근거: `HUMAN-EDIT-APPROVAL-DECISION-v1.md`
최상위 정본: `design/SETTING-BIBLE-v2.0-MASTER.md`

## 1. 목적

개별 파일의 오래된 헤더를 일괄 치환해 기록 이력을 훼손하지 않고, 현재 유효한 정본 상태를 이 문서에서 통제한다.

충돌 시 이 매니페스트와 통합 설정집 v2.0을 우선한다.

## 2. 상태 분류

### `canon`

상위 규칙과 작품 고유 핵심. 후속 설계가 직접 전제로 사용한다.

### `canon-framework`

구조와 운영 원리는 정본이지만 지역·시기·고유명·수치는 사용 전 국소 조정할 수 있다.

### `canon-with-local-verification`

상위 방향은 정본이나 실제 관직·일자·지명·가격·언어 사용 시 추가 고증이 필요하다.

### `deferred-reference`

설정 검증에 사용한 과거 인물·플롯·장면 참고안. 정본 근거로 사용하지 않는다.

## 4. `canon-framework`

- S1 경제·생활·교통·범죄 운영 모듈
- S2 조직·세력·무림맹·표국 운영 모듈
- S3 수련·병장기·의술 운영 모듈
- S4 부적·진법·영물·죽음·종교 모듈
- S5 권역·외부 무림·세력 배치 모듈
- S6 문해·기록·명성·용어 모듈
- 가격·권리·수리·보관 장부

이 문서들의 구조와 금지선은 정본이다. 개별 이름·가격·지역 수치·사건 적용은 국소 검증할 수 있다.

## 5. `canon-with-local-verification`

- 실제 명나라 관직·기관의 세부 명칭
- 음력 일자·절기·풍향·수운 조건
- 실제 지명·거리·이동 시간
- 동전·은·곡물의 특정 시점 환율
- 권역별 방언·외국어 음역
- 실제 병기·의술·종교 용례의 세부 명칭

상위 설정을 변경하지 않고 사용 직전 검증한다.

## 6. `deferred-reference`

- 심소연 주인공안
- 사문경 적대자안
- 핵심 동료·경쟁자안
- 6부 대서사
- 전체 시놉시스
- 60장 개요
- 장면 카드
- 구체적 결말 실행안
- 초고 시도

## 7. 승인된 Human Edit 기준

- 기환: 저강도 확정형
- 세력: 익숙한 명문 35~45%, 고유 기능 세력 55~65%
- 팔대신병: 현재 공통명 + 지역 이형명·원기록명
- 쇄조도: 반쯤 알려진 재난
- 역사/장르: 역사 골격 엄격 + 강호 통칭 가독성 우선

## 8. 변경 관리

- C0 표기·링크: 즉시 수정
- C1 고유명·속칭: 관련 카드 수정
- C2 세력·신병·기환 규칙: 관련 S묶음 재감사
- C3 작품 헌법·기환 강도·역사 구조: 사용자 승인 후 정본 재개방
