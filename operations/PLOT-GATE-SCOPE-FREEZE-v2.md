# 《팔대신병록》 결말·플롯 Gate 범위 동결 v2

상태: `active-scope-freeze`
총괄: PM 오케스트레이터
상위 정본:
- `design/SETTING-BIBLE-v2.0-MASTER.md`
- `design/CHARACTER-BIBLE-v2.0-MASTER.md`

## 1. Gate 목적

설정과 인물 정본에서 자연스럽게 나오는 결말·대서사·미스터리 구조를 설계한다. 기존 6부·60장·장면 카드의 자동 복원은 금지한다.

## 2. 이번 Gate에서 확정할 것

1. 결말의 가치 판정과 실행 원리
2. 대서사 구조 후보와 권고안
3. 중심 권역·중심 신병·간접 신병의 비중
4. 심소연의 과거 인계 사고가 현재 갈등과 연결되는 방식
5. 삼각 적대 압력의 단계별 충돌
6. 미스터리 핵심 질문과 증거 종류
7. 부·막 단위의 목표·비가역 변화
8. 인물 합류·이탈의 원칙
9. 로맨스 기본값의 플롯 영향 여부
10. 플롯 전체 교차감사와 Human Edit 패킷

## 3. 이번 Gate에서 확정하지 않을 것

- 장 수
- 장면 카드
- 장면별 단서 배치
- 대사·문체·연출
- 전투 동작의 상세
- 정확한 날짜·시간표
- 모든 조연 이름
- 실제 사망 장면
- 원고 문장

## 4. 후보 생성 규칙

- 결말 최소 5안
- 대서사 최소 4안
- 첫 사건 최소 3안
- 기존 6부안은 후보 하나로만 취급
- 한 신병씩 수집하는 8단계 구조는 기본 반려 대상
- 최종안은 직접 추적 신병 최대 4~5개
- 나머지 신병은 현 사용자·기록·지역 대표·교차 증언으로 참여 가능

## 5. 중심 질문

> 심소연은 무엇을 발견하는가보다, 누구의 물건·기록·몸·이름을 어떤 조건으로 누구에게 넘길 책임을 지게 되는가?

## 6. 미스터리 질문 층위

### Q1 현재 사건

- 누가 현재 인계·보관·압수 절차를 무너뜨렸는가
- 무엇을 훔치려 했고 무엇을 지우려 했는가

### Q2 쇄조도 역사

- 1368년 이관 목적과 봉쇄의 실제 성격
- 여덟 당사자가 무엇을 살리고 무엇을 희생했는가

### Q3 현재 권리

- 현재 사람과 공동체를 해치지 않으면서 무엇을 공개·봉인·배상할 것인가
- 팔대신병은 누구에게 돌아가야 하는가

Q1의 범인과 Q2의 책임자와 Q3의 정치적 적대자를 한 사람으로 합치지 않는다.

## 7. 대서사 밀도 규칙

- 전체 직접 핵심 세력: 8~10개 이내
- 한 부 직접 핵심 세력: 3~5개
- 상시 핵심 인물 전원 동행 금지
- 한 부 실제 기환 핵심 요소: 최대 1~2개
- 한 부 중심 신병: 1개, 교차 신병 최대 1개
- 동일한 `문서 발견 → 추격 → 탈취` 반복 금지

## 8. 비가역 변화 규칙

각 부는 최소 두 항목을 바꾼다.

- 신병의 점유·보관·사용 상태
- 인물 관계와 보증
- 부상·수배·채무·신용
- 공개된 진실의 범위
- 세력의 동맹·분열
- 지역 공동체의 위험
- 주인공의 직업·가족·사문 위치

## 9. 결말 규칙

결말은 다음을 모두 답해야 한다.

- 사실 골격은 어디까지 공개되는가
- 현재 신원·항로·환자·피난처는 어떻게 보호되는가
- 누가 배상·복권·재심을 담당하는가
- 신병은 어디에 보관되고 누가 사용할 수 있는가
- 사문경·위철산·장계엄의 논리 중 무엇이 남고 무엇이 무너지는가
- 심소연의 직업과 가족·가업에 어떤 비용이 남는가

## 10. 로맨스 잠금

- 로맨스 상대는 이번 Gate의 필수 결정이 아니다.
- 느린 단일 관계 가능성을 열어 둘 수 있으나 대서사 해결 기능을 주지 않는다.
- 플롯 최종안이 특정 관계를 강제하면 별도 Human Edit 항목으로 올린다.

## 11. 완료 조건

- 결말·대서사 후보 비교
- PM 권고안
- 중심 신병·권역·세력 배치
- 첫 사건과 주인공 과거 연결의 구조
- 삼각 적대 압력 단계
- 증거·가설·반전 구조
- 전문가 교차감사
- 사용자 Human Edit 패킷

## 12. 잠금

이 Gate가 끝나도 장면·원고는 자동 개방하지 않는다. 플롯 정본 승인 뒤 별도 `Outline Gate`를 열어야 한다.
