# 《팔대신병록》 Gate 2 전문가 브레인스토밍 운영안

상태: `active`
총괄: PM 오케스트레이터
대상: 세력 아키텍처 / 팔대신병 / 영적 생태계

## 1. 교정 사항

이전의 `여덟 역할과 계승 자리` 안은 최종 설계가 아니다.

현재 지위:

- `brainstorm candidate`
- 다른 기원·구조 가설과 동등하게 비교
- 전문가 반론과 정합성 검토 전 상세화 금지

PM은 하나의 아이디어를 먼저 상세화한 뒤 나머지를 끼워 맞추지 않는다.

## 2. 전문가 에이전트 편성

### A. 무협 장르 에이전트

검토 대상:

- 전통 무협의 질감
- 신병·보패·부적·진법이 무공을 압도하는지
- 문파·세가·사승·은원과 연결되는지
- 보물 쟁탈이 반복 퀘스트가 되는지

산출물:

- 무협적으로 매력적인 기원 가설 5개
- 각 가설의 전투·비무·계승 활용법
- 장르 이탈 위험

### B. 중국 신화·종교 에이전트

검토 대상:

- 사방신·오방·성수·신수
- 도교 법기·부적·진법·강령 의례
- 불교 법구·진언·사리·호법 개념
- 영물·영수·영초의 문화적 사용

산출물:

- 직접 현현형 / 상징형 / 지맥형 / 종문 해석형 비교
- 명나라 시대 종교문화와 충돌 가능성
- 현대 판타지식 오용 경고

### C. 역사·민속 검증 에이전트

검토 대상:

- 가정 연간의 도교·불교·민간신앙
- 장례·시신·강시 전승의 시대성
- 약재·희귀동물·광물·연금술 인식
- 지역별 신앙과 영적 생태 차이

산출물:

- 역사적으로 가능한 요소
- 후대 창작 관습인 요소
- 작품적 각색이 필요한 요소

### D. 시스템 설계 에이전트

검토 대상:

- 팔대신병 / 보패 / 법기 / 부적 / 영약 / 영수의 등급과 역할
- 능력 조건·대가·희소성
- 경제·유통·위조·감정
- 파워 인플레이션

산출물:

- 최소 3개의 유물 생태계 모델
- 각 모델의 장점·복잡도·관리 비용
- 팔대신병의 독점적 위상 보존 방법

### E. 미스터리·서사 에이전트

검토 대상:

- 신병이 무엇을 숨기고 드러내는지
- 단서와 거짓 단서
- 각 신병 사건의 차별화
- 결말에서의 선택과 회수

산출물:

- 기원 가설별 미스터리 구조
- 초반·중반·후반 공개 순서
- 뻔한 반전 위험

### F. 세력·경제 에이전트

검토 대상:

- 누가 영초를 재배·채집·유통하는가
- 영수·영물의 서식지와 보호권
- 보패 제작자와 감정기관
- 황실·문파·상단·암시장의 이해관계

산출물:

- 유물 생태계가 세력 구조를 만드는 방식
- 독점과 거래가 가능한 자원
- 경제적으로 성립하지 않는 설정 경고

### G. 인물·감정 에이전트

검토 대상:

- 신병과 보패가 인물 관계에 주는 영향
- 영수·영물과의 유대가 애완동물 클리셰가 되는지
- 사용 대가가 감정적 선택으로 연결되는지
- 결말 여운과 인간성

산출물:

- 인물 중심의 사용 사례
- 희생이 아닌 대가 모델
- 관계 변화에 유효한 설정

### H. 악마의 변호인

새 설정 생성 금지.

검토 대상:

- 선협화
- 포켓몬식 신수 수집
- 게임 아이템 등급표화
- 사방신 네 속성 단순화
- 영초 복용만으로 경지 상승
- 보패가 모든 위기를 해결
- 강시·시귀가 장르 중심을 빼앗음

산출물:

- 반려안
- 축소안
- 더 단순한 대안

### I. 정합성 감사

검토 대상:

- 용어 중복
- 시대·지역 충돌
- 팔대신병과 보패의 위상 충돌
- 유물 소유권·소모·재생성
- 생태계와 경제 장부

산출물:

- 충돌표
- 미해결 질문
- 정본 승격 조건

## 3. 브레인스토밍 범위

다음 질문을 넓게 탐색한다.

### 팔대신병

- 하나의 사건에서 만들어졌는가?
- 서로 다른 시대의 유물을 후대가 묶었는가?
- 여덟 물건인가, 여덟 역할인가, 여덟 계보인가?
- 사방신·오행·팔괘·팔방과 관계가 있는가?
- 숫자 여덟은 기능적 필연인가 후대의 분류인가?

### 영적 생태계

- 영물: 오래된 사물·광물·장소가 변한 존재인가?
- 영수: 실제 동물의 희귀 변종인가, 명확한 초자연 생물인가?
- 영초: 약성이 강한 희귀 식물인가, 기운을 축적한 식물인가?
- 신수: 실제 개체인가, 종문이 붙인 상징적 칭호인가?
- 사방신: 현현하는 신인가, 지리·진법·권력의 상징인가?

### 술법·유물

- 부적은 약물·기호·내력의 결합인가?
- 진법은 전술·지형·심리인가, 실제 공명 현상인가?
- 보패는 법구인가, 호신구인가, 문파 증표인가?
- 강령술은 잔념 해석인가, 시신 조종인가?
- 강시·시귀는 실제 초자연인가, 독·약·기관으로 위장된 현상도 포함하는가?

## 4. 후보 모델 생성 규칙

최소 다섯 모델을 만든다.

1. 역사무협형 — 초자연 최소
2. 저강도 기환형 — 현재 헌법과 가장 가까움
3. 신화상징형 — 사방신·팔괘를 상징과 체계로 사용
4. 영적 생태형 — 영물·영수·영초가 희귀 생태계를 구성
5. 혼합형 — 위 요소를 제한적으로 결합

각 모델은 다음 표로 평가한다.

- 무협성
- 보물 모험성
- 미스터리성
- 역사 적합성
- 감정 활용도
- 독창성
- 복잡도
- 선협화 위험
- 장편 유지 가능성
