# 《팔대신병록》 집필·설계 최상위 정책

상태: `canon-policy / plot-gate-human-edit`
역할: 저장소 최상위 운영 정책
총괄: PM 오케스트레이터

## 1. 기본 원칙

- 설정집·인물·플롯·Outline을 본문보다 먼저 완료한다.
- 아이디어·후보·참고안은 정본과 분리한다.
- 중요 설계는 복수안·전문가 감사·사용자 Human Edit를 거친다.
- 역사·연표·거리·가격·권리·관계·부상을 장부로 검증한다.
- `canon` 변경은 영향 범위와 재감사를 기록한다.
- 원고는 Plot Gate와 Outline Gate 통과 전까지 잠근다.

## 2. 공식 파이프라인

`Draft → Validate → Human Edit → Finalize`

## 4. 설정집 정본

승인 기준:
- 저강도 확정형 기환
- 익숙한 명문 35~45%, 고유 기능 세력 55~65%
- 팔대신병 공통명 + 지역 이형명·원기록명
- 쇄조도를 반쯤 알려진 재난으로 운용
- 역사 골격 엄격 + 장르 통칭 가독성 우선

## 5. 인물 정본

### 주인공

- 심소연 작업명
- 여성, 26세
- 진원표국 검표수·인계표사
- D3 철조구 제한 적합, 소유권 없음
- 사람을 보호하려 인계 기록을 수정하고 타인에게 책임을 전가한 과오

### 상시 핵심

- 진소무 — 하오문·가명·정보망
- 두약란 — 의술·독·검시
- 유청하 — 정통 검술·명문 정치·불안정한 내부 이탈자
- 나해란 — 해로 통역·궁술·D7 전승궁

### 적대 압력

- 사문경 — 진실 편집·승인 중앙화
- 위철산 — 공동체 보호를 위한 기록·신병 파기
- 장계엄 — 국가 압수·봉인·비밀 구금

세 사람은 하나의 조직이 아니다.

## 6. 플롯 Gate PM 권고

- 결말: 단계적 합증·분산 귀환
- 대서사: 5막·8구간 직조형
- 첫 사건: 조선소 붕괴·이중 인계
- 보호 대상: 피해자 측 직접, 보호 대상은 늦게 제한 등장
- 핵심 인물: 필수 사망 없음
- 최종부: 쇄조도 현장 검증 + 복수 거점 동시 결산
- 로맨스: 느린 단일 관계 가능성, 상대 미확정

이 구조는 사용자 승인 전 `recommended-default`다.
