# 《팔대신병록》 집필·설계 최상위 정책

상태: `canon-policy / character-gate-human-edit`
역할: 저장소 최상위 운영 정책
총괄: PM 오케스트레이터

이 문서는 《팔대신병록》 프로젝트의 모든 설계·검증·집필 작업보다 우선한다.

## 1. 기본 원칙

- 설정집·인물·플롯 설계를 본문보다 먼저 완료한다.
- 아이디어·후보·참고안은 정본과 분리한다.
- 중요한 설계는 복수안·전문가 감사·사용자 Human Edit를 거친다.
- 역사·연표·거리·가격·권리·관계를 장부로 검증한다.
- `canon` 변경은 영향 범위와 재감사를 기록한다.
- 원고는 인물 Gate와 플롯 Gate 통과 전까지 잠근다.

## 2. 공식 파이프라인

`Draft → Validate → Human Edit → Finalize`

설정집은 전 단계를 통과해 `canon`이다.
현재 인물·관계 Gate는 `Human Edit` 단계다.

## 3. 문서 위계

1. 최신 사용자 결정
2. `book_writing.md`
3. `design/SETTING-BIBLE-v2.0-MASTER.md`
4. `operations/SETTING-BIBLE-CANON-MANIFEST-v1.md`
5. `operations/CHARACTER-GATE-SCOPE-FREEZE-v1.md`
6. `operations/CHARACTER-HUMAN-EDIT-PACKET-v1.md`
7. `audit/CHARACTER-GATE-CROSS-AUDIT-v1.md`
8. `lore_index.md`
9. 설정·인물 지원 모듈과 장부
10. `원고_index.md`
11. 과거 플롯·장면 참고안

과거 심소연·사문경 확정안, 6부·60장 구조와 장면 카드는 정본 기준이 아니다.
현재 인물안도 사용자 승인 전에는 정본이 아니다.

## 4. 설정집 정본 기준

- 저강도 확정형 기환
- 익숙한 명문 35~45%, 고유 기능 세력 55~65%
- 팔대신병 현재 공통명 + 지역 이형명·원기록명
- 쇄조도를 반쯤 알려진 재난으로 운용
- 역사 골격 엄격 + 장르 통칭 가독성 우선

## 5. 인물 Gate 현재 권고

### 주인공

- 여성 표국 검표수·인계표사 단일 주인공
- 작업명 심소연
- 26세
- D3 철조구 제한 적합
- 기록 수정과 책임 전가의 과오

### 핵심 인물

- 진소무 — 하오문·가명·정보망
- 두약란 — 의술·독·검시
- 유청하 — 정통 검술·명문 정치
- 나해란 — 해로 통역·궁술

### 적대 구조

- 사문경 — 편집·승인 중앙화
- 위철산 — 기록·신병 파기
- 장계엄 — 국가 압수·봉인

이 구조는 `recommended-default`이며 사용자 승인 전 정본이 아니다.

## 6. 플롯·원고 잠금

현재 금지:

- 특정 범인·결말 확정
- 기존 6부·60장 플롯 복원
- 장면 카드
- 대사·문체·연출
- 장편 초고

## 7. 인물 Gate 승인 뒤 순서

1. 인물 구조 결정 로그
2. 성명·한자·호칭 국소 감사
3. 가족·스승·분점 카드
4. 인물 Gate 정본 승격
5. 별도 지시 후 결말·플롯 Gate 범위 동결

## 8. 변경 관리

- C0 표기·링크
- C1 이름·성별·연령·관계 슬롯
- C2 핵심 인물 구조·신병 적합성·적대 구조
- C3 설정집 헌법·역사 구조

C2 이상은 관련 감사와 사용자 승인이 필요하다.

## 9. 현재 실행 상태

- 설정집: `canon`
- 인물 Gate: `human-edit-ready`
- 플롯 Gate: 잠금
- 원고: 잠금
