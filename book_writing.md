# 《팔대신병록》 집필·설계 최상위 정책

상태: `canon-policy / setting-bible-finalized`
역할: 저장소 최상위 운영 정책
총괄: PM 오케스트레이터

이 문서는 《팔대신병록》 프로젝트의 모든 설계·검증·집필 작업보다 우선한다.

## 1. 기본 원칙

- 설정집·인물·플롯 설계를 본문보다 먼저 완료한다.
- 아이디어·후보·참고안은 정본과 분리한다.
- 중요한 설정은 복수안·전문가 감사·사용자 Human Edit를 거친다.
- 역사·연표·거리·가격·소유권·인물 관계를 장부로 검증한다.
- `canon` 변경은 영향 범위와 재감사를 기록한다.
- 원고는 인물 Gate와 플롯 Gate 통과 전까지 잠근다.

## 2. 공식 파이프라인

`Draft → Validate → Human Edit → Finalize`

현재 설정집은 네 단계를 모두 통과했다.

## 3. 문서 위계

1. 최신 사용자 결정
2. `book_writing.md`
3. `design/SETTING-BIBLE-v2.0-MASTER.md`
4. `operations/HUMAN-EDIT-APPROVAL-DECISION-v1.md`
5. `operations/SETTING-BIBLE-CANON-MANIFEST-v1.md`
6. `lore_index.md`
7. S1~S7 정본 유효 모듈
8. 감사·장부
9. `원고_index.md`
10. 과거 플롯·장면·브레인스토밍 참고안

과거 설정집 v1.0, 심소연·사문경 확정안, 6부·60장 구조와 장면 카드는 정본 기준이 아니다.

## 4. 설정집 정본 상태

- 상태: `canon / setting-bible-master`
- 상위 설정 누락: 0
- 치명적 충돌: 0
- 사용자 Human Edit: 승인 완료
- 정본 승격: 완료

승인 기준:

1. 저강도 확정형 기환
2. 익숙한 명문 35~45%, 고유 기능 세력 55~65%
3. 팔대신병 현재 공통명 + 지역 이형명·원기록명
4. 쇄조도를 반쯤 알려진 재난으로 운용
5. 역사 골격 엄격 + 장르 통칭 가독성 우선

## 5. 정식 작업 용어

- 진원표국
- 검표수
- 인계표사
- 표사
- 표두
- 금의위 남방 회수별조

지명:

- 쇄조도
- 쇄조도 이관 참변
- 쇄조 밀약
- 쇄조합증

## 6. 설정과 후속 창작의 경계

정본에 포함:

- 세계·역사·권역·국가·강호·경제·생활
- 세력·조직·무공·병장기·의술·기환
- 기록·명성·가격·권리·보관 장부
- 팔대신병과 쇄조도 원사건의 세계사적 골격

후속 Gate로 이관:

- 특정 주인공·동료·적대자
- 구체적 결말·사건 순서
- 장면 카드·대사·연출·원고
- 실제 고수 순위·대회 우승자·세부 도감

## 7. 원고·플롯 잠금

현재 금지:

- 기존 6부·60장 플롯의 자동 복원
- 특정 주인공안의 자동 채택
- 새 장면 카드
- 대사·문체·연출
- 장편 초고

기존 플롯·장면 자료는 `deferred-reference`로만 사용한다.

## 8. 다음 Gate

설정집 다음 단계는 **인물·관계 설계 Gate**다.

개방 시:

1. 범위 동결
2. 주인공 구조 최소 3안 비교
3. 사용할 권역·세력·신병 선택
4. 핵심 인물·세대·관계 설계
5. 전문가 감사
6. 사용자 Human Edit

인물 Gate 뒤에만 결말·플롯 Gate를 연다.
