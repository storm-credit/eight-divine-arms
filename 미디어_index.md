# 《팔대신병록》 미디어 인덱스

상태: `planning-only`
상위 정책: [`book_writing.md`](book_writing.md)
설정 허브: [`lore_index.md`](lore_index.md)
생성 규칙: [`미디어_생성.md`](미디어_생성.md)

## 목적

소설 설계와 파생 미디어 제작을 분리한다. 이미지·영상·음악·홍보 문구는 정본 설정을 참고하지만 정본을 새로 만들 권한은 없다.

## 향후 범위

- 인물 콘셉트 아트
- 지역·문파·신병 시각자료
- 지도와 세력 관계도
- 표지·로고·타이포그래피
- 티저·트레일러
- 음악·효과음 콘셉트
- 소개문·홍보 카피

## 상태 규칙

- `concept`: 자유 시안
- `lore-checked`: 설정 검증 완료
- `approved`: 사용자 승인
- `deprecated`: 설정 변경으로 폐기

## 정본 관계

- 미디어 시안에서 새로 생긴 설정은 자동 정본이 아니다.
- 설정과 다른 시각적 표현은 `concept`로만 유지한다.
- 인물 외형·신병 형태·문파 문양은 Gate 2~3 승인 뒤 확정한다.

## 현재 잠금

다음은 아직 생성하지 않는다.

- 팔대신병 최종 디자인
- 주인공 최종 외형
- 핵심 세력 문양
- 공식 지도

상위 설계가 확정되기 전에는 레퍼런스 수집과 무드보드만 허용한다.
