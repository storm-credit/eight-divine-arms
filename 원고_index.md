# 현재 결정

정식 원고·장면 카드·대사·문체 작업은 **설정집 완성 뒤**로 잠근다.

방금 시작했던 제1부 초고 골격은 삭제했다.

기존 문서는 다음 상태로 보존한다.

- 전체 시놉시스: `deferred-reference`
- 6부 개요: `deferred-reference`
- 60장 개요: `premature-detail / deferred-reference`
- 60장 장면 카드: `premature-detail / deferred-reference`

이 문서들은 나중에 살을 붙일 때 참고할 후보이며 현재 설정 정본의 근거가 아니다.
