# 《팔대신병록》 원고 인덱스

상태: `locked-until-gate-5`
상위 정책: [`book_writing.md`](book_writing.md)
설정 허브: [`lore_index.md`](lore_index.md)

## 현재 원칙

정식 장편 원고 집필은 Gate 5 통과 뒤 시작한다.

현재 허용:

- 문체 시험
- 장면 구조 실험
- 전투 설계 예제
- 대사 톤 테스트

현재 금지:

- 미확정 설정을 정본처럼 사용하는 본문
- 전체 플롯 없이 장편 1화부터 연속 집필
- 시험 장면을 정식 원고로 자동 승격

## 공식 집필 파이프라인

`Draft → Validate → Human Edit → Finalize`

### Draft

장면 목표·갈등·결과를 갖춘 초고.

### Validate

- 설정 정합성
- 복선 장부
- 연표·거리·부상
- 인물 욕망과 변화
- 전투 목적·지형·비용

### Human Edit

사용자의 감정·톤·리듬 수정.

### Finalize

정본 설정과 장부를 동기화한 최종 원고.
