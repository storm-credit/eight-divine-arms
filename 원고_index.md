## 현재 결정

정식 원고·장면 카드·대사·문체 작업은 사용자 Human Edit 뒤로 잠근다.

- 실제 초고: 삭제·중단 상태
- 전체 시놉시스: `deferred-reference`
- 6부 개요: `deferred-reference`
- 60장 개요: `premature-detail / deferred-reference`
- 장면 카드: `premature-detail / deferred-reference`

이 문서들은 나중에 살을 붙일 때 참고할 후보이며 현재 설정 정본의 근거가 아니다.

## 원고 재개 조건

다음 뒤에만 원고 설계를 재개한다.

1. 사용자가 통합 설정집의 상위 방향을 검토
2. 수정 항목의 영향 모듈 재감사
3. 설정집을 `canon`으로 승격
4. 그 뒤 인물·플롯을 새로 설계하거나 기존 참고안을 재평가

설정집 승인 전에는 새 초고·장면 카드·대사·연출을 작성하지 않는다.
