# 정식 작업 용어

사용자 결정에 따라 원고에서는 다음을 사용한다.

- 진원표국
- 검표수
- 인계표사
- 표사
- 표두
- 금의위 남방 회수별조

폐기된 검토안:

- 진원행
- 검화수
- 인계호송인
- 호송인·압운인
- 호송두·행관
- 남방 회수차견

지명은 다음을 사용한다.

- 쇄조도
- 쇄조도 이관 참변
- 쇄조 밀약
- 쇄조합증

핵심 원칙:

- 기본 시점은 심소연 제한시점.
- 잔흔은 사실 설명이 아니라 몸의 감각과 오인 가능성으로 표현.
- 전투는 목표·지형·상성·비용·부상·결과를 가짐.
- 한 장면의 새 핵심 개념은 최대 두 개.
- 신병·문서·권리 설명은 반드시 인물의 선택과 비용 안에서 드러냄.

# 공식 원고 파이프라인

`Draft → Validate → Human Edit → Finalize`

## Draft

- 장면 카드에 따라 부별 초고 작성
- 장면 사이의 호흡·감각·대사·내면을 확장

## Validate

- 설정 정합성
- 복선·가짜 가설
- 연표·거리·부상
- 신병 위치·권리
- 인물 지식·시점
- 전투 목적·지형·비용

## Human Edit

- 사용자 감정·톤·리듬·취향 조정

## Finalize

- 정본 설정·장부·원고 동기화
