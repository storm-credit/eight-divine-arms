# 공식 원고 파이프라인

`Draft → Validate → Human Edit → Finalize`

## 1. 장별 개요

각 장에 기록한다.

- 시점 인물
- 시각·장소
- 즉시 목표
- 갈등·반대자
- 핵심 단서
- 전투·협상·회복 기능
- 인물 관계 변화
- 부상·소유·지식 장부 변화
- 장 종료 전환점

## 2. 장면 카드

장면 목표·저항·선택·결과·비용을 작성한다.

## 3. 초고

장면 카드가 준비된 장부터 작성한다.

## 4. 검증

- 설정 정합성
- 복선·가짜 가설
- 연표·거리·부상
- 신병 위치·권리
- 인물 지식·시점
- 전투 목적·지형·비용

## 5. Human Edit

사용자가 감정·톤·리듬·취향을 조정한다.

## 6. Finalize

정본 설정과 모든 장부를 동기화한다.

---

# 장별 개요 전 검증

- 태주·금릉·중원 경계·서남 실제 경로
- 계절·풍향·수운
- 금의위 차견의 직급·문서
- 진원행·창해 상업망 용어
- 해문합증 문서의 작품 내 명칭

이 항목은 필요한 부를 상세화할 때 검증하며 전체 시놉시스를 막지 않는다.

---

# 현재 잠금

아직 하지 않는다.

- 장별 개요 없이 1화부터 장기 연속 집필
- 미확정 이름·경로를 정본처럼 사용
- 시험 장면을 자동으로 공식 원고 승격
- 장면 카드 없이 복잡한 전투·재심 장면 집필
