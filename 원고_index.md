# 최신 지명·용어

새 원고 설계는 다음을 사용한다.

- 해문도 → 쇄조도
- 해문합증 → 쇄조합증
- 진원표국 → 진원행
- 검표수 → 검화수
- 표사 → 호송인·압운인
- 표두 → 호송두·행관
- 남방 회수별조 → 남방 회수차견·금의위 차견 일행

이전 문서의 옛 표현은 레거시 작업명이다.

# 공식 원고 파이프라인

`Draft → Validate → Human Edit → Finalize`

## 1. 장별 개요 — 완료

각 장에 다음이 배치됐다.

- 시점 인물
- 장소·즉시 목표
- 갈등·반대자
- 핵심 단서
- 전투·협상·회복 기능
- 관계 변화
- 장 종료 전환점

## 2. 장면 카드 — 다음 단계

장면마다:

- 진입 상태
- 목표
- 저항
- 선택
- 결과
- 비용
- 장부 갱신

을 작성한다.

먼저 제1부 8장을 묶음 검증한 뒤 나머지 부로 확장한다.

## 3. 초고

장면 카드가 준비되고 검증된 장부터 작성한다.

## 4. 검증

- 설정 정합성
- 복선·가짜 가설
- 연표·거리·부상
- 신병 위치·권리
- 인물 지식·시점
- 전투 목적·지형·비용

## 5. Human Edit

사용자가 감정·톤·리듬·취향을 조정한다.

## 6. Finalize

정본 설정과 모든 장부를 동기화한다.

---

# 장면 카드 전 검증

각 장에 필요한 범위에서 순차 확인한다.

- 태주·금릉·중원 경계·서남 실제 경로
- 계절·풍향·수운
- 금의위 차견의 직급·문서
- 진원행·창해 상업망 용어
- 쇄조합증 문서의 작품 내 한자 용어

이 항목은 원고 구조를 막지 않는다.

---

# 현재 잠금

아직 하지 않는다.

- 장면 카드 없이 1화부터 장기 연속 집필
- 미확정 지명·직급을 정본처럼 사용
- 시험 장면을 자동으로 공식 원고 승격
- 장면 카드 없이 복잡한 전투·재심 장면 집필
