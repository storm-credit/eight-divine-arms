# 《팔대신병록》 최초 여덟 당사자 v0.1

상태: `provisional-canon`
관문: Gate 3
작업 패킷: WP-302
총괄: PM 오케스트레이터

인물의 최종 이름은 역사·언어 검증과 현재 계보 설계 뒤 정한다. 현재는 `O1~O8` 코드와 작업 호칭을 사용한다.

## 1. 공통 원칙

최초 여덟은 참변 전부터 한 팀이 아니었다.

- 서로 다른 명령과 책임으로 해문도에 옴
- 일부는 적대하고 일부는 거래 관계
- 모두 같은 수준의 고수가 아님
- 각자 자기 전문 영역 밖에서는 다른 사람에게 의존
- 사건의 비밀 목적·배신·구조·증거 처리 중 둘 이상을 직접 경험
- 적어도 한 사람을 살리고 다른 사람에게 구조됨
- 자기 과오 때문에 진실을 즉시 전부 공개할 수 없음

현재 대문파와 세가가 여덟의 유산을 일대일로 계승하지 않는다.

## 2. O1 — 명 측 이관 무관

작업 호칭: `수항관`

사회적 위치:
- 새 왕조의 지방 군무 실무자
- 항복 선단의 무장·인원·병기 수량을 확인하는 책임자
- 중앙 고위 인물이 아니라 현장 명령과 결과 사이에 낀 중간 지휘관

공개 임무:
- 병기와 선박 인계
- 군인·선원 분류
- 경비와 질서 유지

비밀 임무:
- 특정 장인·항해자·지휘관을 별도 구금
- 항로·병기 장부를 다른 관부보다 먼저 확보

병기 계통:
- 장창·군용 장병

전문 능력:
- 군진과 경비 배치
- 공문·군령 해독
- 다수 인력 통제

과오:
- 비밀 구금 명령의 정당성을 의심하면서도 너무 늦게 거부
- 민간 구역 봉쇄를 즉시 해제하지 못함
- 현장의 경고보다 명령 전달 체계를 우선

결정적 행동:
- 화재 뒤 자신의 장창으로 봉쇄문과 지지대를 부수고 민간 구역을 엶
- 부하에게 철수를 명령해 공식 회수보다 구조를 우선

주요 목격:
- 서로 충돌하는 명령과 인장
- 명 측 내부의 지휘 혼선
- 민간 구역 봉쇄 해제 과정

사각지대:
- 최초 장부 소각
- 외항의 탈출 신호와 선박 충돌
- 상업망의 실제 배후

구조 관계:
- O5와 민간 생존자를 구함
- O8이 연결한 반대 명령과 증거 덕분에 현장에서 즉결 구금·처형되는 것을 피함

잔흔 방향 후보:
- 무게·밀기·버티기·문을 여는 동작
- 명령에 복종하려는 의지와 거부하는 순간의 충돌

## 3. O2 — 항복 선단의 부장·선주

작업 호칭: `귀항선주`

사회적 위치:
- 방국진 계열 해상망에 속했던 선박 가문의 과부 또는 후계 선주 우선안
- 군사 지휘와 상업 운송을 모두 경험
- 항복 뒤 가족·선원·재산을 지켜야 함

공개 임무:
- 선박과 무장 인계
- 선원 명부 제출
- 항복 조건에 따른 인원 이동

비밀 임무:
- 사면받지 못할 선원·가족을 소형선으로 탈출
- 선단의 핵심 해도와 일부 병기를 숨김

병기 계통:
- 선상 도·장도 또는 절단과 갈고리를 겸한 선상 병기

전문 능력:
- 조류·풍향·선박 운용
- 선원 지휘
- 근접 선상 전투

과오:
- 초기에 민간인보다 선박·해도·무장을 먼저 지키려 함
- 위조 명부와 탈출 계획으로 경비의 의심과 봉쇄를 키움
- 잔존 강경파가 병기를 빼돌리는 것을 완전히 막지 못함

결정적 행동:
- 불붙은 선박의 계류삭과 돛대를 잘라 화재 확산을 막음
- 자신의 배를 돌려 의무소와 민간 부두를 화염·충돌에서 가림

주요 목격:
- 항구의 무단 출항과 신호 충돌
- 항복 세력 내부의 생존파·강경파 분열
- 불붙은 선박과 바깥 포구의 움직임

사각지대:
- 문서고 내부의 소각 지시
- 명 측 비밀 구금 명령의 원문

구조 관계:
- O7이 쏜 줄과 신호 덕분에 불붙은 선박에서 탈출·조종 가능
- O6의 의무소와 부상자를 보호함

잔흔 방향 후보:
- 절단·회수·흔들리는 발판의 균형
- 가족과 선박 중 무엇을 먼저 지킬지에 대한 죄책감

## 4. O3 — 해문도 선장·수리 책임자

작업 호칭: `도장선`

사회적 위치:
- 해문도와 인근 포구의 선박 수리·창고·도선 노동을 관리
- 국가·항복 선단·상단 모두에게 고용된 적이 있는 지역 기술자
- 중앙 정치보다 섬 사람과 생업을 우선

공개 임무:
- 인계 선박의 상태 점검
- 창고·부두·계류 시설 유지
- 화약과 기름의 안전 분리

비밀 임무:
- 지역민과 선주가 사용하던 썰물 통로·숨은 정박지 은폐
- 밀무에 쓰인 수리 장부 일부 감춤

병기 계통:
- 갈고리 장병·구겸·선상 공구를 겸한 병기

전문 능력:
- 선박 구조와 목재·밧줄·도르래
- 조수와 암초 길
- 화재 확산과 창고 구조

과오:
- 돈과 지역 생존을 위해 비공식 창고·화약 은닉을 묵인
- 자신이 아는 통로를 관부와 선단 양쪽에 다르게 설명
- 화재 위험을 알면서도 모든 창고를 비우지 못함

결정적 행동:
- 막힌 배수·수리 통로를 열어 연기와 인파의 탈출로를 만듦
- 작은 수리선을 띄워 감시대와 외항 사이를 연결

주요 목격:
- 병기고·화약고의 상태
- 불이 번진 경로와 시설 파손
- 숨은 통로를 사용한 사람

사각지대:
- 비밀 명령의 정치적 목적
- 장부 내용과 계보

구조 관계:
- O4가 건넨 창고 열쇠와 장부 표식으로 봉쇄된 시설에 접근
- O7을 감시대에서 구조할 수리선을 보냄

잔흔 방향 후보:
- 걸기·당기기·버티기·구조물의 저항
- 섬을 지키기 위해 불법을 묵인한 책임

## 5. O4 — 강남 상업망 장부 실무자

작업 호칭: `철척장계`

사회적 위치:
- 강남 상단·전장과 동절강 선단 사이의 채무·창고·화물 장부를 관리
- 상업 가문 출신 여성 실무자 우선안
- 글과 숫자·인장에 강하지만 공개 무인 평판은 높지 않음

공개 임무:
- 선박·화물·채무·압수품 대조
- 소유권과 배상 자료 정리

비밀 임무:
- 후원 상단의 이중 거래와 밀무 투자 기록 제거
- 특정 가문과 관원의 이름을 장부에서 보호

병기 계통:
- 철척·단병·문서함 방어 도구

전문 능력:
- 장부·어음·인장·필체
- 물건과 사람의 이동 추적
- 협상과 거래 구조

과오:
- 상부 명령에 따라 장부 일부를 먼저 소각
- 사면과 가족 관계를 증명할 인명 기록까지 잃게 할 뻔함
- 누가 살아남을 가치가 있는지 거래 가치로 판단

결정적 행동:
- 화재 속에서 재산 장부 대신 생존자·사면·인계 명부를 선별해 반출
- 장부의 일부를 서로 다른 사람에게 나눠 줌

주요 목격:
- 상업망과 관원·선단의 이중 거래
- 위조 명부와 장부 소각
- 누가 어떤 기록을 노렸는가

사각지대:
- 외항의 첫 충돌
- 화약고의 물리적 발화 과정

구조 관계:
- O5가 연기 속에서 구조
- O3에게 창고 열쇠와 숨은 통로 표식을 제공

잔흔 방향 후보:
- 짧은 막기·누르기·손가락과 손목의 정밀 동작
- 지워야 할 이름과 살려야 할 이름을 고르는 죄책감

## 6. O5 — 불문 구휼 수행자

작업 호칭: `해등승`

사회적 위치:
- 해안 사찰과 구휼망을 오가며 부상자·가족·시신을 돌보는 수행자
- 구파일방 핵심 문파의 공식 대표가 아니라 지역 불문 네트워크 소속

공개 임무:
- 부상자와 가족 보호
- 식량·물·숙박 배분
- 사망자와 이주자의 명단 보조

비밀 임무:
- 보복과 강제 징발을 피하려는 선원·가족 일부를 수행자·난민 명부에 숨김

병기 계통:
- 선장·봉·호신 장병

전문 능력:
- 군중 안정과 구휼
- 비살상 제압
- 여러 지역 사찰의 보증

과오:
- 보호를 위해 인원 수와 신분을 속여 공식 분류를 더 혼란스럽게 함
- 특정 강경 인물을 민간인 무리에 숨겼을 가능성
- 증거보다 사람을 우선해 일부 문서 훼손을 묵인

결정적 행동:
- 민간 구역에서 줄과 대열을 만들어 연기 속 탈출을 조직
- 봉으로 무너지는 지지대를 버티고 통로를 유지

주요 목격:
- 민간인 사망과 구조
- 봉쇄문이 닫힌 뒤의 상황
- 구금·분류 과정의 폭력

사각지대:
- 장부의 전체 내용
- 항구·문서고의 첫 충돌
- 누가 봉쇄를 최초 명령했는지

구조 관계:
- O1이 봉쇄문을 열어 O5와 민간인을 구함
- O4를 연기 속에서 끌어냄

잔흔 방향 후보:
- 지지·밀기·대열을 유지하는 리듬
- 사람을 살리기 위해 진실과 신분을 숨긴 책임

## 7. O6 — 도문계 의원·검시인

작업 호칭: `현침의`

사회적 위치:
- 도문 호흡·양생을 익혔으나 특정 대종문의 핵심 제자는 아닌 의원
- 항구·선박의 외상·중독·익사·시신 검시에 경험이 있는 여성 의원 우선안

공개 임무:
- 부상자 분류와 치료
- 약재·독물·화상·익사 판별
- 사망 원인의 기초 기록

비밀 임무:
- 한쪽 보증인의 부탁으로 구금자에게 쓰인 약물과 시신 흔적을 별도 기록
- 비공식 탈출을 돕기 위해 일부 경비를 침·약물로 무력화

병기 계통:
- 의술용 단도·침구·짧은 제압 도구

전문 능력:
- 외상·독·호흡·맥진
- 시신과 현장 흔적
- 통증·경련·공황 관리

과오:
- 경비 무력화가 독 공격이라는 소문을 일으켜 공포와 선제 공격을 키움
- 누구를 먼저 치료할지 선택하며 일부 부상자를 포기
- 관부와 항복 세력 양쪽에 완전한 기록을 주지 않음

결정적 행동:
- 연기·독·공황의 증상을 구분해 의무소의 집단 혼란을 막음
- O8을 소생·안정시키고 그가 문서를 연결할 시간을 벌어 줌

주요 목격:
- 시신과 부상의 방향
- 약물·연기·화상의 차이
- 일부 사람이 화재 전 이미 무력화된 사실

사각지대:
- 정치적 명령과 상업 거래
- 외항 신호의 의미

구조 관계:
- O2가 불붙은 선박을 돌려 의무소를 보호
- O8을 살림

잔흔 방향 후보:
- 맥박·호흡·정밀한 압박과 절개
- 살릴 사람을 골라야 했던 죄책감과 냉정

## 8. O7 — 원계 해운망 출신 다언어 통역·궁수

작업 호칭: `해로역궁`

사회적 위치:
- 원대 해운·교역망을 거쳐 동절강에 정착한 다언어 가정 출신
- 몽골어·한어·항구 교역어 중 둘 이상을 다루는 통역·선원·궁수
- 특정 민족 전체를 대표하지 않는 혼합 해상 실무자

공개 임무:
- 외지 선원·상인·포로·가족의 신원과 계약 통역
- 외항 감시와 신호 해석

비밀 임무:
- 공식 명단에 없는 외국계 선원과 한 가족의 탈출을 도움
- 원대 해운망의 옛 표식과 항로 일부를 숨김

병기 계통:
- 각궁·선상 궁술

전문 능력:
- 원거리 감시와 신호
- 다언어 명령·계약 번역
- 외항과 조류 관찰

과오:
- 서로 양립하지 않는 명령을 의도적으로 모호하게 번역
- 한 경고 신호가 다른 선박에는 탈출 신호로 전달됨
- 자신이 보호하려는 사람을 위해 전체 상황을 늦게 보고

결정적 행동:
- 감시대에서 구조선과 불붙은 선박 사이에 줄을 쏘아 연결
- 폭풍 속 안전한 암초 길과 위험한 출항로를 신호로 구분

주요 목격:
- 외항의 첫 무단 출항
- 서로 다른 언어로 전달된 명령
- 바깥에서 섬으로 접근하거나 떠난 선박

사각지대:
- 내륙 장부와 명 측 계보
- 민간 구역 내부

구조 관계:
- O3의 수리선이 감시대에 도달해 구조
- O2의 선박에 구조선을 연결

잔흔 방향 후보:
- 시위의 긴장·바람·거리·소리
- 말 하나의 번역이 생사를 바꾼 책임

## 9. O8 — 강호 중재망의 문서 사자

작업 호칭: `봉서객`

사회적 위치:
- 강남·절강의 여러 문파·세가·상단 사이에서 서신·인장·보증인을 연결하는 무인 사자
- 후대 무림맹과 유사한 보증망의 전신과 관계하지만 공식 중앙 대표는 아님

공개 임무:
- 항복·보증·사면 관련 서신과 인장을 당사자에게 전달
- 무인과 상단 보증인의 입회를 연결

비밀 임무:
- 서로 다른 후원자의 밀봉 문서를 따로 보관
- 한 문파의 책임을 감추기 위해 전달 시점을 조절

병기 계통:
- 직검 또는 포승·쇄를 겸한 제압 병기

전문 능력:
- 사람과 문서의 보증 관계
- 빠른 이동과 제한 비무
- 봉인·인장·전달 경로

과오:
- 봉인을 무단으로 열고도 즉시 공개하지 않음
- 한 경고를 늦게 전달해 구금과 탈출 준비가 동시에 진행되게 함
- 자신이 속한 보증망의 체면을 보호하려 함

결정적 행동:
- 서로 충돌하는 명령과 인장을 연결해 O1에게 비밀 명령이 유일한 합법 명령이 아님을 증명
- 구금된 O1을 빼내고 구조 지휘가 계속되게 함

주요 목격:
- 여러 명령·인장·보증인의 관계
- 문서고와 지휘소 사이의 이동
- 참변 뒤 각 당사자가 숨기려 한 기록

사각지대:
- 최초 발화와 항구의 세부 전투
- 시신·독의 정확한 원인

구조 관계:
- O6의 치료로 생존
- O1의 구금·처형을 막음

잔흔 방향 후보:
- 발도·제압·봉인을 끊거나 지키는 손동작
- 진실을 전달할 때와 숨길 때를 스스로 결정한 오만

## 10. 상호 구조망

설계상 구조 인과:

1. O1이 봉쇄문을 열어 O5와 민간인을 구한다.
2. O5가 연기 속에서 O4를 구조한다.
3. O4가 O3에게 창고 열쇠와 숨은 통로 표식을 건넨다.
4. O3가 수리선을 보내 O7을 감시대에서 구조한다.
5. O7이 구조 줄과 신호로 O2의 불붙은 선박을 연결한다.
6. O2가 선박을 돌려 O6의 의무소를 보호한다.
7. O6가 O8을 살려 문서 연결을 가능하게 한다.
8. O8이 반대 명령과 보증 증거를 연결해 O1의 구금·처형을 막는다.

이 구조는 완벽한 운명적 원을 본문에서 강조하기 위한 것이 아니다.

목적:
- 누구도 혼자 참변을 해결하지 못함
- 각자의 생존과 책임이 타인의 선택에 빚짐
- 배신하더라도 상대의 증언을 완전히 지울 수 없음

실제 장면에서는 실패·지연·민간 희생과 불완전한 구조를 포함한다.

## 14. 후대 계보의 분열 원칙

한 당사자의 유산은 여러 갈래로 나뉠 수 있다.

- 병기 보관자
- 가족과 혈통
- 제자와 무공
- 문서 보유자
- 구조된 민간 공동체
- 상단·사찰·관부의 후원망

따라서:
- 현재 세력이 병기는 가졌지만 기록은 없을 수 있음
- 혈통은 있지만 사용권이 없을 수 있음
- 다른 세력이 실제 책임을 수행했을 수 있음
- 한 현재 문파에 두 당사자의 유산이 합쳐질 수 있음
- 현재의 정파·사파 구분과 최초 당사자의 선택은 일치하지 않을 수 있음
