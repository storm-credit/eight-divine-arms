# 《팔대신병록》 진법·풍수·점복 운영 체계 v0.1

상태: `provisional-canon / S4-02`
설정 분야: D12 기환 상한, D13 진법·풍수·점복, D10 기관·군진
총괄: PM 오케스트레이터

## 1. 설계 원칙

진법·풍수·점복은 서로 다른 기능을 가진다.

- 진법: 사람·지형·신호·도구를 배치해 행동과 인식을 통제
- 풍수: 지형·물·바람·방위·장례·거주를 해석하고 설계하는 체계
- 점복: 불확실한 상황에서 징후·시간·선택을 해석하는 의례·판단 기술

셋 모두 실제 관찰·심리·기관·사회 권위와 희귀한 기환성이 섞일 수 있다.

## 2. 진법의 네 층

### 전술 진법

- 대형·간격·역할·교대
- 북·기·횃불·호각 신호
- 군진·호송·문파 합격술
- 초자연 효과 없음

### 지형 진법

- 문·벽·수로·절벽·수풀·안개·빛
- 길 선택·시야·소리·퇴로 통제
- 사전 답사와 설치 필요

### 기관 진법

- 함정·낙문·쇠뇌·연기·거울·종·줄
- 장인·재료·유지 관리 필요
- 파손·습기·화재·오작동 가능

### 기환 진법

- 오래된 장소·유물·잔념·의례가 결합
- 감각 혼란·반복 동작·공포·방향 상실을 강화
- 희귀하며 범위·시간·매개체가 제한
- 공간 자체를 무한 변형하거나 다른 차원으로 보내지 않음

## 3. 진법의 필수 요소

- 목적
- 경계와 범위
- 인원·역할
- 지형·시설
- 신호
- 중심점·퇴로
- 유지 시간
- 실패·해제 조건

`진안` 하나를 부수면 모든 진법이 폭발한다는 공식을 쓰지 않는다.

## 4. 진법 운용자

- 군문 지휘관·교관
- 문파 장로·진법 분과
- 공병·장인
- 선장·나루·성문 책임자
- 의례 전문가
- 지역 안내자

제갈세가형 전략 가문이 모든 진법을 독점하지 않는다.

## 5. 진법의 강점과 한계

강점:

- 약한 개인을 역할 분담으로 강화
- 적의 정보·거리·퇴로 제한
- 시간 지연·보호·체포
- 지형과 장비 활용

한계:

- 사전 준비
- 신호·지휘
- 구성원 훈련
- 지형 의존
- 내부 배신
- 예상 밖 화재·홍수·민간인

고수가 구조를 파악하면 일부 약점을 노릴 수 있지만 준비된 다층 진법을 한 초식으로 해제하지 않는다.

## 6. 미로·환진의 표현

환진처럼 보이는 원인:

- 안개·연기·거울·등불
- 반복 구조·문양
- 소리·향·약물
- 수면 부족·공포
- 지형의 실제 곡선
- 기환 잔흔의 방향 강박

판별:

- 동료와 신호
- 줄·표식·걸음 기록
- 바람·물 흐름
- 시간·별·해 위치
- 약물·향 검사

완전한 집단 환각을 손짓만으로 만들지 않는다.

## 7. 풍수의 영역

### 거주·도시

- 물·배수·바람·햇빛
- 도로·문·우물·화재
- 묘지·시장·관청 배치

### 장례·묘지

- 토질·수맥·홍수·침식
- 조상·종족·정치 상징
- 무덤 도굴·시신 보존과 연결

### 군사·강호

- 관문·수로·고지·퇴로
- 산문·진법·은거지
- 지역민이 축적한 환경 지식

풍수는 현대 지질학도 순수 미신도 아니다.

## 8. 지맥·용맥의 정의

작품에서 지맥·용맥은 세 층의 언어다.

1. 실제 지형·수계·바람·토질
2. 정치·종교가 부여한 상징
3. 드물게 장소 잔흔과 기환 공명이 집중되는 환경

지맥을 무한 내공 발전소나 땅속 마력선으로 만들지 않는다.

## 9. 풍수의 실제 효과

확실한 효과:

- 배수·환기·식수·농사·방어
- 길·시장·무덤의 장기 안정
- 공동체의 신뢰와 의례

조건부 기환 효과:

- 특정 장소에서 잔흔 접근 강화·약화
- 수면·감정·방향 감각 변화
- 영초 생육과 연관 가능

불확실:

- 가문의 운명·후손 출세
- 천하 패권의 자동 보증

## 10. 풍수 전문가와 사기

전문가:

- 지관·감여가
- 도사·승려
- 장인·수리 담당
- 지역 노인·농민·선원

사기:

- 묘지·가옥 공포 조장
- 경쟁 가문 흉지 소문
- 숨은 광산·보물을 용맥으로 과장
- 관부·향신의 이주 명분

실제 환경 지식이 일부 맞아 사기 판정이 복잡할 수 있다.

## 11. 점복의 종류

- 역·괘 해석
- 시일·방위 선택
- 꿈·징조
- 별·기상·동물 행동
- 제비·첨·패
- 얼굴·손·기색 관찰
- 군사·항해 징후 판단

모든 점복을 하나의 예언 마법으로 묶지 않는다.

## 12. 점복의 기능

- 결정을 구조화
- 위험·계절·심리 상태 점검
- 공동체 합의와 의례
- 책임 분산 또는 권위 부여
- 정보가 부족할 때 선택의 언어 제공

점복 결과는:

- 모호함
- 해석자 편향
- 사후 합리화
- 정치적 이용

을 가진다.

## 13. 예지의 상한

허용:

- 관찰과 경험에 기반한 높은 확률 예측
- 기환 매개체와 공명해 감각·감정 파편을 얻음
- 꿈·징조가 후대 사건과 유사하게 해석됨

금지:

- 정확한 미래 날짜·대사·승패를 확정
- 운명을 피할 수 없는 객관 법칙
- 점괘가 모든 수사·전략을 대체

## 14. 점복과 정치

- 출정·혼인·장례·입문 날짜
- 지도자 정통성
- 사교·요언 혐의
- 민심·시장 공포

점복가는 권력을 얻을 수 있으나 틀린 예측·매수·요언 처벌 위험을 진다.

## 15. 사방신과 방위 체계

청룡·백호·주작·현무는:

- 방위·계절·색·상징
- 군기·도시·묘지·진법 문양
- 종문·세력의 정치적 상징

으로 널리 사용된다.

실제 거대 신수 종족이 상시 존재하거나 명령에 따라 강림하지 않는다.

희귀 기환 현상이 사방신 이미지로 해석될 수 있으나 문화적 해석과 실재를 분리한다.

## 16. 해제·파훼

진법 파훼 방법:

- 목적·경계 파악
- 신호 차단·위조
- 지형 변경
- 인원 분리·휴식
- 기관 파손·동력 차단
- 주민·장인에게 구조 질문
- 우회·퇴각

무조건 중심 돌 하나를 깨는 것보다 정보·시간·협력이 중요하다.

## 17. 기록과 감정

진법·풍수·점복 기록에는:

- 작성자·계보
- 장소·날짜·기후
- 목적
- 실제 배치
- 관측 결과
- 후대 해석

을 분리한다.

옛 진법도는 당시 장소가 바뀌면 그대로 작동하지 않는다.

## 18. 악마의 변호인 위험

- 진법을 게임 던전 기믹으로 만들지 않는다.
- 풍수를 모든 가문 흥망의 과학으로 확정하지 않는다.
- 점복을 정확한 미래 스포일러로 쓰지 않는다.
- 제갈세가·모산파가 모든 관련 기술을 독점하지 않는다.
- 사방신 상징을 실제 신수 강림 증거로 자동 처리하지 않는다.
- 파훼가 천재 한 명의 한눈 보기로 끝나지 않는다.

## 19. 교차 감사 질문

- 인원·지형·신호·설치가 있는가
- 기환 효과가 공통 상한법을 통과하는가
- 기관·약물·심리 가능성을 검토했는가
- 장소 변화와 유지 비용을 반영했는가
- 점복 결과가 선택과 책임을 없애지 않는가
- 풍수의 환경 효과와 운명 주장을 구분했는가
