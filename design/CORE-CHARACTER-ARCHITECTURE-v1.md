# 가족·스승 축

## F1 심가·진원표국 절강 분점

개별 인물은 후속 상세화하되 다음 역할을 분리한다.

- 가업 장인: 소연의 몸과 도구 감각의 근원
- 장부 관리 가족: 가업 신용과 채무를 실제로 책임
- 분점 표두: 소연을 보호하면서도 분점 생존 때문에 침묵을 요구
- 젊은 견습: 소연의 선택을 영웅화하거나 원망할 수 있음
- 과거 책임 피해자: 소연이 기록을 고친 결과 실제 손실을 입은 사람
