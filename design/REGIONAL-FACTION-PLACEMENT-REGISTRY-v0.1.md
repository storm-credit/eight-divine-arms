# 《팔대신병록》 권역별 세력 배치 등록부 v0.1

상태: `provisional-canon / S5-04`
설정 분야: D08 세력, D03 권역, D21 외부 무림
총괄: PM 오케스트레이터

## 1. 배치 원칙

- A등급 세력은 작품 전체 관리용 14개이며 본문 집합 명칭이 아니다.
- 권역 하나에 상주·직접 영향 A등급은 최대 3개를 기본으로 한다.
- 사건 하나의 직접 핵심 A등급은 최대 5개다.
- B등급은 지역 사회와 기능을 실제로 움직인다.
- C등급은 이름과 한 기능만 가지며 사용 전 B등급 승격 감사를 한다.
- 전국 조직도 지역 지부마다 내부 파벌·재정·관부 관계가 다르다.

## 3. A등급의 지역화

### 무림맹

- 본단·중앙 기록
- 강남 회합처
- 지역 회원 회의

지역 회의가 본단 명령을 그대로 집행하지 않을 수 있다.

### 개방

- 분타별 구휼·정보 우선순위 차이
- 연해 분타는 난파·실종·염민
- 강남 분타는 도시 빈민·장례·시장
- 북방 분타는 군호 가족·도주민

### 하오문

- 중앙 문주보다 지역 향주·중개인 실권
- 부두·유곽·도박·장례·마방 등 생업 차이

### 사도련

- 상설 수도 없음 또는 이동 회합
- 회원마다 계약 질서 수용 범위 차이
- 녹림·수로채가 자동 회원이 아님

### 천마신교

- 본교 요새·교단
- 대상로 제휴 거점
- 중원 연락처는 교단 전체 대표가 아님

## 4. 동남 연해 B등급 후보

### 보타암

- 유형: 해안 불문 여성 종교 공동체
- 기능: 구휼·의료·장례·난파자·잔흔 안정
- 수입: 시주·약재·선주 후원·토지
- 갈등: 중앙 불문 인정·여성 지도권·해상민 보호
- 상태: B 우선 후보

### 청계 선박공방연합

- 유형: 조선·수리·밧줄·기중 장인 결사
- 기능: 선박 감정·수리·난파 구조·D3 지역 사용 기록
- 갈등: 관부 징발·상단 채무·기술 계승
- 상태: B 우선 후보

### 연해 염민자위회

- 유형: 염전·어민·촌락 방어·구휼
- 기능: 조수·해안길·밀무 오인·피난
- 상태: B 또는 C 묶음

### 창룡표국

- 유형: 수운·해상 특화 표국
- 기능: 선박 호송·나루·수적 협상
- 진원표국과 경쟁·공동 표행

### 연안 수로채 연합

- 유형: 도선·통행세·밀무·구조
- 사도련 일부 회원과 비회원 혼재
- 상태: B 기능 세력

## 5. 강남·남직례 B등급 후보

- 유형: 중립 검계·병기·문서 감정기관

## 12. 구파일방 작업 후보

최종 확정 전 후보군:

- 소림
- 무당
- 화산
- 종남
- 아미
- 청성
- 공동
- 점창
- 곤륜
- 개방

검토 원칙:

- 실제 지리·종교·무협 장르 친숙성
- 권역 대표성
- 기능 중복
- 본편 사용 여부

해남파는 구파일방 진입을 정치 목표로 가질 수 있으나 현재 의석으로 자동 편입하지 않는다.

## 13. 오대세가 작업 후보

우선 후보:

- 남궁
- 하북 팽가
- 사천 당문
- 제갈
- 모용

대체·도전 후보:

- 산동 악가
- 황보가
- 독고계 검가
- 금릉 이씨가
- 진주 연씨가

원칙:

- 무력만이 아니라 재정·지역·지속성·세가권 대표 의무
- 실제 본편에 모두 상세 등장하지 않음
- 오대세가 의석과 유력 팔대가 후보를 구분

## 14. A등급 과밀 감사

직접 작품 엔진에 필요한 A등급:

- 무림맹
- 해남파
- 금의위 남방 회수별조
- 하오문
- 창해상련
- 진원표국
- 만독문

장르 기준점·정치 배경 A등급:

- 소림사
- 무당파
- 개방
- 남궁세가
- 산동악가
- 사도련
- 천마신교

작품 한 부에서 후자 전부를 직접 활성화하지 않는다.

## 15. C등급 생성 규칙

C등급 세력은 다음 형식만 가진다.

- 이름
- 권역
- 조직 유형
- 한 가지 생업·기능
- 누구에게 의존하는가

사용 시:

- 수입
- 지도부
- 민간 책임
- 내부 갈등

을 추가해 B등급으로 승격한다.

## 16. 내부 파벌 의무

A등급은 최소 세 파벌 또는 이해집단을 가진다.

예:

- 본산/지역
- 보수/개혁
- 재정/무공
- 관부 협력/독립
- 공개/비밀

파벌은 선악이 아니라 자원·책임·위험 기준이 다르다.

## 17. 세력 관계 장부 항목

- 공식 동맹·적대
- 경제 의존
- 인적·혼인 관계
- 정보 거래
- 관부 협약
- 신병 권리
- 과거 판정·채무
- 지역 갈등

단순 `우호/적대` 두 칸으로 관리하지 않는다.

## 18. 남은 확정 조건

S6에서:

- 명칭·한자·직함
- 시대착오
- 지역 언어
- 무림대회·서열 제도

를 감사한다.

S7에서:

- 신병과 세력의 직접 관계
- 보관·감정·수리·사용 권리

를 적용한다.

## 19. 악마의 변호인 위험

- 익숙한 성씨·문파를 전부 최고 세력으로 올리지 않는다.
- 지역마다 A등급 중앙 조직 지부만 배치하지 않는다.
- B/C세력을 소모성 악당·피해자로만 쓰지 않는다.
- 구파일방·오대세가를 고정 전투력 명단으로 만들지 않는다.
- 외부 세력을 한 집단으로 묶지 않는다.
- 핵심 14개가 천하의 모든 권력을 독점하지 않는다.
