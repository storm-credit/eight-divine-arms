# 《팔대신병록》 5막·8구간 기능 행렬 v1

상태: `draft / outline-gate`
총괄: PM 오케스트레이터
장 수 작업 기준: 48장, 허용 범위 44~52장

## 1. 전체 시간축

- 시작: 1555년 늦봄~초여름 작업 범위
- 종료: 다음 해 늦여름~가을 작업 범위
- 전체 기간: 약 14~17개월
- 정확한 음력 일자·풍향·역참 시간은 장 단위 Outline 뒤 국소 감사

시간축 원칙:

- 권역 이동은 장 사이 또는 막 사이 시간 경과를 사용
- 중상·감염·잔흔 침식은 다음 구간까지 영향을 남김
- 모든 핵심 인물이 전 구간을 함께 여행하지 않음
- 각 권역의 주도 인물이 현지 연결망을 제공

# 제1막 — 끊어진 인계

## 구간 1 — 조선소 붕괴와 이중 인계

- 장 배분: 1~6장
- 기간: 3~5일
- 권역: 절강 태주부 연해, 허구 조선·선창 거점
- 주시점: 심소연 중심
- 주도 인물: 심소연·한목수
- 압력 인물: 고정문·표국 분점·관부 현장 책임자
- 중심 신병: D3 철조구

### 시작 상태

- 심소연은 표국의 정확한 인계가 사람과 책임을 보호한다고 믿음
- D3는 일반 기중·계류 공구로 장기간 사용
- 노동자 보상·압수선 부품·관부 증거·실종자 유류품의 인계가 같은 장소에서 겹침

### 구간 목표

- 붕괴 현장에서 생존자 구조
- 무엇이 누구에게 인계되기 전 어떤 상태였는지 보존
- 사고·부실 수리·고의 훼손 가능성을 동시에 열어 둠

### 핵심 행동 압력

- 구조를 위해 봉인·계약·압수 명령을 깨야 함
- D3 사용이 심소연 어깨와 감각에 실제 비용을 남김
- 관부 봉쇄가 구조와 증거 보존을 동시에 방해

### 공개 정보

- 부실 수리와 안전 부품 누락
- D3의 비정상적 하중 잔흔
- 복수 인계서의 대상과 수량 불일치

### 숨은 정보

- 안전 부품 제거는 신병 확인을 위한 제한 훼손이었음
- 붕괴 뒤 별도 탈취 계약이 대기 중
- 일부 명부는 지역 보호망이 먼저 회수

### 비가역 변화

- D3 존재가 관부·표국·장인·사설 중개망에 알려짐
- 심소연이 관부와 표국 명령을 동시에 위반
- 부상·보증 제한·권리 분쟁 시작

### 필요한 국소 상세화

- 선창·기중 구조 평면
- 표국·관부 인계서 형식
- 수리 책임자와 고의 훼손 실행자의 기능 카드
- 현장 구조용 무공·병장기

## 구간 2 — 공동 보관 실패

- 장 배분: 7~11장
- 기간: 2~4주
- 권역: 태주 연해의 표국·장인촌·객잔·관부 보관처
- 주시점: 심소연·진소무
- 주도 인물: 진소무·한목수
- 적대 압력: 사문경 중개망·위철산 보호망·고정문
- 중심 대상: D3, 수리 기록, 피해자 명부, 봉인 탁본

### 시작 상태

- D3 현물과 문서가 임시 공동 봉인
- 심소연은 직무 정지·보증 제한 상태
- 현장 생존자와 유가족은 배상·시신·장물 문제로 분열

### 구간 목표

- 공동 보관의 권한과 절차 설정
- 사라진 기록의 방향을 추적
- 진소무와 위철산 보호망의 차이를 확인

### 중심 질문

> 사람을 보호하기 위해 이름을 숨기는 것과, 그 사람의 권리를 지우는 것은 어디서 갈리는가?

### 행동 압력

- 봉인고 침입 또는 화재가 아니라, 서로 다른 주체가 합법·비공식 권리로 자료를 분산 반출
- 유가족 배상 기한과 관부 압수 기한이 동시에 다가옴
- 심소연은 D3 현물을 쫓는 대신 사람·사본·수리 계보 중 무엇을 먼저 지킬지 선택

### 비가역 변화

- 공동 보관이 실패하고 현물·문서·증언이 다른 경로로 분산
- 진소무가 가명망과 쇄조도 후손 공동체의 접점을 발견
- 사문경이 중앙 감정과 신용 복구를 제안
- 위철산은 단순 파괴자가 아니라 현재 공동체 보호자임이 드러남

### 필요한 국소 상세화

- D3 보관함·봉인 방식
- 하오문 지역 경로
- 피해자·유가족 대표 2~3인
- 표국 분점 가족·가업 관계

# 제2막 — 바다의 두 신호

## 구간 3 — 절류도와 해명궁

- 장 배분: 12~17장
- 기간: 6~10주
- 권역: 복건 연안·해남계 선단·류큐 연결 해역
- 주시점: 나해란 중심, 심소연 보조
- 주도 인물: 나해란
- 중심 신병: D2 절류도, D7 해명궁 복원궁·전승궁
- 적대 압력: 장계엄 계열·국가 감정망·해상 계약 세력

### 시작 상태

- D3 자료 일부가 해상 신호·구조 계보를 가리킴
- 국가 기록은 D7 복원궁을 우선 진품으로 인정
- 현지 공동체는 전승궁과 D2의 실제 사용 책임을 중시

### 구간 목표

- 첫 경고·공격·구조 신호의 차이를 복원
- D7의 이중 정통성을 인정할 증거 확보
- 나해란의 통역과 구조 사격 과오를 현재 협상과 연결

### 중심 질문

> 정확한 번역과 당장의 생명 보호가 충돌할 때 누가 의미를 결정하는가?

### 행동 압력

- 구조선·밀무선·약탈선 분류가 관부·선주·피해자마다 다름
- 폭풍·추적·봉쇄 속에서 신호 체계를 실제로 사용해야 함
- D2·D7을 같은 선박에 오래 두지 않고 사용자·문서·부품을 분리 이동

### 비가역 변화

- D7 복원궁과 전승궁의 병존 정통성이 공론화
- 해상 공동체가 쇄조도 합증의 당사자 지위를 확보
- 장계엄은 사건을 군기·밀무·외부 세력 문제로 확대
- 심소연은 문서보다 실사용·자칭·구전 계보가 우선할 수 있음을 인정

### 필요한 국소 상세화

- 선박 유형 3~4종
- 해상 신호 규칙
- D2 현 사용자·보관 공동체
- D7 복원궁 감정기관과 전승궁 계보
- 항해·체류 시간과 계절풍

# 제3막 — 몸과 장부의 값

## 구간 4 — 금릉의 수량과 채무

- 장 배분: 18~23장
- 기간: 4~6주
- 권역: 금릉 상업권·장부고·전장·창고
- 주시점: 심소연·진소무, 마윤경 선택 시점
- 주도 인물: 마윤경·진소무
- 중심 신병: D4 묵형척
- 적대 압력: 사문경의 제한 공개파·채권자·세가·상단

### 시작 상태

- 해상 기록과 D3 수리 문서가 쇄조도 이주·화물·보상 수량의 불일치를 드러냄
- 심소연의 과거 인계 사고와 유사한 가명·배상 구조가 현재 장부에 존재

### 구간 목표

- 서로 다른 장부가 같은 사람과 손실을 어떻게 다르게 기록했는지 밝힘
- 심소연의 과거 보호 대상·피해자 측과 직접 충돌
- 제한 공개안의 실제 이익과 피해를 체험

### 중심 질문

> 한 사람을 명부에서 지워 살린 일이 다른 사람의 배상권을 지웠다면, 무엇을 되돌려야 하는가?

### 행동 압력

- 채무 동결·창고 압수·가구 퇴거·표국 신용 정지
- 장부 탈취보다 열람 순서와 채권 우선순위가 물리적 충돌을 만듦
- D4는 수량·재료·덧댄 종이를 드러내지만 거짓말을 판정하지 않음

### 비가역 변화

- 심소연의 과오가 핵심 인물·가족·표국에 공개
- 공식 보증 상실 또는 중대한 제한
- 진소무와 심소연 관계가 가장 크게 균열
- 사문경의 중앙 편집안이 현실적인 유혹으로 상승

### 필요한 국소 상세화

- 심소연 과거 사고 당사자 2축
- 금릉 장부고·전장·창고 세 장소
- 배상·채무 우선순위 사례
- D4 감정 절차

## 구간 5 — 서남의 몸과 사인

- 장 배분: 24~29장
- 기간: 이동 포함 8~12주
- 권역: 서남 약재·광산·산악 교통권
- 주시점: 두약란 중심
- 주도 인물: 두약란
- 중심 신병: D6 청맥인
- 적대 압력: 의가·독가·광산 권력·위철산 보호망

### 시작 상태

- 장부는 희생자 수를 보여 주지만 사망 원인은 상충
- 현재 신병 사용자와 증인의 잔흔 침식·부상 누적
- 치료 자원이 부족

### 구간 목표

- 화재·독·질식·압궤·익사 사인 경쟁 가설 검증
- 의료 기록의 보호 누락과 체면 왜곡 분리
- 두약란의 과거 자원 배분 피해와 현재 선택 연결

### 중심 질문

> 몸의 흔적도 누가 치료받았고 누가 기록됐는지에 따라 편향된다면, 의술은 무엇을 증명할 수 있는가?

### 행동 압력

- 희귀 해독·치료 자원을 누구에게 먼저 쓸지 결정
- 환자 이송·검시 방해·광산 봉쇄·약재 운송
- D6 사용이 통증 전이·부정맥·손떨림을 남김

### 비가역 변화

- 두약란의 과거 편향 피해 공개
- 핵심 인물 최소 한 명에게 장기 후유증 확정
- 전면 공개가 현재 환자·공동체를 실제 위험에 빠뜨림을 증명
- 위철산의 보호 논리가 일부 정당성을 얻음

### 필요한 국소 상세화

- 치료 자원 2~3종
- 의가·독가·광산 세력 관계
- 환자·피해자 대표
- 이동·고도·계절·회복 시간

# 제4막 — 문을 지키는 자들

## 구간 6 — 봉서검의 재심

- 장 배분: 30~35장
- 기간: 4~6주
- 권역: 중원 무림맹 회합처·도문 대종문·감정기관
- 주시점: 유청하 중심
- 주도 인물: 유청하
- 중심 신병: D8 봉서검
- 적대 압력: 사문경·무림맹 의석파·유청하 사문

### 시작 상태

- 해상·장부·의료 증거는 원사건의 복합 책임을 가리키지만 공식 계승·명령 판정은 그대로
- 유청하는 사문 보증과 동문 제압 과오를 피할 수 없음

### 구간 목표

- 상충 명령의 전달·봉인·지연 순서를 재심
- D7 이중 정통성 판정을 D8 계승 심사와 연결
- 유청하가 공개 절차와 실제 공정을 구분

### 중심 질문

> 공개된 절차가 이미 참가자와 증거를 골라 놓았다면, 명예 있는 재심은 가능한가?

### 행동 압력

- 공개 비무·재심 호위·증인 보호가 동시에 진행
- D8은 문서 내용을 주지 않고 전달 순서와 명령 압력을 몸으로 재현
- 유청하가 사문 명령을 따를수록 핵심 증언이 배제되는 구조

### 비가역 변화

- 유청하가 사문 보증 일부를 반납하거나 상실
- 무림맹이 봉인·제한 공개·전면 재심파로 분열
- 사문경의 과거 계승 판정이 공개 검토 대상이 됨
- 국가·무림맹·지역 공동체가 서로 다른 합증을 준비

### 필요한 국소 상세화

- 무림맹 회합·재심 절차
- 유청하 사문과 동문 피해자
- D8 보관·감정기관
- 공개 비무의 실제 법적 한계

## 구간 7 — 개문창과 연생장

- 장 배분: 36~42장
- 기간: 6~9주
- 권역: 산동·북방 군문 거점과 피난·구휼 통로
- 주시점: 심소연·고정문, 위철산 제한 외부 관찰
- 주도 인물: 고정문·심소연
- 중심 신병: D1 개문창, D5 연생장
- 적대 압력: 장계엄·위철산·군문·구휼 공동체

### 시작 상태

- 국가가 신병·증인·문서를 군기·공공 위험으로 묶어 집중 회수하려 함
- 위철산은 공동체를 이동시키고 기록을 분산·파기하려 함
- 고정문은 과거 집중 보관 실패를 반복할 위험

### 구간 목표

- 사람·증거·신병을 다른 경로로 이동시키는 분산 인계 실행
- 군문 봉쇄와 민간 피난의 충돌 해결
- D1·D5의 책임 계보를 전투와 구조 속에서 검증

### 중심 질문

> 문을 닫아 지키는 책임과 문을 열어 살리는 책임은 누가 판단하는가?

### 행동 압력

- 좁은 통로·검문·피난 대열·군문 화기·집단 공황
- D1은 명령·저지 압력, D5는 호흡·대열 유지
- 신병·증인·환자·문서를 같은 이동대에 둘 수 없음

### 비가역 변화

- 장계엄과 사문경의 일시 협력 붕괴
- 위철산이 전면 파기의 한계를 일부 인정
- 고정문이 국가 집중 보관을 거부하거나 변형
- 쇄조도 현장 검증이 불가피해짐

### 필요한 국소 상세화

- 군문·피난 통로 지형
- D1 군호·무가와 D5 구휼 공동체
- 분산 인계 노선·통신
- 군진·화기·민간 구조 전술

# 제5막 — 분산 합증

## 구간 8 — 쇄조도와 복수 거점

- 장 배분: 43~48장
- 기간: 준비·실행 4~6주
- 권역: 쇄조도 + 금릉·해상·중원·서남 복수 거점
- 주시점: 심소연 중심, 핵심 인물 1~2회 선택 시점
- 주도 인물: 심소연과 각 거점 책임자
- 중심 신병: 현물 전부가 아니라 네 짝의 분산 교차
- 최종 압력: 사문경·위철산·장계엄의 서로 다른 결말 강요

### 시작 상태

- 사건 골격은 대부분 드러났으나 공개·보관·배상·귀환 방법이 결정되지 않음
- 현물·사본·증언·현장 흔적이 여러 지역에 분산
- 핵심 인물의 신용·신체·관계가 손상된 상태

### 구간 목표

- 쇄조도 현장과 복수 거점의 증거를 시간차로 교차
- 사실·책임·개인 권리·현재 보호를 나눠 판정
- 신병을 지역과 사용자 책임망으로 귀환

### 중심 질문

> 진실을 넘기는 일도 인계라면, 누구에게 무엇을 넘기고 무엇을 넘기지 않아야 하는가?

### 행동 압력

- 쇄조도 현장의 붕괴·수로·옛 방책 검증
- 각 거점의 봉인 개방·환자 보호·장부 공개·군문 압수 시도가 동시에 발생
- 최종 경쟁은 문서 발표가 아니라 서로 다른 현물·사람·경로를 지키는 행동

### 결산

#### 공개

- 이관 작전·봉쇄·과밀 구금·구조·장부 소각의 복합 골격
- 조직 책임과 복권 근거

#### 제한

- 개인 과오·가족·상속·소유 분쟁

#### 보호 봉인

- 현재 거처·가명·환자·피난처·항로

#### 귀환

- 신병을 각 지역 책임망으로 돌려보내고 감정기관 독점을 해체

### 인물 비가역 결산

- 심소연: 중립 신화와 표국 신용 일부를 잃고 분산 인계 선례를 남김
- 진소무: 정보 거래를 당사자 통제·배상과 연결
- 두약란: 치료·기록·자원 기준을 바꿈
- 유청하: 사문 지위보다 재심 책임을 선택
- 나해란: 전승궁과 해상 공동체의 자칭을 공식 기록에 남김
- 고정문: 집중 보관 원칙을 폐기
- 사문경: 자료는 남지만 승인 독점권 상실
- 위철산: 보호 봉인은 얻지만 전면 파기의 피해를 책임
- 장계엄: 국가 관할 일부는 남지만 비밀 구금·전면 압수 정당성 붕괴

### 필요한 국소 상세화

- 쇄조도 현장 평면과 접근 조건
- 네 거점 간 통신·봉인 개방 순서
- 최종 공개 문서의 층위
- 신병 귀환·배상·재심 실행표
