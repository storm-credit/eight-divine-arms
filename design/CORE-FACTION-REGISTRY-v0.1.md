# 《팔대신병록》 핵심 세력·대표 인물 등록부 v0.1

상태: `provisional-canon`
관문: Gate 3
작업 패킷: WP-309
총괄: PM 오케스트레이터

A등급 14개는 작품 제작용 설정 밀도 분류다. 본문에서 `14대세력` 같은 집합 명칭으로 사용하지 않는다.

실제 역사 기관과 가상 파견조·문파·세가는 구분한다. 현재 이름과 직함은 역사·언어 감사에서 조정할 수 있다.

## 1. 무림맹

유형:
- 정파·세가 회원망의 중앙 중재·기록 기관

권역:
- 중원 중심, 전국 회원망

대표:
- 맹주 `조문헌(趙文憲)` — 의석 안정과 전면전 방지를 우선
- 기록총관 `방여진(方如眞)` — 과거 판정 재심과 증거 교차를 주장

내부 갈등:
- 구파일방 보수파
- 세가 현실파
- 지역 회원 확대파
- 관부 협력파
- 재심 개혁파

원사건·신병 연결:
- D8 봉서검과 O8 계보의 후대 기록 일부
- 해문 밀약을 절대병기 분할 협약으로 잘못 정리한 편찬본
- 신병의 계승·보관·공개 포럼을 장악하려 함

현재 목표:
- 진위 검증
- 회원 간 중재
- 국가 압수와 사도련·상단의 독점 방지

금지선:
- 증거 없이 전 세력 추살·압수

## 2. 소림사

- 계율원 책임 `혜각(慧覺)` — 산문 명예와 기록 공개에 신중

- 계율·산문 수호파
- 속가·강호 개입파
- 구휼·민생파

- D5 연생장의 계승품 인정 여부에 영향

현재 목표:
- 민간 피해 보호
- 위험 기억의 제한
- 불문 구휼망의 책임 인정

금지선:
- 사문 명예를 위해 생존자 명부를 폐기
