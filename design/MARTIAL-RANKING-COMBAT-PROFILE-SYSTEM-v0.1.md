# 《팔대신병록》 경지 언어·숙련·실전 프로필 체계 v0.1

상태: `provisional-canon / S3-02`
설정 분야: D09 무공, D18 명성·서열, D27 상한 장부
총괄: PM 오케스트레이터

## 1. 핵심 원칙

이 작품은 `삼류→이류→일류→절정→화경→현경` 같은 단일 절대 레벨표를 정본으로 쓰지 않는다.

대신 네 층을 분리한다.

1. 강호 외부 평판
2. 특정 기술 숙련
3. 종문 내부 수련 언어
4. 실제 전투 프로필

같은 인물도 네 층에서 다른 평가를 받는다.

## 2. 외부 평판 언어

### 초입·하수

- 정식 기초가 부족하거나 실전에서 안정적으로 기술을 재현하지 못함
- 모욕적 표현으로도 사용될 수 있어 객관적 등급이 아님

### 숙수

- 한 계통의 기초를 실전에서 운용
- 일반 무관 교두·표사·군영 정예 중 상당수가 포함 가능
- 지역과 직업에 따라 기준 차이

### 고수

- 자기 전문 분야에서 명확한 우위와 전적
- 여러 상황에 적응하거나 후배를 지도할 수준
- 전국적 명성이 없어도 지역 고수 가능

### 일류로 꼽힘

- 넓은 권역에서 검증된 전적·공적·보증
- 기술뿐 아니라 판단·생존·지휘가 평가됨
- ‘일류’는 정확한 인원 제한이 없음

### 절정급

- 당대 최상위권이라는 평판
- 지역·병기·시대에 따라 복수 인물 존재
- 상성·부상·군진·독·화기에 무적이 아님

### 종사급

- 한 무학의 원리·교육·변형에 기준을 세운 사람
- 절정급 전투력과 겹칠 수 있으나 필수는 아님
- 노년 교관·의술가·병기 장인도 종사로 불릴 수 있음

## 3. 평판의 형성

평판 자료:

- 공개 비무
- 실제 호송·토벌·전쟁
- 구휼·중재·교육
- 문파·세가 보증
- 무림맹 공적록
- 개방·상단·하오문의 소문
- 적대 세력의 생존 증언

평판 왜곡:

- 사문 홍보
- 패배 은폐
- 상대의 정체 오인
- 병기·독·지형 도움 누락
- 익명·가명
- 오래된 전적
- 은거·부상

## 4. 특정 기술 숙련 언어

작가용 다섯 단계:

### 문턱

- 원리를 이해하기 시작
- 자세와 구결을 재현하지만 안정성 부족

### 등당

- 기본 구조를 무너지지 않고 운용
- 예측 가능한 상황에서 실용

### 입실

- 변화·상성·실패 복구를 이해
- 실제 상황에서 자기 몸에 맞춰 조정

### 순숙

- 불필요한 힘과 동작이 줄음
- 다른 기술과 자연스럽게 연결
- 후배의 오류를 진단 가능

### 화경·등봉의 찬사

- 기술이 의식적 절차를 넘어 몸과 판단에 통합
- 객관적 생리 변신이 아니라 극찬
- 작품 인물이 과장·아첨·전설로 사용할 수 있음

이 단계는 검술·창술·경공·내공·의술·진법마다 별도다.

## 5. 종문 내부 수련 언어

예시:

- 주천·소주천·대주천
- 정기신
- 삼화취정·오기조원
- 선정·관법·심인
- 혈해·마심·성화 등 교단 고유어

원칙:

- 다른 계보와 자동 환산하지 않음
- 교리·체감·교육의 언어
- 실제 전투 결과는 별도 검증
- 신선화·불사·영혼 승천의 객관적 단계로 쓰지 않음

## 6. 실전 프로필 열두 축

주요 무인은 다음으로 평가한다.

1. 순간 출력
2. 장기 지구력
3. 보법·거리
4. 반응·감각
5. 주 병기
6. 부 병기·맨손
7. 방어·갑주 대응
8. 협공·지휘

10. 정보·기만
11. 부상·회복
12. 심리·잔흔 저항

프로필은 수치보다 `강·중·약·조건부`와 근거를 기록한다.

## 7. 전투 상황 태그

- 일대일 공개 비무
- 좁은 실내
- 선상·부두
- 산악·수풀
- 평지·군진
- 야간·연기
- 호송·보호
- 추격·도주
- 부상·피로
- 다수전
- 원거리 사격
- 비살상 체포

인물의 강함은 상황 태그에 따라 크게 달라진다.

## 8. 승패 예측 원칙

전투 전에 비교한다.

- 목표: 죽이기·지연·탈출·보호·회수
- 거리와 첫 위치
- 병기·갑주
- 지형·날씨
- 정보 비대칭
- 숫자·협공
- 부상·수면·식사
- 퇴로·시간 제한

경지 평판은 이 요소 중 하나일 뿐이다.

## 9. 격차의 표현

### 작은 격차

- 실수·지형·상성으로 뒤집힘
- 승자가 다칠 가능성 큼

### 명확한 격차

- 약자가 정면 승부는 어렵지만 도주·지연·보호 가능
- 협공·함정·원거리·독이 의미 있음

### 극단적 격차

- 정면 개인전은 거의 불가능
- 목표 변경·환경·군진·관부·협상 필요
- 약자가 한 수에 반드시 즉사하는 연출로 고정하지 않음

## 10. 일류·절정의 희소성

정확한 전국 인원은 정하지 않는다.

작가 기준:

- 숙수·지역 실무 무인은 많음
- 일류 평판은 권역마다 제한된 수가 거론됨
- 절정급은 전국에서 이름이 알려질 가능성이 높지만 은거·외부 무림·군문은 누락
- 종사급은 전투보다 후학·체계 영향으로 더 희귀할 수 있음

`천하십대고수`가 존재하더라도 사설 목록이며 실제 상위 열 명을 작가가 보증하지 않는다.

## 11. 무림 서열과 실제 프로필의 분리

서열록에는:

- 공개 전적
- 최근 활동
- 사문·지역
- 평가 병기
- 미확인 정보
- 제외 사유

를 기록한다.

실제 프로필은 작가 장부에만 있으며 본문 인물은 전부 알지 못한다.

## 12. 은거·익명·살수·군문 인물

서열 제외 이유:

- 공개 전적 없음
- 가명·신분 은닉
- 군사 기밀
- 전문 임무가 공개 비무와 다름
- 부상·은퇴
- 외부 무림의 다른 평가 체계

`서열에 없다 = 약하다`가 아니다.

## 13. 팀 전투 평가

팀은 개인 등급 합이 아니다.

평가:

- 역할 분담
- 신호·언어
- 병기 거리 조합
- 서로의 약점 보완
- 부상자 보호
- 후퇴 순서
- 지휘권 충돌

훈련된 숙수 다수가 일류 개인을 압박할 수 있지만 공포·지형·지휘 실패로 무너질 수도 있다.

## 14. 군대·화기와 고수

절정급도:

- 다방향 활·쇠뇌
- 조총·화포·연기
- 방패·장창 대형
- 장애물·수문·성벽
- 교대 추격

에 노출되면 위험하다.

고수의 강점:

- 지휘관·전령·포수 접근
- 좁은 틈 돌파
- 사기·혼란
- 야간 기습

군대 전체를 정면에서 혼자 쓸어버리는 힘은 허용하지 않는다.

## 15. 신병과 프로필

신병은:

- 특정 감각·동작·환경 판단을 좁게 보조
- 기존 전문성과 맞을 때 효과 증가
- 다른 축을 자동 상승시키지 않음
- 사용 대가와 침식으로 장기 프로필을 악화시킬 수 있음

신병 보유자라는 이유로 절정급으로 분류하지 않는다.

## 16. 작가용 프로필 카드

- 외부 평판
- 분야별 숙련
- 종문 수련어
- 12축 실전 프로필
- 강한 상황 태그
- 약한 상황 태그
- 주요 상성
- 현재 부상·피로
- 정보 공개 수준
- 신병·특수 장비

## 17. 악마의 변호인 위험

- 외부 평판어를 숨은 숫자 레벨로 쓰지 않는다.
- 절정이 숙수 백 명을 무조건 이기지 않는다.
- 화경·현경·삼화취정을 한 줄로 배열하지 않는다.
- 작가 프로필을 등장인물들이 자동으로 알지 않는다.
- 서열 밖 인물을 매번 더 강한 반전으로 쓰지 않는다.
- 상성만으로 노력·격차를 전부 무효화하지 않는다.

## 18. 교차 감사 질문

- 분야별 숙련을 구분했는가
- 실제 전투 목표와 환경이 무엇인가
- 부상·정보·협공을 반영했는가
- 종문 수련어를 객관 레벨로 오해하지 않았는가
- 신병 효과가 다른 능력을 자동 보완하지 않는가
