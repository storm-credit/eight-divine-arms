# 《팔대신병록》 인물 관계·세대 지도 v1

상태: `recommended-default / human-edit-pending`
관문: Character Gate
총괄: PM 오케스트레이터

## 1. 관계 설계 목적

인물 관계를 `친구·연인·적` 한 단어로 고정하지 않는다. 모든 핵심 관계는 다음 축 중 최소 세 개를 가진다.

- 신뢰
- 권리
- 정보
- 채무
- 보증
- 사승·직업
- 가족·돌봄
- 무공 상성
- 신병 적합성
- 과거 과오의 유사점과 차이

## 2. 세대 구조

### 청년 실무 세대 — 20대 초중반

- 진소무: 하층 정보망의 다음 운영 방식을 고민
- 표국 견습·가업 후계 슬롯: 심소연을 영웅 또는 배신자로 볼 수 있음

특징:
- 공식 의석과 권력은 약함
- 새로운 정보·이동·직업 경로를 가장 빨리 사용
- 선배 세대의 비밀 비용을 떠안음

### 책임 전환 세대 — 20대 후반~30대

- 심소연
- 유청하
- 나해란
- 두약란
- 고정문
- 위철산

특징:
- 직접 현장을 맡지만 최종 제도 권한은 없음
- 선배가 만든 규칙을 실행하거나 어길 책임
- 작품의 핵심 선택과 관계 변화 담당

### 제도 유지·수정 세대 — 40대

- 마윤경
- 사문경
- 장계엄
- 각 문파·표국의 표두·장문 실무자 슬롯

특징:
- 개인 과오가 조직 절차가 된 세대
- 질서를 유지하면 누군가가 지워지고, 공개하면 전체 신용이 흔들리는 위치

### 기억·기술 보관 세대 — 50대 이상

- 한목수
- 지역 사찰·의가·군문·표국의 노실무자 슬롯

특징:
- 문서보다 몸·도구·지역 기억을 보존
- 젊은 세대가 모든 것을 새로 발견한다는 착각을 반박
- 보수적이지만 단순한 과거 집착이 아님

## 3. 핵심 관계 행렬

기호:
- `+` 협력 기반
- `-` 실제 이해 충돌
- `±` 조건부 동맹·경쟁
- `S` 공유 비밀 또는 유사 과오
- `R` 권리 분쟁
- `M` 무공·신병 상호보완

| 관계 | 기호 | 핵심 내용 |
|---|---|---|
| 심소연–진소무 | ± S | 공식 기록과 기록 밖 신분, 사람을 보호하려 기록을 바꾼 공통 과오 |
| 심소연–두약란 | ± M | 증거 공개와 환자 비밀, D3 부상과 D6 판별 |
| 심소연–유청하 | ± M | 비공식 보호와 공개 명예, 실무 무공과 정통 검술 |
| 심소연–나해란 | + - M | 문서 원문과 통역의 보호적 변형, D3와 D7의 해상 교차 |
| 심소연–고정문 | ± R | 표국 인계와 국가 압수, 실무자끼리의 상호 이해 |
| 심소연–마윤경 | + - S | 직업적 선배, 신용망 보호를 위한 기록 지연의 유혹 |
| 심소연–한목수 | + R M | 장인 기술 스승 가능, D3 사용·보관·수리권 충돌 |
| 심소연–사문경 | - S | 사람을 보호하려 기록을 고친 공통 경험, 책임 분산과 승인 중앙화 대립 |
| 심소연–위철산 | ± R | 기록이 약자를 해친다는 공감, 제한 공개와 전면 파기의 대립 |
| 심소연–장계엄 | - R | 절차 언어는 공유하지만 최종 국가 관할을 두고 충돌 |
| 진소무–마윤경 | ± | 가명·비공식 정보와 상업 장부·신용망의 거래 |
| 진소무–고정문 | - | 정보원 보호와 국가 증인 확보 충돌 |
| 진소무–위철산 | ± | 기록 밖 공동체 보호에 공감하나 소무는 협상 가능성을 남김 |
| 두약란–유청하 | ± | 독·기환 의심과 정파 명예·금기 판정 충돌 |
| 두약란–고정문 | ± R | 검시·환자 비밀과 관부 증거 요구 |
| 두약란–장계엄 | - | 시신·환자를 국가 증거로 우선하는 강경 압력 |
| 두약란–위철산 | ± | 공동체 보호 공감, 치료 기록·시신 파기에는 반대 |
| 유청하–사문경 | ± S | 잘못된 계승·징계 판정의 과오, 재심과 중앙 승인 방식 대립 |
| 유청하–장계엄 | ± | 국가와 정파의 무력·관할 경쟁, 공개 질서에 대한 유사 믿음 |
| 유청하–나해란 | + - M | 정통 검술과 해상 궁술, 외부 집단을 강호 규범으로 재단하는 문제 |
| 나해란–고정문 | ± R | 외부 집단 통역 기록과 국가 밀무·왜구 분류 충돌 |
| 나해란–장계엄 | - | 해상 공동체를 안보 범주로 통합하려는 압력 |
| 나해란–사문경 | - R | D7 복원궁·전승궁 중 무엇을 공식 진품으로 인정할지 충돌 |
| 마윤경–사문경 | ± S | 신용 붕괴를 막기 위한 선택적 공개, 누가 승인권을 갖는지 대립 |
| 마윤경–장계엄 | ± R | 상업 기록·전장 사본과 국가 압수권 충돌 |
| 한목수–위철산 | ± | 외부 약탈 방지에는 공감, 지역 기술 기록 파기에는 대립 |
| 한목수–사문경 | - R | D3를 감정기관의 자산으로 편입하려는 시도 거부 |
| 사문경–위철산 | ± - | 제한 편집과 전면 파기, 필요할 때 자료 회수 협력 가능 |
| 사문경–장계엄 | ± - | 사설 승인 체계와 국가 독점, 서로를 이용하지만 최종 적대 |
| 위철산–장계엄 | - | 현재 공동체의 생존과 국가 안보·수배 논리의 정면 충돌 |

## 4. 주인공 중심 관계의 단계가 아닌 구조

플롯 순서를 정하지 않고도 다음 긴장이 존재한다.

### 심소연–진소무

- 서로의 정보가 필요함
- 상대가 이름과 기록을 다루는 방식을 위험하게 여김
- 둘 다 과거에 한 사람을 보호하려 다른 사람에게 비용을 전가
- 친밀해져도 직업 윤리가 완전히 합쳐지지 않음

### 심소연–두약란

- 소연은 상태 기록을 남기려 함
- 약란은 환자의 동의와 치료를 우선
- 소연의 몸이 신병 증거가 될 때 약란은 환자화를 주장
- 약란의 기록 축소 과오가 드러나면 역할이 역전됨

### 심소연–유청하

- 청하는 공개 절차·명예·강한 무공으로 문제를 막으려 함
- 소연은 비공식 이동·거절·은폐가 사람을 살리는 경우를 앎
- 청하는 소연에게 전통 무공과 강호 정치의 문을 열지만 심판자가 아님
- 소연은 청하에게 기록 밖 사람과 계약 비용을 보여 줌

### 심소연–나해란

- 둘 다 중개자이지만 기록과 번역의 책임 방식이 다름
- 해란은 외부 공동체 보호를 위해 표현을 바꾸려 함
- 소연은 바뀐 말이 인계 책임을 흐린다고 봄
- 둘 사이의 신뢰는 원문 공개가 아니라 상호 동의 절차에서 생김

## 5. 로맨스 가능성 지도

현재 정본화하지 않는다.

가능 구조:

- 심소연–진소무: 공범성·가명·취약한 신뢰
- 심소연–유청하: 경쟁·존중·계층 차이
- 심소연–나해란: 중개자끼리의 책임과 거리
- 심소연 비연애: 가족·동료·직업 공동체 중심

금지:

- 세 후보를 동시에 삼각관계로 운영
- 여성 인물 관계를 연애 경쟁으로 축소
- 연애가 과거 과오의 용서 장치가 됨
- 명문 남성의 보호가 주인공 신용 회복을 해결

Human Edit에서 `로맨스 없음`, `느린 단일 관계`, `후속 검토` 중 하나를 선택한다.

## 6. 가족·스승 관계 슬롯

### 심소연

- 생업 기술을 가르친 가족 장인
- 장부·채무·거래를 관리한 여성 가족
- 진원표국 분점 표두
- 기록 수정의 손실을 떠안은 동료 또는 스승
- 소연 선택으로 도움과 피해를 모두 받은 보호 대상

### 유청하

- 공개 질서를 가르친 스승
- 누명을 쓰거나 희생된 동문
- 사문 내 재심을 지지하지만 공개하지 못하는 선배

### 두약란

- 치료 실패를 함께 은폐한 계보 장로
- 피해자 가족 또는 생존 환자
- 중원에서 독가로 낙인찍힌 후배

### 나해란

- 외부 해상 언어와 신호를 가르친 가족·선주
- D7 전승궁 제작·수리 계보
- 복원궁을 지지하는 공식 감정가와의 오랜 논쟁

### 진소무

- 하오문 지부 책임자
- 실종된 증인의 가족
- 가명을 물려받은 어린 정보원 또는 견습

## 7. 비밀 분류

### 개인 과오 비밀

- 심소연: 인계 기록 수정
- 진소무: 증인 신분 판매
- 두약란: 치료 실패 축소
- 유청하: 잘못된 징계 공개 지지
- 나해란: 보호적 오역
- 고정문: 불법 압수의 피해 축소
- 마윤경: 파산·피해 명부 지연
- 한목수: 수리 기록 파기

### 조직 비밀

- 진원표국 분점의 손실·보증 조작 가능성
- 명문 사문의 잘못된 계승·징계
- 금의위 남방 회수별조의 비공식 구금
- 상업 장부고의 선택적 공개
- D7 복원궁 감정기관의 정치적 판정

### 원사건 비밀

개별 인물이 처음부터 전체를 알지 않는다.

- 사문경도 기록 일부와 계승 논리만 앎
- 위철산은 공동체 피해와 명부 위험을 앎
- 장계엄은 국가 문서와 안보 위험을 앎
- 한목수·나해란은 지역 기술·전승 기억을 앎

## 8. 정보 독점 금지

- 한 인물이 모든 기록을 소유하지 않음
- 한 인물이 모든 언어를 번역하지 않음
- 한 인물이 모든 신병을 사용할 수 없음
- 한 인물이 쇄조도 진실 전체를 기억하지 않음
- 적대자 사망이 자동 진실 공개로 이어지지 않음

## 9. 관계 변화의 허용 범위

가능:

- 불신→조건부 보증
- 경쟁→상호 이의 제기권
- 적대→일시적 공동 보호
- 스승→피심문자
- 보호자→권리 침해자
- 피해자→공개 반대자

금지:

- 비밀 하나 공개 후 완전 신뢰
- 전투에서 구해 준 뒤 모든 가치 충돌 해소
- 연애 성사로 직업·계층 갈등 삭제
- 악역 개심으로 제도 문제 자동 해결

## 10. 인물 과밀 통제

- 상시 동행은 최대 4~5명
- 한 권역에서 새 핵심 이름 최대 3명
- 사건당 적대 압력 대표 최대 2명 전면화
- 나머지는 서신·대리인·세력 효과로 존재
- 같은 기능을 가진 B/C 인물은 기존 핵심 인물과 연결하거나 병합

## 11. Human Edit 확인 항목

- 핵심 5인 구조 승인 여부
- 나해란의 성별·외부 해상 배경
- 유청하를 완전 파문이 아니라 불안정한 내부 이탈자로 둘지
- 사문경·위철산·장계엄 삼각 적대 승인 여부
- 로맨스 기본값
- 가족 생존과 직업 공동체 비중
