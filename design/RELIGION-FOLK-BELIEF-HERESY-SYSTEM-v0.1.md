# 《팔대신병록》 종교·철학·민간신앙·사교 판정 체계 v0.1

상태: `provisional-canon / S4-05`
설정 분야: D16 종교·철학·민간신앙, D12 기환, D08 종교 조직, D20 탄압·구휼
총괄: PM 오케스트레이터

## 1. 설계 원칙

종교 조직은 무공을 가르치는 문파의 장식이 아니다.

종교는:

- 세계와 죽음을 해석
- 일상 규율·의례
- 교육·필사
- 구휼·의료·장례
- 토지·기부·순례
- 지역 정체성·정치 권위

를 만든다.

실제 기환 효과와 신앙의 의미는 구분하되, 효과가 없다고 신앙이 무가치한 것은 아니다.

## 2. 불문 공동체

### 형태

- 대사찰·명산 사원
- 지역 사찰·암자
- 율원·선원·강원
- 재가 신도·속가 제자
- 구휼·장례·숙박 시설

### 기능

- 계율·수행·교육
- 필사·경전·문서 보관
- 시주·토지 관리
- 구휼·의료·장례
- 지역 중재·피난

### 무공과의 관계

- 몸과 마음의 수련은 종교 수행 일부일 수 있음
- 모든 승려가 무승이 아님
- 무공 지도자와 주지·계율 책임자가 다를 수 있음
- 살상·호국·구휼 사이 내부 논쟁

### 내부 갈등

- 계율 엄수 대 현실 구휼
- 산문 보존 대 강호 개입
- 출가자 대 속가·후원 가문
- 토지·면역·부역 문제
- 기환 유물 보관과 공개

## 3. 도문 공동체

### 형태

- 도관·궁관
- 산중 수련처
- 재가 도사·법사
- 의술·점복·장례 계보
- 국가 제사·지역 신앙과 연결

### 기능

- 경전·수련·재초·의례
- 역법·점복·풍수·의술
- 장례·천도·부적·법구
- 숙박·약재·지역 중재

### 무공과의 관계

- 호흡·양생·검술·보법
- 내단 용어는 종문 내부 수행 언어
- 삼화취정·오기조원을 객관 전투 레벨로 쓰지 않음

### 내부 갈등

- 내단 수련 대 의례 실무
- 국가 인정 계보 대 지역 법사
- 은거 대 강호 정치
- 기환 연구 대 사기·오인 경계

## 4. 민간신앙

구성:

- 성황·토지·수신·산신
- 조상·가신
- 항해·어업·제염 신앙
- 장인·시장·질병 수호
- 지역 영웅·열녀·전사자 추모

민간신앙은:

- 지역 공동체의 기억
- 위험 관리
- 축제·장날·기부
- 장례·혼인·출산

과 연결된다.

국가·불문·도문과 경쟁·혼합할 수 있다.

## 5. 국가 제사와 종교 행정

국가는:

- 공식 제사·사전
- 사찰·도관 등록과 승도 관리
- 요언·사교·집단 결사 단속
- 재난·기우·호국 의례

에 관여한다.

그러나 실제 지방 신앙을 전부 통제하지 못한다.

관부가 종교를 보는 기준:

- 호적·세금·토지
- 집단 동원
- 예언·요언
- 무장·비밀 결사
- 사기·치안

## 6. 종교와 구휼

종교 조직의 구휼:

- 죽·곡물
- 약·치료
- 숙박·피난
- 고아·노인·무명 시신
- 장례·운구

위험:

- 명부·개종·노동 요구
- 재산·토지 축적
- 특정 신분 배제
- 후원 가문의 영향
- 전염병 확산

구휼은 자선과 권력의 양면을 가진다.

## 7. 교리와 인물 사고

### 업보·인과

- 행동의 결과와 책임을 해석
- 피해자에게 전생 죄를 돌리는 방식은 비판 가능

### 천명·천도

- 왕조·정치 정통성
- 개인 승패를 천명 하나로 정당화하지 않음

### 음양오행

- 의술·풍수·무공·계절을 설명하는 공통 언어
- 모든 현상의 정확한 자연과학 공식이 아님

### 무상·무위·자비·수양

- 인물의 선택과 조직 윤리를 형성
- 실제 행동과 교리 사이 모순이 갈등

## 8. 출가와 신분

출가 동기:

- 신앙·수행
- 교육
- 가난·구휼
- 전쟁·가족 문제
- 범죄·도피
- 정치 후원

출가가 모든 채무·범죄·가족 책임을 자동 소멸시키지 않는다.

종교 이름을 이용한 가짜 출가·면역·도피가 존재한다.

## 9. 종교 무인의 지위

- 출가 계율과 무력 사용의 긴장
- 사찰·도관 방어
- 호송·구휼·토벌 참여
- 관부·무림맹의 이중 보증

종교 무인은 사문 명예와 신앙 공동체 모두에 책임을 진다.

## 10. 사교 판정의 위험

`사교`는 신학적 진위보다 정치·치안의 낙인이 될 수 있다.

사교로 몰리는 이유:

- 비공식 집회·예언
- 세금·호적 이탈
- 집단 이동·무장
- 비밀 의례
- 지역 신앙·외부 언어
- 정치 반대

따라서 관부·정파의 사교 판정을 자동 진실로 받아들이지 않는다.

## 11. 금단 결사의 실질 판정

실제 강한 금지 근거:

- 비자발적 희생
- 인체 실험
- 강제 정신 침식
- 시신·혈액의 대량 자원화
- 아동·환자·포로 이용
- 신분·증거 말소
- 집단 살해·반역 무장

기괴한 의례·외부 신앙·피 사용만으로 확정하지 않는다.

## 12. 혈교형 조직

혈교는 단일 천년 악의 제국으로 만들지 않는다.

가능 형태:

- 분산 금단 계보
- 인체 실험 결사
- 시체·독·기환 침식 연구 집단
- 지역 반란·복수와 결합한 교단

천마신교와 별개이며 사파 전체의 본체가 아니다.

모든 기환 사건의 배후로 사용하지 않는다.

## 13. 천마신교의 종교성

천마신교는 중원에서 `마교`로 불릴 수 있으나 독자 공동체다.

- 교리·의례
- 행정·법도
- 대상로·오아시스·요새
- 가족·상업·외교
- 무공 계승

을 가진다.

중원 정파의 이단 판정과 실제 비자발적 희생 여부를 분리한다.

## 14. 이단·분파·개혁

모든 종교는 내부 분파를 가진다.

- 계율 엄수
- 의례 실무
- 구휼·민생
- 무공·호국
- 기환 연구
- 관부 협력
- 지역 자치

분파가 곧 사교는 아니다.

## 15. 종교 유물과 기환

종교 유물은:

- 신앙·역사·공동체 권위
- 의례 기능
- 실제 재료·기관
- 희귀 기환 잔흔

을 가질 수 있다.

성물의 기환 효과가 입증되지 않아도 공동체의 보관·예배 권리는 존재한다.

## 16. 신앙과 치료

가능:

- 공포·애도·생활 리듬 안정
- 공동체 지원
- 의례적 집중
- 환자·가족의 의미 형성

금지:

- 신앙이 감염·골절·독을 자동 치료
- 의학이 신앙을 자동 무가치화

의원과 종교자는 협력·갈등할 수 있다.

## 17. 신앙과 기환 판별

세 층을 기록한다.

1. 신자에게 무엇을 의미하는가
2. 사회적으로 무엇을 하는가
3. 반복 가능한 기환 현상이 있는가

셋 중 하나만으로 다른 둘을 부정하지 않는다.

## 18. 종교 갈등

- 토지·세금·면역
- 순례·시장
- 장례·묘지
- 개종·혼인
- 사교 단속
- 기환 유물·예언
- 무장·구휼망

교리 차이만으로 전쟁이 일어나기보다 자원·정치·신분과 결합한다.

## 19. 대표 종교 조직의 설정 밀도

A/B등급 종교 세력은:

- 교리 핵심
- 일상 의례
- 재정·토지
- 구휼·교육
- 무공·법구
- 내부 분파
- 관부·무림맹 관계
- 장례·기환 관점

을 가진다.

개별 경전·신 목록을 과도하게 만들지 않는다.

## 20. 악마의 변호인 위험

- 불문·도문을 무공 속성 학교로 만들지 않는다.
- 민간신앙을 미신·사기의 집합으로만 보지 않는다.
- 사교 판정을 관부·정파의 객관 진실로 쓰지 않는다.
- 천마신교와 혈교를 합치지 않는다.
- 종교 지도자를 모두 위선자 또는 성인으로 만들지 않는다.
- 기환 효과의 유무로 신앙의 가치를 판정하지 않는다.

## 21. 교차 감사 질문

- 종교 조직이 무엇으로 먹고사는가
- 비무인 신도·노동자·구휼 대상이 있는가
- 국가·지역 신앙·종문이 어떻게 충돌하는가
- 사교 판정의 정치적 이해가 있는가
- 실제 비자발적 희생 근거가 있는가
- 종교 유물의 사회적 권리와 기환 효과를 구분했는가
- 교리와 인물 행동 사이 모순이 가능한가
