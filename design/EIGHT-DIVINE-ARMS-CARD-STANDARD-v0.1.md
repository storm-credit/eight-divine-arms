# 《팔대신병록》 팔대신병 개별 카드 표준 v0.1

상태: `provisional-canon`
묶음: S7 팔대신병 개별 설정집
총괄: PM 오케스트레이터

## 1. 목적

팔대신병을 능력 설명만 있는 보물 목록으로 만들지 않는다. 각 신병은 실제 병기·도구로서 제작·사용·수리·운송·보관·감정·권리·지역문화의 연속성을 가진다.

1. 코드·작업명·이형명
2. 병장기 계통과 실전 역할
3. 최초 사용자·원사건 역할
4. 최초 형태
5. 현재 형태·규격 범위
6. 재료와 공정
7. 원본 부품·교체 부품
8. 수리·개조 이력
9. 평시 관리와 소모
10. 운송·보관 방식
11. 일반 무기로서 강점·약점
12. 잔흔 채널
13. 촉발 조건
14. 접근 가능한 단서
15. 사각지대·오인
16. 신체·정신 대가
17. 적합성 조건
18. 비적합 사용자 반응

23. 후대 전설과 실제

25. 파괴·분해·수리 금지선
26. 의도적 미상

## 3. 원본성 평가

원본성은 단일 점수가 아니다.

- M 물질 연속성: 최초 재료·핵심 부품
- S 구조 연속성: 비례·결합·무게중심·공정
- U 사용 연속성: 같은 생업·전술·수련 계보
- R 기록 연속성: 수리·보관·인계 문서
- C 공동체 연속성: 지역·사문·사용자의 책임 기억
- E 잔흔 교차성: 다른 신병·장소·기록과의 반응

`원본`, `수리 원본`, `계승품`, `이중 계승`은 위 요소의 조합으로 판정한다.

## 4. 규격 운용

- 길이·무게는 현재 수리 상태에 따라 범위를 기록한다.
- 전통 단위를 우선하고 독자용 근사치를 작가 장부에 병기한다.
- 신병이라는 이유로 비현실적으로 무겁거나 거대하지 않다.
- 실제 사용자의 체격·병장기 훈련과 맞아야 한다.

## 5. 재료·공정

가능한 재료:
- 단조 철·강
- 목재·대나무·등나무
- 뿔·힘줄·가죽
- 황동·청동·은입사·칠
- 천·실·아교·수지

기환성은 희귀 광물 하나에서 자동 발생하지 않는다. 반복 사용·원사건·감정·수리 계보와 물질이 함께 매개한다.

## 6. 수리 원칙

- 자루·손잡이·칼집·활시위·가죽은 소모품이며 교체 가능
- 검신·창두·슴베·갈고리 머리·핵심 철물은 원본성에 큰 비중
- 수리 흔적을 완전히 지우는 복원은 금지
- 사용 불가능하게 보존하는 것과 실제 사용을 위한 수리 사이에 갈등 존재
- 부품 교체는 잔흔 채널의 강도·왜곡을 바꿀 수 있음

## 7. 보관 환경

- 금속: 습기·염분·녹·응력 균열 관리
- 목재: 건조·뒤틀림·충해·균열
- 활: 온습도·시위 분리·뿔과 아교층
- 문서 결합 병기: 종이·인장·칼집·수납 구조 분리 보관 가능
- 의술 병기: 세척·부식·오염·독 잔류

기환 현상보다 먼저 물리 열화를 검토한다.

## 8. 운송 원칙

- 포장·봉인·충격 방지·습기 방지
- 병기임을 숨길 때도 무게·형태·통관 위험 존재
- 운송인과 사용자는 다를 수 있음
- 대형 장병은 해체 운송 가능하나 핵심 결합부 봉인 필요
- 위험도 EX 계약은 복수 인계·사본·경로 분산을 사용

## 9. 감정 절차

1. 현장 상태 기록
2. 오염·수리·최근 사용 분리
3. 재료·공정·마모 검사
4. 수리 기록·계보·보관 이력 대조
5. 사용 시험은 최소화
6. 잔흔 반응은 복수 관측자와 의료 감시 아래 기록
7. 다른 신병·원사건 장소·관련 문서와 교차
8. 권리 판정과 진품 감정을 분리

## 10. 위조 유형

- 외형 모사품
- 원본 부품을 소량 넣은 혼합품
- 후대 명병에 거짓 계보를 붙인 사칭품
- 실제 계승품을 `가짜`로 몰아내는 정치적 위조
- 잔흔 반응을 독·향·음향·기관으로 연출
- 수리 기록·탁본·인장을 위조
- 진품의 일부 특징을 의도적으로 훼손해 다른 계보로 편입

## 11. 잔흔 접근 규칙

- 강한 채널 최대 두 개
- 실제 동작·환경·목적 필요
- 사용자의 전문성·상처·언어가 해석에 영향
- 장시간 사용은 정보가 아니라 침식을 늘릴 수 있음
- 완전 영상·정확한 대화·객관적 속마음 제공 금지

## 12. 대가 기록

대가는 세 층으로 기록한다.

- 신체: 관절·감각·호흡·통증·피로
- 인지: 강박·오인·기억 혼동·주의 편향
- 사회: 소유 주장·감시·수배·사용자 낙인

## 13. 권리 원칙

- 점유·소유·보관·사용·감정·수리·계승·공개·파기 권리 분리
- 적합성이 소유권을 만들지 않음
- 원본 재료가 계승권을 독점하지 않음
- 지역 공동체의 피해·보관 책임을 반영
- 국가 압수와 무림맹 판정은 최종 진실이 아님

## 14. 파괴·분해 금지선

- 진품 확인을 위해 되돌릴 수 없는 절단·연마 금지
- 잔흔을 강제로 끌어내기 위한 고문성 사용 금지
- 현재 사람의 안전을 이유로 모든 역사 증거를 자동 파기하지 않음
- 파괴해야만 멈추는 절대 재앙 장치로 만들지 않음

## 15. 카드 상태

- `draft`: 개별 항목 미완
- `review`: 감정·역사·병기·권리 교차검토 중
- `provisional-canon`: 설정집 통합 사용 가능
- `canon`: 사용자 승인 뒤 확정

## 16. 완료 조건

여덟 카드 모두에서:
- 기능 중복 없음
- 속성 세트화 없음
- 원사건 역할과 현재 지역 문화 연결
- 수리·감정·권리 분쟁 존재
- 잔흔의 정보·사각지대·대가 균형
- 하나를 잃어도 세계가 붕괴하지 않음
- 여덟이 모여도 전투력이 단순 합산되지 않음
