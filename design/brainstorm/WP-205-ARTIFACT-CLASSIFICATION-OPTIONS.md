# WP-205 희귀 병기·유물 분류 구조안 비교

상태: `review-complete`

## 후보 A — 단일 위력 등급

`범병 → 명병 → 보병 → 신병`처럼 강한 순서로 배열한다.

장점:
- 이해가 쉬움
- 보물 쟁탈의 위계가 선명

위험:
- 치료 법구·기록 유물·봉인 도구를 전투력으로만 평가
- 신병이 반드시 최강 무기가 되어야 함
- 숫자·등급 중심 게임화

판정: 반려.

## 후보 B — 기환 유무 이분법

기환 흔적이 없는 물건과 있는 물건으로만 나눈다.

장점:
- 단순함
- WP-204와 연결 용이

위험:
- 뛰어난 공예·역사·계승·권리의 차이를 설명하지 못함
- 미약한 기환 물건이 최고 명병보다 상위로 오인

판정: 반려.

## 후보 C — 용도별 분류

전투·치료·의례·기록·운송·봉인 도구로 나눈다.

장점:
- 보패와 법구의 기능 차별화
- 세력·직업과 연결

위험:
- 하나의 물건이 여러 용도를 가질 때 분류 혼란
- 품질과 희귀성 평가가 빠짐

판정: 보조 축으로 채택.

## 후보 D — 다축 분류

각 물건을 다음 축으로 평가한다.

- 제작 품질
- 실용 기능
- 재현 가능성
- 기환 흔적의 지속성
- 역사·계보
- 권리와 공적 인정
- 수리·파손·운송 난도

통칭은 대표 성격을 말할 뿐 절대 위력을 뜻하지 않는다.

장점:
- 일반 명병과 기환 법구를 비교하지 않아도 됨
- 팔대신병의 특별함을 역사·권리·지속성에서 확보
- 위조·수리·계승 분쟁 설계 가능

위험:
- 감정표가 복잡해질 수 있음
- 등장 장면에서 설명 과잉 위험

판정: 채택.

## PM 통합

후보 D를 기본으로 후보 C의 용도 축을 결합한다.

분류 명칭은 사회적 통칭이며, 같은 명칭 안에서도 품질과 가치 차이가 크다.
