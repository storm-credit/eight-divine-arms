# WP-304 현재 팔대신병 구조안 비교

상태: `review-complete`

## 후보 A — 여덟 속성 무기형

화·수·풍·지·빛·어둠 등 속성 능력을 하나씩 배정한다.

장점:
- 구분이 쉽고 시각적
- 수집 판타지의 직관성

위험:
- 저강도 기환과 충돌
- 원사건·생업·증언 기능이 약해짐
- 게임 아이템 세트처럼 보임

판정: 반려.

## 후보 B — 여덟 전투 역할형

공격·방어·치유·봉인·추적·기록 등 역할을 하나씩 배정한다.

장점:
- 기능 중복 통제
- 파티 역할과 플롯 연결

위험:
- 여덟 슬롯이 먼저이고 인물이 나중이 됨
- 한 신병이 한 기능만 수행하는 도구로 축소
- 기존 `여덟 역할` 조기 확정 문제 반복

판정: 내부 비교 기준으로만 사용.

## 후보 C — 최초 병기 완전 보존형

원본 병기와 기억이 187년 동안 거의 그대로 유지된다.

장점:
- 진품 추적이 간단
- 유물의 낭만

위험:
- 목재·가죽·시위·자루 등 물리적 보존 문제
- 수리·계승·위조·권리 설계가 무의미
- 후대 역사의 흔적이 약함

판정: 반려.

## 후보 D — 생업 병기 + 후대 역사 변형형

최초 당사자의 직업·환경·해문도 행동에서 병기 형태가 나오고, 187년의 수리·개조·계승·오해가 현재 모습과 기능에 층을 더한다.

장점:
- 여덟 형태가 자연스럽게 다름
- 원본·수리 원본·계승품 구조 활용
- 신병마다 후대 시대와 세력의 흔적
- 실전 무기성과 미스터리 동시 유지

위험:
- 현재 형태와 원형 설명이 복잡
- 수리품을 어디까지 같은 병기로 볼지 논쟁

판정: 채택.

## PM 통합 원칙

- 여덟은 전투력 순위가 없다.
- 각 신병의 강한 기억 채널은 최대 두 개다.
- 각 신병은 실전 병기 기능과 좁은 기환 보조를 가진다.
- 기환 기능은 원사용자의 행동을 반영하지만 주문처럼 재현하지 않는다.
- 본편 시작 시점에는 안정된 후보 일곱 자리와 이중 계승 분쟁 한 자리를 둔다.
- 목재·시위·자루는 교체될 수 있고 핵심 금속·구조·계보·반응으로 정통성을 판단한다.
