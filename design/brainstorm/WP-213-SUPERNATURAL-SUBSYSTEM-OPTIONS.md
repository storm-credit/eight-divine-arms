# WP-213 기환 하위 현상 구조안 비교

상태: `review-complete`

## 후보 A — 독립 술법 학파형

부적술·진법·강령술·영수술 등이 각각 별도 원리와 성장 체계를 가진다.

장점:
- 판타지 다양성
- 전문 문파 설계가 쉬움

위험:
- 고도 판타지화
- 설정집 무한 확장
- 팔대신병의 희소성 붕괴

판정: 반려.

## 후보 B — 전부 오인·사기형

모든 기환 현상을 독·기관·심리·자연 현상으로 설명한다.

장점:
- 역사무협 질감
- 수사와 반전 강화

위험:
- 팔대신병의 핵심 기환과 충돌
- 사용자가 원하는 신비감 부족
- 반복적인 ‘사실 사기였다’ 결말

판정: 오인 가능성의 한 축으로만 채택.

## 후보 C — 제한적 개별 실재형

각 현상 일부가 실제지만 서로 독립 원리로 작동한다.

장점:
- 민담적 다양성

위험:
- 공통 규칙이 없어 편의적 예외 증가

판정: 반려.

## 후보 D — 공통법 하위 유형형

모든 실제 기환 현상은 WP-204의 흔적·매개체·촉발·관측·대가 규칙을 따른다.

하위 유형은:
- 사용하는 매개체
- 준비 방식
- 효과 범위
- 사회적 해석

이 다르다.

장점:
- 요소 누락 없이 상한 통제
- 의학·기관·사기와 복합 원인 가능
- 전문기관 역할 분리

위험:
- 모든 것이 같은 공명 변주처럼 단조로울 수 있음

판정: 채택.

## PM 통합

후보 D를 기본으로 한다.

각 하위 현상은 실제·오인·사기의 세 가능성을 유지하되, 정본으로 실제 효과가 인정될 경우 WP-204의 아홉 필수 항목을 모두 작성한다.
