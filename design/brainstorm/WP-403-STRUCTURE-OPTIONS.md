# 《팔대신병록》 전체 구조 후보 비교

상태: `review`
작업 패킷: WP-403

## 후보 A — 4부 압축형

1. 절강 발화
2. 강남 장부
3. 중원 합증
4. 해문도 결말

장점:
- 속도가 빠름
- 핵심 미스터리가 선명함

위험:
- D6 의술 증거와 D7 이중 계승이 설명 부속으로 밀림
- 서남·외부 무림·지역 보관권이 약해짐
- 동료의 전문 선택이 한 부에 과밀

판정: 반려.

## 후보 B — 5부 권역형

1. 절강
2. 해상
3. 강남·중원
4. 서남
5. 해문도

장점:
- 권역 변화가 명확함
- 여행 서사의 균형이 좋음

위험:
- 강남 상업 증거와 중원 계승 재심을 한 부에 합치면 정치·증거·인물 갈등이 과밀
- 사문경의 계획이 중반에 충분히 드러나지 않음

판정: 보류 후 반려.

## 후보 C — 6부 책임 확장형

1. 청계의 불 — 현장과 인계
2. 절류의 항로 — 해상과 기록 밖 사람
3. 금릉의 저울 — 장부·채무·사람의 물건화
4. 봉서의 재심 — 강호 판정·국가 명령·스승 구출
5. 청맥의 길 — 사인·의술·D7 이중 계승
6. 해문합증 — 공개·보호·분산 보관

장점:
- 주인공의 전문성이 현장→권리→공개 책임으로 확장
- 핵심 동료의 전문 영역이 한 번씩 중심이 됨
- 사문경의 계획이 여러 단계에서 수정·노출됨
- 직접 신병 5개와 간접 신병 3개를 구분 가능
- 각 부에 독립 사건과 관계 결산을 둘 수 있음

위험:
- 이동이 길어질 수 있음
- 부별 설명 주제가 너무 선명하면 교과서처럼 보일 수 있음

보정:
- 이동 중에도 추적·호송·부상·채무 갈등을 유지
- 각 부의 주제어를 본문에서 직접 설명하지 않음
- 앞 부의 미해결 관계가 다음 부의 실무 조건이 되게 함

판정: 채택.

## 후보 D — 8부 신병별 구조

신병 하나당 한 부를 배정한다.

장점:
- 제목과 구조가 직관적
- 개별 신병을 충분히 보여줄 수 있음

위험:
- 보물 수집 퀘스트화
- 여덟이라는 숫자가 서사 리듬을 지배
- 신병을 얻은 뒤 다음 물건으로 넘어가는 반복
- 간접 보관·권리·증언 체계가 무의미해짐

판정: 명시적 반려.
