# WP-204 기환 공통 상한법 구조안 비교

상태: `review-complete`

## 후보 A — 개별 괴이형

각 유물·영물·강시·부적이 독자 원리로 작동한다.

장점:
- 사건마다 새로움
- 민담적 불가해함 유지

위험:
- 예외가 끝없이 증가
- 해결책이 작가 편의가 됨
- 팔대신병과 하위 유물의 격차가 무너짐

판정: 기본 구조로 반려. 설명되지 않는 잔여는 허용하되 공통 감사 규칙은 필요.

## 후보 B — 기의 잔류형

강한 수련·죽음·감정으로 발생한 기가 사람·물건·장소에 남는다.

장점:
- 무공과 자연스럽게 연결
- 신병 기억·영물·잔념을 한 체계로 설명

위험:
- 모든 현상을 ‘기’ 한 단어로 덮을 위험
- 기가 만능 에너지로 팽창

판정: 핵심 재료로 채택하되 단독 사용 반려.

## 후보 C — 신앙·의례형

집단의 믿음과 의례가 현상을 만들거나 강화한다.

장점:
- 종교·민간신앙·정치와 연결
- 부적·진법·장례의 사회적 의미 강화

위험:
- 믿으면 무엇이든 가능한 체계가 될 수 있음
- 실제 신격 강림과 구분이 어려움

판정: 사람의 집중·행동·기억을 조직하는 효과로 제한 채택.

## 후보 D — 공명·매개 혼합형

기환 현상은 다음 요소가 겹칠 때 발생한다.

- 강한 반복 또는 극단적 사건
- 감정·의념·무공 운용
- 흔적을 붙잡는 매개체
- 접근을 여는 촉발 조건
- 현상을 받아들이는 관측자의 몸과 정신

장점:
- 무공·신병·장소·생물·시신을 함께 설명
- 조건과 실패를 만들기 쉬움
- 실제 현상과 사기·오인을 병렬 배치 가능

위험:
- 모든 것을 ‘공명’으로 설명하는 새 만능어가 될 수 있음
- 규칙이 추상적이면 독자가 이해하기 어려움

판정: 채택.

## PM 통합

후보 D를 기본으로 한다.

보완:
- 후보 B의 기·행동 흔적
- 후보 C의 의례가 집중과 경계를 조직하는 기능
- 후보 A의 완전히 설명되지 않는 잔여와 지역별 해석 차이

공명은 에너지 폭발이 아니라 **이미 남은 흔적이 특정 조건에서 감각·행동·생리·기억에 영향을 주는 현상**으로 제한한다.
