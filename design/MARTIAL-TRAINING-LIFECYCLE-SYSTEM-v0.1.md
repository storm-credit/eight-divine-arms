# 《팔대신병록》 무공 수련 생애주기 체계 v0.1

상태: `provisional-canon / S3-01`
설정 분야: D09 무공·수련, D11 부상, D19 연령·생활, D24 세대
총괄: PM 오케스트레이터

## 1. 설계 원칙

무공 수련은 비급·영약·기연을 얻은 순간 끝나는 능력 해금이 아니다.

수련 성취는:

- 성장기 신체
- 교정받은 반복
- 생업·영양·수면
- 스승·동료·안전한 공간
- 부상과 회복
- 실제 임무 경험
- 성격과 판단

이 오랜 시간 축적된 결과다.

## 2. 수련 시작 연령

### 아동기

가능:

- 균형·유연성·놀이형 보법
- 호흡 습관
- 낙법·기초 체력
- 병기 안전·예법

금지 또는 제한:

- 과도한 근력·외공
- 장시간 좌공·폐기
- 성장판·관절을 손상시키는 반복
- 독·약물·극단 절식

아동 천재도 성인 고수의 출력과 판단을 갖지 않는다.

### 청소년기

- 전문 병기·보법·내공 기초
- 신체 성장에 따라 기술을 다시 조정
- 비무·호송 견습·지역 임무
- 과훈련·명성 압박·조기 부상 위험

### 성인기

- 신체와 전문 기술의 통합
- 실전·교육·지휘·계승 역할
- 생업·가족·부양과 수련 시간 충돌

### 중년·노년

- 출력·회복은 감소 가능
- 거리·타이밍·심리·교육·전술은 향상 가능
- 병기·보법·호흡을 몸에 맞게 수정
- 고수의 은퇴·교관·감정가 전환 경로 존재

## 3. 수련 단계의 시간 범위

정확한 연수는 개인·조직·기술마다 다르지만 작가 기준 범위를 둔다.

### 기초 적응

- 수개월~2년
- 자세·호흡·낙법·체력·안전
- 일반 성인이 생업과 병행하면 더 오래 걸림

### 실용 숙련

- 2~5년 이상
- 한 병기·보법·기본 내공을 실제 임무에서 사용
- 지역 무관·군영·표국 실무자의 기반

### 전문 입실

- 5~12년 이상
- 무학의 변화·상성·교정·후배 지도
- 재능이 있어도 실제 몸과 경험이 필요

### 고도 통합

- 10~20년 이상이 흔함
- 내공·병기·보법·감각·전술의 낭비가 줄어듦
- 장기간 부상·실전·교육 경험 포함

### 종사급 성취

- 단순 연수·승수로 결정하지 않음
- 무학을 정리·변형·교육하고 후대 기준을 세움
- 젊은 종사도 가능하지만 매우 드물며 기존 기반·교사·환경 설명 필요

이 범위는 절대 레벨 조건이 아니라 급성장 억제와 인물 경력 감사 기준이다.

## 4. 하루·한 해의 수련

### 일일 수련

- 준비·관절·호흡
- 기본 자세·보법
- 병기 반복
- 짝훈련
- 체력·회복
- 이론·장비 관리

고강도 수련을 매일 같은 양으로 지속하지 않는다.

### 주기

- 고강도일·기술일·회복일
- 계절별 체력·임무·농번기·교단 의례
- 부상 뒤 단계적 복귀
- 시험·비무 전 감량·집중의 부작용

### 생업 병행

표사·군인·장인·선원은 생업 자체가 일부 수련이지만:

- 같은 동작 편향
- 과사용 손상
- 전문 무학 시간 부족

이 생긴다.

## 5. 재능의 분해

`천재`를 하나의 수치로 쓰지 않는다.

재능 축:

- 신체 비율·근력·유연성
- 균형·반응·공간 감각
- 동작 기억
- 호흡·긴장 조절
- 통증·공포 대응
- 관찰·모방
- 전술 판단
- 교육 수용·자기 교정
- 회복력·부상 저항

한 축의 재능이 다른 축의 부족을 자동 보완하지 않는다.

## 6. 체질과 내공

체질은 마법 속성이나 운명 등급이 아니다.

영향 요소:

- 심폐·혈압·체온
- 관절·척추·골격
- 소화·영양
- 수면·불안
- 과거 질병·중독
- 성장기 상태

특정 내공이 맞지 않는다는 것은:

- 호흡 패턴
- 자세·관절
- 열·피로 반응
- 심리적 훈련 방식

이 개인에게 위험하다는 뜻이다.

## 7. 스승과 교정

스승이 제공하는 핵심:

- 오류를 조기에 발견
- 몸에 맞는 변형
- 부상·주화입마 경고
- 실제 거리·상성
- 실패 뒤 복귀법
- 윤리·임무 판단

비급만으로 얻기 어려운 것은 `무엇이 틀렸는지 알아채는 눈`이다.

교사가 나쁘거나 권력 남용을 하면:

- 제자 부상 은폐
- 금단 수련 강요
- 특정 제자 편애
- 수련비·노동 착취
- 후계 경쟁 조작

이 발생한다.

## 8. 자원과 환경

필요 자원:

- 충분한 식사·단백질·수분
- 수면·보온
- 안전한 훈련장
- 병기·보호구
- 약재·의원
- 교관·짝훈련자
- 시간과 생계 지원

가난한 천재가 자원 부족을 의지만으로 무시하지 못한다.

## 9. 내공 수련 단계

내공은 `기 저장량` 하나로 평가하지 않는다.

### 호흡 안정

- 자세·긴장·맥박·복압 조절

### 주천 습관

- 특정 호흡·의념·동작을 연결
- 종문별 설명 언어 차이

### 발경 통합

- 지면 반력·관절·호흡을 짧은 출력에 결합

### 회복·지구력 운용

- 피로 속 호흡·동작 유지
- 손상 치유가 아니라 부담 관리

### 고도 내기 현상

- 접촉·병기·발성의 제한적 전달
- 극소수 고수
- 강한 신체·정신 부담

삼화취정·오기조원은 일부 도가계의 내부 해석이며 공통 레벨이 아니다.

## 10. 외공 수련

- 근력·지구력
- 타격 구조
- 피부·근육·골격 적응
- 낙법·충격 분산
- 도구·하중 작업

외공의 위험:

- 미세 골절·관절 퇴행
- 만성 통증
- 감각 저하
- 젊은 시절 강함과 노년 장애의 교환

철포삼류 외공도 칼·화살·총포를 무시하는 금강불괴가 아니다.

## 11. 경공 수련

구성:

- 발목·무릎·고관절
- 균형·착지
- 시야·경로 선택
- 짧은 가속
- 손·도구·지형 활용

환경별 차이:

- 지붕·도시
- 산악·수풀
- 선상·부두
- 설원·사막

한 환경의 경공 명수가 다른 환경에서 자동 우위가 아니다.

## 12. 실전 경험

실전은 수련보다 무조건 우월하지 않다.

얻는 것:

- 공포·혼란 대응
- 거리·속임수
- 부상 상태 판단
- 후퇴·보호·협공

잃는 것:

- 부상·트라우마
- 잘못된 습관
- 살상 의존
- 교정되지 않은 생존 기술

비무 전적과 살인 경험을 동일한 실전치로 계산하지 않는다.

## 13. 빠른 성장의 허용 조건

가능:

- 이미 가진 유사 기반
- 고급 교사의 집중 교정
- 부상·질병 치료로 본래 능력 회복
- 생활·식사·훈련 환경 개선
- 특정 동작의 신병 잔흔 단서
- 타 분야 전문성이 전술 판단에 기여

불가능:

- 영약 한 알로 수십 년 숙련
- 비급 읽고 즉시 완성
- 타인의 내공을 무상 흡수
- 신병이 몸을 자동 조종해 영구 고수화

대가:

- 기술 편향
- 과사용 손상
- 정서·잔흔 침식
- 다른 수련 시간 상실

## 14. 수련 정체와 퇴보

원인:

- 잘못된 기초
- 부상·질병
- 교관 부재
- 생업·가족 부담
- 공포·트라우마
- 특정 성공 방식 집착
- 체격 변화·노화

퇴보는 내공이 숫자로 증발하는 것보다:

- 타이밍·호흡·지구력 저하
- 통증 회피
- 감각 둔화
- 자신감·판단 불안

으로 나타난다.

## 15. 주화입마

주화입마는 단일 귀신 들림이 아니다.

가능한 상태:

- 과호흡·실신
- 불면·강박
- 자율신경 이상
- 통증·근육 경직
- 망상·공황
- 약물·독 부작용
- 기환 잔흔 침식

진단은 의원·사문 교관·도문·민속 전문가가 다르게 해석할 수 있다.

치료:

- 수련 중단·수면·영양
- 호흡 재교육
- 약물·침구
- 심리적 안정·환경 변화
- 잔흔 매개체 격리

## 16. 폐공·수련 중단·전환

폐공은 마법 능력 삭제가 아니다.

- 특정 호흡법 중단
- 고강도 발경 금지
- 병기·교관 접근 박탈
- 부상으로 출력 상실
- 약물·침술의 일시 억제

재활·다른 병기·교육 역할로 전환 가능하다.

## 17. 인물 무공 이력 장부

주요 인물마다 기록한다.

- 시작 연령
- 교관·계보
- 실제 수련 기간
- 생업 병행
- 중단·부상
- 전문 환경
- 교육 경험
- 현재 유지 가능한 훈련량
- 잘못된 습관

## 18. 교차 감사 질문

- 나이와 수련 기간이 가능한가
- 생계·식사·수면·교관이 있는가
- 비급·기연이 몸과 경험을 대체하지 않는가
- 어린 인물이 성인 출력을 갖는 이유가 있는가
- 중년·노년 고수가 노화 영향을 받는가
- 실전 경험을 살인 횟수로 계산하지 않는가
- 부상 뒤 수련이 실제로 조정되는가
- 주화입마가 편리한 광폭화 장치가 아닌가
