# 《팔대신병록》 신병 권리·계승·집합 현상 체계 v0.1

상태: `provisional-canon`
관문: Gate 2
작업 패킷: WP-209
총괄: PM 오케스트레이터

## 1. 핵심 원칙

신병을 손에 넣은 사람이 모든 권리를 자동으로 얻지 않는다.

다음 권리는 서로 분리된다.

1. 점유권 — 현재 물건을 실제로 지니고 통제하는 상태
2. 소유권 — 재산으로서 반환·양도·담보를 주장할 권리
3. 보관권 — 안전하게 보존하고 접근을 제한할 권리와 책임
4. 사용권 — 병기를 실제로 운용하고 기환 흔적에 접근할 권리
5. 계승권 — 특정 신병 지위와 책임을 다음 세대에 이어갈 권리
6. 열람·감정권 — 기록·기억·구조를 조사할 권리
7. 공개권 — 신병과 원사건 정보를 외부에 발표할 권리
8. 이관권 — 다른 사람·기관에 보관·사용을 맡길 권리
9. 파기·봉인권 — 물건의 기능을 중단·변형·파괴할 결정을 내릴 권리

한 개인이나 세력이 여러 권리를 가질 수 있지만 자동 일체화하지 않는다.

## 2. 점유와 소유

### 점유

물건을 실제로 가진 상태다.

점유자가 할 수 있는 것:
- 이동과 은닉
- 물리적 사용 시도
- 거래와 협상의 사실상 주도

점유가 주지 않는 것:
- 정당한 계승
- 공개와 파기 권한
- 원사건 책임의 자동 승계
- 다른 세력의 인정

### 소유

소유 주장은 다음 근거를 가질 수 있다.

- 상속
- 매매
- 증여
- 담보 변제
- 관부 판결
- 전리·압수
- 분실물 반환
- 장기간 공개 점유와 지역 관행

관부의 소유 판정과 강호의 계승 판정이 다를 수 있다.

## 3. 보관권

보관자는 물건을 지키는 대신 다음 책임을 진다.

- 부식·파손·도난 방지
- 접근자 기록
- 봉인과 인계
- 사용 중 사고와 침식 보고
- 긴급 이동과 피난
- 독립 감정 요청

보관자는 자동으로 사용할 수 없다.

문파·사찰·상단·표국·관부 창고·개인이 보관자가 될 수 있다.

## 4. 사용권

사용권은 다음을 종합한다.

- 병기 계통 숙련
- 신체·정신 상태
- 보관자와 소유자의 허가
- 계보와 책임
- 과거 사용 사고
- 기환 흔적과의 적합성
- 공공 위험

사용 적합성이 높다고 소유·계승·공개권이 생기지 않는다.

사용권은 제한될 수 있다.

- 장소 제한
- 시간 제한
- 특정 동작·기능 제한
- 독립 관측자 동반
- 치료·안정 상태 확인
- 다른 신병과의 근접 금지

## 5. 계승권

계승은 물건뿐 아니라 책임과 기록을 이어받는 행위다.

증거:
- 원본 또는 계승품의 물증
- 보관·사용 계보
- 다른 신병과의 교차 반응
- 원사건 관련 기록
- 전임자의 명시적 이관
- 장기간의 책임 이행
- 무림맹·지역·유족·관련 세력의 인정

혈통과 사승은 강한 증거가 될 수 있으나 자동 결정이 아니다.

계승권은:
- 인정
- 조건부 인정
- 공동 인정
- 논쟁
- 공석

상태가 될 수 있다.

## 6. 열람·감정권

신병을 조사하는 행위도 위험과 권리를 가진다.

필요한 합의:
- 누가 병기를 만지는가
- 어떤 촉발을 시도하는가
- 기록을 어디까지 복사하는가
- 감정 중 파손 책임은 누구에게 있는가
- 발견된 정보의 공개 범위

감정가는 물건을 소유하지 않지만 새로운 비밀과 위험을 알게 되어 별도 책임을 질 수 있다.

## 7. 공개권

원사건과 신병 정보를 공개할 권리는 단순 소유권보다 무겁다.

고려 대상:
- 생존자와 후손의 안전
- 거짓 누명과 명예 회복
- 관부·군사·반역 위험
- 위험한 무공·제작법 노출
- 현재 세력의 폭력 가능성
- 피해 지역과 유족의 이해
- 증거의 충분성

공개는 전부 또는 비공개의 이분법이 아니다.

가능:
- 특정 사실만 공개
- 이름과 위치 비공개
- 복수 기관에 분산 보관
- 일정 조건 뒤 공개
- 위조 가능성을 줄인 요약본 공개

최종 선택은 Gate 4에서 결정한다.

## 8. 파기·봉인권

신병을 파괴하거나 기능을 막는 행위는 다음을 동시에 바꾼다.

- 실전 위험
- 기억 증거
- 계승과 명예
- 다른 신병의 교차 반응
- 현재 보관자의 책임

물리적 파괴가 흔적과 권리까지 완전히 없애는 것은 아니다.

파기·봉인에는:
- 충분한 증거
- 위험 평가
- 관련 권리자의 동의 또는 긴급성
- 대체 기록과 증언 보존
- 실행자의 책임

이 필요하다.

## 9. 권리 이전

### 상속

가문·사문·개인 유언과 관부 판정이 충돌할 수 있다.

### 매매

소유권은 이전할 수 있어도 계승·사용·공개권은 남을 수 있다.

### 담보

전장·상단이 물건을 점유할 수 있지만 기능 사용과 감정은 제한된다.

### 위탁·표행

표국은 운송 중 보관 책임과 제한된 점유를 가진다.

### 압수

관부는 범죄·반역·위험을 이유로 물건을 압수할 수 있지만 강호의 계승 인정은 별개다.

### 전리

전투에서 빼앗았다는 사실은 점유와 위험을 주지만 정당한 계승을 자동 만들지 않는다.

### 발견

분실 신병의 발견자는 보상·비용·정보에 대한 청구를 가질 수 있지만 자동 소유자는 아니다.

## 10. 권리 분쟁 포럼

### 관부

- 법적 소유와 압수
- 범죄·반역·민간 피해
- 상속과 계약

### 무림맹

- 계승·사용·공적·파문
- 회원 간 보관과 중재

### 상단·표국·전장

- 담보·운송·배상·채무

### 문파·세가

- 사승·가산·유품·내부 책임

### 원 협약 관련 계보와 유족

- 역사적 책임·공개·파기

어느 하나도 모든 권리를 단독 확정하지 못한다.

## 11. 신병 간 교차 반응

신병은 관계와 과거 접점에 따라 서로 다르게 반응한다.

가능한 반응:
- 같은 사건의 감각이 강화
- 동작 순서가 이어짐
- 모순된 감정이 동시에 침입
- 진품·계승품의 일부 특징 확인
- 한 신병의 촉발이 다른 신병을 열거나 막음
- 과도한 공명으로 기능 불안정

반응은 단순히 수량이 많을수록 강해지는 구조가 아니다.

어떤 두 병기는 강하게 반응하고, 어떤 조합은 거의 반응하지 않으며, 어떤 조합은 서로 방해할 수 있다.

## 12. 합증

`합증`은 인정된 여러 신병과 기록·보관자·사용자를 통제된 조건에 모아 원사건의 공통 사실과 권리 계보를 교차 검증하는 행위다.

합증의 목적:
- 진품·계승품의 관계 확인
- 공통 기억과 모순 파악
- 원 협약의 위반·공석·왜곡 확인
- 특정 기록·봉인·보관 조건 인증
- 중대한 처리 결정을 내릴 자격 검토

합증은 마법 장치의 자동 작동이 아니다.

필요 조건:
- 신병의 물리적 존재와 감정
- 보관·인계 기록
- 현재 사용자 또는 보관자의 참여
- 안정된 장소와 시간
- 의술·기환·기록 전문가의 관측
- 침식과 폭력을 막을 경계

## 13. 여덟 합증

인정된 여덟 지위가 모두 참여하면 가장 넓은 교차 검증이 가능하다.

가능해지는 것:
- 원사건 기억의 공통 골격 확인
- 고의적으로 지워진 자리와 기록 탐지
- 계승품 간 모순과 원본 결손 비교
- 원 협약이 끝나지 않았거나 잘못 계승되었다는 사실 확인
- 공개·봉인·분산·파기·재정의 같은 중대한 선택을 논의할 정당성 성립

자동으로 일어나지 않는 것:
- 완전한 과거 영상
- 절대무공과 무한 내공
- 고대 인격의 부활
- 황실·무림의 자동 복종
- 병기 스스로의 최종 결정
- 세계를 바꾸는 단일 주문

합증은 **사실과 책임을 더 명확히 할 뿐, 결정을 실행하는 정치·관계·행동을 대신하지 않는다.**

## 14. 집합의 위험

- 기억과 감정의 동시 침입
- 사용자 간 공황·공격성·오인
- 계승품의 불완전 반응
- 병기 파손과 흔적 혼합
- 외부 세력의 습격
- 관부의 압수와 반역 혐의
- 결과 해석을 둘러싼 분열
- 합증 참가자의 책임 독점

여덟을 모으는 일은 최강 전력을 만드는 것이 아니라 가장 큰 정치적·정신적 위험을 한곳에 모으는 일이다.

## 15. 중단과 분산

합증은 다음 상황에서 중단한다.

- 참가자의 정신·신체가 위험
- 병기 하나의 진위가 중대한 의심을 받음
- 폭력·협박·강제 사용
- 관측·기록 독립성이 무너짐
- 병기 파손 또는 촉발 불안정
- 외부 공격

중단 뒤:
- 병기를 다시 분산
- 기록 사본을 복수 보관
- 참가자 치료와 증언 분리
- 오염된 기억과 실제 관측 기록 구분

을 수행한다.

## 16. 결말과의 경계

이 문서는 여덟 합증이 가능한 구조까지만 확정한다.

확정하지 않는 것:
- 실제로 여덟이 모두 모이는가
- 누가 합증을 주도하는가
- 어떤 사실이 최종 확인되는가
- 공개·봉인·분산·파기 중 무엇을 선택하는가
- 팔대신병의 최종 상태

이 결정은 Gate 3~4에서 인물과 결말을 기준으로 선택한다.
