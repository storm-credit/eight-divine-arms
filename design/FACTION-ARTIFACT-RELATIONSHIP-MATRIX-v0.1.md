# 《팔대신병록》 세력–신병 관계 행렬 v0.1

상태: `provisional-canon`
관문: Gate 2
작업 패킷: WP-212
총괄: PM 오케스트레이터

## 1. 관계 유형

- 확보 — 물리적 점유·소유를 원함
- 보관 — 안전한 관리와 접근 제한
- 사용 — 전투·방호·연구 기능 운용
- 검증 — 진위·기억·계승 확인
- 기록 — 문서·증언·원사건 추적
- 보호 — 증인·유족·지역 피해 우선
- 공개 — 진실·명예 회복·책임 추궁
- 은폐 — 위험·체면·정치 때문에 제한
- 무력화 — 봉인·분산·기능 정지·파기
- 중개 — 거래·호송·협상·배상

모든 세력이 확보를 최우선으로 하지 않는다.

## 2. 운영 상한

- 물리적 소유를 상시 최우선으로 하는 A등급 슬롯 최대 4개
- 한 신병 사건의 직접 물리 추적 세력 최대 3개
- 현장 중개·보호·수사 세력 최대 2개
- 한 신병에 직접 얽히는 주요 세력 총합 최대 5개
- 같은 다섯 세력이 모든 신병을 반복 추적하지 않음

## 7. 세력의 목표 변화

목표는 다음 사건이 발생할 때 바뀔 수 있다.

- 신병의 진위가 뒤집힘
- 증인·유족의 피해 발생
- 지도자 또는 대표 인물 교체
- 채무·배상·압수 관계 변화
- 신병 사용 사고
- 원사건 기록 발견
- 다른 신병과의 교차 공명
- 내부 파벌이 주도권 획득

변경 시 기록:
- 이전 목표
- 새 목표
- 직접 원인
- 이익을 얻는 파벌
- 잃는 파벌
- 새 동맹과 적대

정보가 없는데 갑자기 선악 또는 동맹이 바뀌지 않는다.

## 8. 신병 사건 배치 템플릿

개별 신병 사건마다 작성한다.

### 물리 추적 세력 — 최대 3

- 누가 물건을 실제 확보하려 하는가
- 어떤 권리를 주장하는가
- 어떤 인력·노선·무력을 쓰는가

### 현장 기능 세력 — 최대 2

- 수사·보호·호송·감정·치료·중개 중 무엇을 맡는가

### 간접 반응 세력

- 자금·명령·소문·정치 압력만 행사
- 같은 현장에 모두 집결시키지 않음

### 핵심 충돌

- 물건
- 기록
- 증인
- 소유·보관·사용·계승권
- 공개·은폐·무력화
- 배상·민간 피해

중 하나를 중심으로 잡는다.

## 9. 직접 소유가 아닌 서사 동기

세력은 다음을 추적할 수 있다.

- 신병을 운반한 표사
- 원사건을 기록한 장부
- 감정 가능한 장인
- 계승을 인정할 증인
- 사용 사고의 피해자
- 가짜 신병의 유통망
- 파손된 원본 재료
- 합증 장소와 방법
- 공개될 경우 위험한 이름

따라서 신병이 현장에 없어도 팔대신병 서사가 진행된다.

## 10. 부별 운용 원칙

- 한 부의 중심 세력 3개
- 보조 세력 최대 2개
- 지역 B등급 세력 2~4개를 생활·현장 기능으로 활용
- 이전 부의 세력은 후속 정치·정보·채무로 연속성 유지
- 새 부마다 전국 A등급 5개를 전부 교체하지 않음
- 동일한 추격·탈취·도주 결산을 반복하지 않음

## 11. Gate 3 이관

다음은 인물·원사건 설계 뒤 채운다.

- 각 A등급 슬롯의 최종 고유명사
- 대표 인물과 내부 반대자
- 최초 여덟 당사자와 현재 세력의 연결
- 신병별 직접 추적 세력
- 세력별 공개 명분과 실제 목표의 구체 내용
- 금지선을 깨는 조건
