# 《팔대신병록》 현재 팔대신병 여덟 지위 v0.1

상태: `provisional-canon`
관문: Gate 3
작업 패킷: WP-304
총괄: PM 오케스트레이터

현재 명칭은 기능과 구분을 위한 작업명이다. 최종 명칭은 지역·계보·후대 전설의 언어를 반영해 다시 정한다.

## 공통 원칙

- 여덟 사이에 전투력 순위가 없다.
- 각 병기는 실전 병기로서 고유 거리·회수·방호·제압 기능을 가진다.
- 강한 잔흔 채널은 최대 두 개다.
- 기환 효과는 과거 행동의 좁은 단서를 보조할 뿐 주문처럼 작동하지 않는다.
- 원본·수리 원본·계승품의 정통성은 재료·구조·계보·기능·교차 반응으로 판단한다.
- 본편 시작 시 일곱 자리는 안정된 진품 후보, 한 자리는 이중 계승 분쟁 상태다.

# D1. 개문창 — 장창 지위

최초 사용자:
- O1 수항관

최초 형태:
- 군용 장창
- 문과 대열을 지키고 여는 데 사용

현재 상태:
- `수리 원본`
- 최초 창두와 심부 금속은 남아 있음
- 자루·술·손잡이는 여러 차례 교체
- 한 차례 창두가 짧게 갈려 군진창과 개인창의 중간 형태가 됨

강한 잔흔 채널:
1. 동작 — 찌르기보다 밀기·버티기·지렛대·대열 전환
2. 의도 — 명령을 따르려는 힘과 거부하는 결심의 충돌

실전 기능:
- 중거리 통제
- 문·통로·계단·선상 입구 방어
- 다수의 접근로 제한
- 동료 대열의 기준점

기환 보조:
- 사용자가 충분히 숙련되면 과거의 압력 방향과 구조물 저항을 짧게 체감
- 사람의 흐름과 붕괴 직전의 밀림을 감지하는 데 도움
- 관련 장소·연생장과 가까울 때 봉쇄문 구조의 감각이 강화

촉발 조건:
- 창두와 양손이 안정적으로 연결
- 버티기·밀기·문을 여는 동작
- 다수의 사람이 뒤에 있거나 통로가 제한된 상황

사각지대:
- 누가 명령했는가
- 문서와 이름
- 외항 전투

대가:
- 손목·어깨·척추의 누적 손상
- 물러나야 할 때도 자리를 지키려는 강박
- 타인의 명령과 자기 결심을 혼동

후대 전설:
- 어떤 성문도 한 번에 부수는 파성신창
- 실제 정본에서는 자동 파괴 능력 없음

강한 교차:
- D5 연생장
- D8 봉서검과는 명령·거부 잔흔에서 중간 반응

# D2. 절류도 — 선상 도 지위

최초 사용자:
- O2 귀항선주

최초 형태:
- 선상에서 밧줄·돛·목재와 사람을 상대하는 장도
- 절단과 갈고리 보조 구조를 가진 형태 가능

현재 상태:
- `원본에 가까운 수리 원본`
- 칼날의 중심부와 슴베는 원본
- 끝부분과 등 일부가 여러 차례 수리·절삭됨
- 손잡이·칼집은 후대 해상 세력의 양식

강한 잔흔 채널:
1. 동작 — 흔들리는 발판의 균형·절단·회수
2. 감정 — 사람·선박·계류 중 무엇을 끊을지 고르는 죄책감

실전 기능:
- 근중거리 절단
- 선상·부두·좁은 수로에서의 빠른 회수
- 밧줄·그물·무기·장애물 절단
- 상대 병기의 궤적을 흘리고 안쪽으로 진입

기환 보조:
- 밧줄·목재·천·금속에 남은 반복 장력과 절단 방향을 감각적으로 비교
- 흔들리는 선박과 구조물에서 과거 균형 감각이 짧게 겹침
- D7과 공명하면 해문도 외항의 바람·줄·선박 이동 파편이 강화

촉발 조건:
- 흔들리는 발판
- 장력이 걸린 물체의 절단
- 바닷물·비·강한 바람
- 구조와 파괴의 선택이 동시에 존재

사각지대:
- 문서고와 민간 구역
- 정치 명령
- 화재의 최초 원인

대가:
- 현기증·멀미·평형 감각 혼란
- 사람·조직·관계를 끊어야 한다는 충동
- 도주와 구조의 감정이 뒤섞임

후대 전설:
- 물길과 파도를 베는 신도
- 실제로 물을 지배하거나 강을 가르지 않음

강한 교차:
- D7 해명궁
- D3 철조구와 선박 구조 영역에서 보조 반응

# D3. 철조구 — 갈고리 장병 지위

최초 사용자:
- O3 도장선

최초 형태:
- 갈고리·송곳·도끼날 일부를 겸한 선박 수리·제압 장병
- 배·밧줄·목재·사람을 걸고 끌거나 밀어내는 도구

현재 상태:
- `수리 원본`
- 금속 갈고리 머리는 원본
- 자루와 보조 날은 반복 교체
- 후대에 산악·성벽·수로 용도로도 개조됨

강한 잔흔 채널:
1. 신체 감각 — 걸린 무게·당김·미끄러짐·붕괴 저항
2. 공간 감각 — 벽·바닥·통로·빈 공간의 구조적 반응

실전 기능:
- 장거리 걸기와 끌기
- 상대 무기·방패·팔다리 제압
- 통로와 높은 지형에서 이동 보조
- 문·창고·선박 구조물 해체

기환 보조:
- 걸거나 두드린 구조물의 속 빈 곳·부식·균열·이전 하중을 제한적으로 체감
- 과거 반복 사용된 통로·계류 지점에서 손과 어깨에 잔존 저항이 나타남
- D6과 공명하면 화재 구조·의무소 이동의 공간 파편이 강화

촉발 조건:
- 실제 걸림과 하중
- 오래된 목재·금속·벽체
- 사용자가 무게를 버티며 이동

사각지대:
- 사람의 이름과 동기
- 명령과 장부 내용
- 외항 신호

대가:
- 어깨·팔꿈치·손목 손상
- 존재하지 않는 무게를 계속 느끼는 환각
- 숨은 통로와 구조적 약점을 강박적으로 찾음

후대 전설:
- 지맥과 용맥을 끌어당기는 신구
- 실제로 지형을 이동시키지 않음

강한 교차:
- D6 청맥인
- D2 절류도와 선박·부두의 물리 흔적에서 보조 반응

# D4. 묵형척 — 철척·단병 지위

최초 사용자:
- O4 철척장계

최초 형태:
- 화물·직물·목재·상자를 재고 장부함을 지키는 철척
- 모서리와 무게 중심이 단병·제압 도구로 사용 가능

현재 상태:
- `원본`
- 여덟 가운데 물리적 원형이 가장 잘 남은 후보
- 표면 각인 일부가 갈려 나가고 후대 척도·상단 표식이 추가됨

강한 잔흔 채널:
1. 촉각 — 압력·마찰·열·종이·목재·금속의 미세 차이
2. 언어 파편 — 반복해 적고 읽은 이름·수량·인장 일부

실전 기능:
- 근접 막기·누르기·관절 제압
- 좁은 틈의 지렛대
- 물건의 규격·무게·균형 비교
- 문서함·봉인·상자 검사

기환 보조:
- 반복적으로 만지고 고친 물건의 마모·열·압력 차이를 예민하게 느끼게 함
- 문서와 인장의 진위 단독 판별이 아니라 수정·덧댐·불균일한 취급의 단서 제공
- D8과 공명하면 장부 소각·봉인 개방·전달 지연의 촉각·언어 파편이 겹침

촉발 조건:
- 실제 물건·문서·인장의 비교
- 일정한 호흡과 반복 측정
- 손가락과 손목의 정밀 움직임

사각지대:
- 작성자의 진짜 의도
- 문서에 기록되지 않은 사람
- 선상 전투와 구조

대가:
- 손가락 마비·감각 과민
- 이름·수량·진위 분류에 대한 집착
- 사람을 장부 항목처럼 판단하려는 냉정

후대 전설:
- 모든 거짓과 위조를 재는 천형신척
- 실제로 거짓말과 소유권을 자동 판정하지 않음

강한 교차:
- D8 봉서검
- D1 개문창과 명령 문서의 물리 흔적에서 약한 반응

현재 상태:

- 원목은 화재와 세월로 소실
- 최초 장봉의 철제 끝·고리·탄화된 심재 일부를 새 봉 안에 계승
- 지역 사찰과 구조 공동체가 책임 계보를 유지

강한 잔흔 채널:
1. 호흡·동작 리듬 — 다수의 걸음·밀기·버티기
2. 집단 감정 — 공황·의존·안도·희생

실전 기능:
- 비살상 제압과 방호
- 다수 상대의 거리 유지

- 부상자·짐·문을 지지하는 도구

기환 보조:
- 여러 사람이 같은 호흡과 걸음으로 움직일 때 공황과 흐트러짐을 감지
- 사용자가 중심을 유지하면 좁은 범위의 호흡·움직임을 안정시키는 데 도움
- 정신 지배가 아니라 훈련·신뢰·실제 지시에 의존
- D1과 공명하면 봉쇄문·민간 탈출 대열의 감각이 강화

촉발 조건:
- 다수의 사람과 제한된 통로
- 반복 호흡·걸음·봉의 지면 접촉
- 보호·퇴각·구조 목적

사각지대:
- 공격자의 신원과 동기
- 문서와 외항 신호
- 혼자 사용하면 집단 잔흔이 약함

대가:
- 타인의 공포와 호흡 불안을 자기 몸에 받아들임
- 모두가 빠져나갈 때까지 자신은 남아야 한다는 강박
- 심한 탈진·과호흡·무기력

후대 전설:
- 죽은 사람도 일으켜 세우는 활인신장
- 실제 부활·완전 치료 능력 없음

강한 교차:
- D1 개문창
- D6 청맥인과 치료·호흡 안정에서 제한적 반응

- 현재 하나의 병기 지위로 인정되는 핵심 물건은 단도 우선안

강한 잔흔 채널:
1. 신체 감각 — 맥박·호흡·통증·열·경련

기환 보조:

- 과거 사용자의 진단 감각이 단서로 겹칠 수 있음
- 모든 병과 독을 알아내지 못하며 현재 사용자의 의학 지식이 필수
- D3과 공명하면 화재 구조·압궤·의무소의 공간과 부상 잔흔이 연결

촉발 조건:
- 직접 접촉
- 안정된 호흡과 맥진
- 실제 상처·독·통증

사각지대:
- 정치 명령과 장부
- 환자의 거짓말과 타인의 동기
- 접촉하지 않은 사람

대가:
- 통증 전이·부정맥·손떨림
- 치료 우선순위에 대한 강박
- 죽은 환자의 마지막 신체 감각이 자기 기억처럼 남음

후대 전설:
- 한 번 찌르면 생사와 모든 독을 판별하는 신의도
- 실제 만능 진단·완치 능력 없음

강한 교차:
- D3 철조구
- D5 연생장과 호흡 안정 영역에서 보조 반응

- 원본 물질은 약하지만 기능·행동 잔흔과 교차 반응이 강함

강한 잔흔 채널:
1. 소리·바람·거리 감각
2. 의도 — 경고·구조·공격 중 어느 신호를 보냈는가

실전 기능:
- 원거리 사격
- 선상·해안 신호
- 줄·표식·가벼운 물건 전달
- 바람과 움직이는 발판의 계산

기환 보조:
- 특정 장소에서 반복된 바람·소리·거리·신호 방향이 짧게 겹침
- 화살의 자동 유도나 보이지 않는 적 탐지는 불가
- D2와 공명하면 외항의 구조 줄·불붙은 선박·첫 공격의 방향 파편이 강화

촉발 조건:
- 실제 활시위와 장력
- 바람·소리·장거리 시야
- 신호와 공격의 구분이 중요한 상황

사각지대:
- 내륙 문서와 민간 구역
- 신호를 받은 사람의 의도
- 시야 밖의 사건

대가:
- 이명·청각 과민·거리감 혼란
- 먼 표적과 신호에 대한 집착
- 번역·신호의 의미를 잘못 확신

후대 전설:
- 천 리 밖의 사람과 운명을 꿰뚫는 신궁
- 실제 초장거리 자동 명중·원격 대화 능력 없음

강한 교차:
- D2 절류도
- D8 봉서검과 신호·전달 순서에서 약한 반응

# D8. 봉서검 — 직검·문서 사자 지위

최초 사용자:
- O8 봉서객

최초 형태:
- 호신·제압·제한 비무에 쓰는 직검
- 칼집과 손잡이에 서신·인장·봉인 도구를 보관하는 구조가 있었을 가능성

현재 상태:
- `원본 또는 원본성이 매우 강한 수리 원본`
- 검신과 슴베는 원형 보존
- 칼집·인장 고리·문서 수납 구조는 여러 차례 교체
- 특정 감정 기관이 장기간 봉인 보관

강한 잔흔 채널:
1. 동작 순서 — 받기·봉인 열기·전달·지연·발도·제압
2. 언어 파편 — 반복된 명령·이름 일부·서약·거부

실전 기능:
- 근중거리 직선 검술
- 빠른 발도와 제압
- 문서·사람을 호송하는 방어
- 포승·봉인 도구와 함께 사용 가능

기환 보조:
- 반복적으로 같은 손을 거친 봉인·칼집·문서함의 개폐 순서를 감각적으로 비교
- 명령의 내용 전체가 아니라 전달·지연·무단 개봉의 동작 흔적을 보존
- D4와 공명하면 문서고와 지휘소의 명령·장부 파편이 연결

촉발 조건:
- 실제 봉인·칼집·문서함의 개폐
- 전달과 거부의 선택
- 제한된 공간에서의 호송·제압

사각지대:
- 문서의 객관적 진위와 작성자의 속마음
- 외항·화재 구조의 물리 원인
- 긴 대화와 정확한 문구

대가:
- 명령과 자기 의지의 혼동
- 모든 봉인과 전달을 의심하는 편집증
- 공개해야 할 정보와 숨겨야 할 정보를 스스로 결정하려는 오만

후대 전설:
- 모든 봉인과 비밀 문서를 여는 신검
- 실제 자동 해독·진실 판정 능력 없음

강한 교차:
- D4 묵형척
- D1 개문창과 명령·거부의 감정에서 보조 반응

# 교차 반응 기본 지도

## 강한 역사적 짝

- D1 개문창 ↔ D5 연생장
  - 민간 구역 봉쇄와 탈출 대열

- D2 절류도 ↔ D7 해명궁
  - 외항·구조 줄·불붙은 선박·첫 공격

- D3 철조구 ↔ D6 청맥인
  - 화재 구조·의무소·시설과 부상

- D4 묵형척 ↔ D8 봉서검
  - 문서고·인장·명령·장부

## 교차 원칙

- 강한 짝은 같은 사건 영역을 공유하지만 완전한 장면을 자동 재생하지 않는다.
- 다른 조합도 약한 공명·간섭·침식을 일으킬 수 있다.
- 신병 수가 많을수록 무조건 강해지지 않는다.
- 현재 사용자들의 상태와 의도·병기 숙련·환경이 결과를 바꾼다.

# 중심 비중 원칙

Gate 4에서:
- 중심 신병 4~5개
- 보조·간접 신병 3~4개

로 비중을 나눈다.

모든 신병에 동일한 발견·탈취·회상·전투 구조를 반복하지 않는다.

# 최종 명칭 검토 조건

- 명대 병기 용어와 후대 강호의 별칭 구분
- 중국어 원명·한국어 독음·강호 통칭의 자연성
- 기능을 지나치게 직접 설명하지 않는가
- 사방신·팔괘·속성 세트처럼 보이지 않는가
- 현재 보유 세력의 선전과 실제 기원이 다른가

현재 작업명은 Gate 3 종료 전 다시 검토한다.
