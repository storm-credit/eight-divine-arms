# 《팔대신병록》 부적·법구·보패 운영 체계 v0.1

상태: `provisional-canon / S4-01`
설정 분야: D12 기환 상위 법칙, D13 부적·법구·보패, D17 기록·위조
총괄: PM 오케스트레이터

## 1. 설계 원칙

부적·법구·보패는 휴대 마법 아이템의 등급표가 아니다.

실제 효과는 다음이 섞인다.

- 상징·신앙
- 사용자의 집중·암시
- 약재·향·독
- 기관·재료
- 사회적 권위
- 희귀한 기환 공명
- 사기·오인

각 물건은 `무엇을 하는가`보다 `어떤 조건에서 무엇처럼 보이는가`를 먼저 기록한다.

## 2. 용어 구분

### 부적

- 글·도형·인장·재료를 일정 의례로 만든 소모성 또는 제한 사용 도구
- 집중·금기·경고·봉인·신원 표시·심리 안정
- 종교·민간·강호 용례가 다름

### 법구

- 의례·수행·장례·봉인에 쓰는 도구
- 검·령·경·인·경패·향로·염주·목어·종 등
- 무기일 수도 있으나 전투력이 핵심 정의는 아님

### 보패

- 강호에서 희귀하고 특이한 효과가 있다고 인정되는 휴대 유물의 통칭
- 제작품·오래된 법구·기환 매개체가 포함
- 국가나 종교의 공식 분류가 아니라 강호 시장 용어

### 신병

- 역사·잔흔·권리·계승·현재 정치가 복합된 최상위 특수 유물
- 보패보다 강해서가 아니라 사회적·증거적 복합성이 다름

## 3. 부적의 구성 요소

- 종이·비단·목패·금속편
- 먹·안료·피·약재·광물
- 문자·부호·도형
- 제작 시각·방위·의례
- 제작자 인장·계보
- 사용자의 호흡·낭독·행동
- 부착·소각·매장·복용·휴대 방식

하나가 빠졌다고 자동 실패하지 않으며 계보별 핵심 조건이 다르다.

## 4. 부적의 기능 유형

### 표식·권위

- 사찰·도관·문파의 출입·보호·경고
- 사회적 효과가 가장 확실
- 위조 시 신원 사기

### 집중·심리 안정

- 공포·강박·잔흔 접근을 끊는 의례적 초점
- 호흡·암시·반복과 결합
- 모든 정신 질환을 치료하지 않음

### 약물·향·방충

- 향·연기·약재·광물의 실제 물질 효과
- 독·환각·진정과 오인 가능

### 봉인·경보

- 문·상자·시신·법구에 부착
- 뜯김·습기·열·진동을 확인하는 물리 봉인
- 희귀 기환 매개체가 접근자에게 감각 반응을 줄 수 있음

### 기환 공명

- 특정 장소·유물·잔념과 조건부 반응
- 제작자와 사용자 모두 결과를 완전히 통제하지 못함
- 한 계보의 일부 고급 부적에서만 인정

## 5. 제작과 수련

필요 능력:

- 문자·도형
- 재료 지식
- 의례 순서
- 호흡·집중
- 계보의 해석
- 위조 판별

고급 부적 제작은 짧은 주문 암기보다:

- 수년의 필획·재료·의례 훈련
- 실패 기록
- 장소·계절 경험

을 필요로 한다.

## 6. 부적의 실패

- 잘못된 재료·보관
- 습기·불·오염
- 필획·인장 오류
- 사용자가 조건을 모름
- 심리 암시가 반대로 공포 강화
- 약재 과량·알레르기·독성
- 대상 현상을 잘못 진단
- 사기품

실패는 폭발보다 무효·오인·증상 악화가 흔하다.

## 7. 법구의 역할

### 의례 법구

- 수행·예배·천도·장례
- 공동체의 권위와 기억을 조직

### 봉인 법구

- 상자·문·시신·유물 접근 통제
- 물리 자물쇠·인장·부적·보관 규칙 결합

### 감응 법구

- 소리·진동·향·금속·거울을 통해 사용자의 감각 집중
- 기환 현상을 직접 측정하는 과학 기계가 아님

### 호신 법구

- 방호구·경패·단병·염주
- 심리 안정·신원 보증·제한적 기환 차단
- 총포·칼날을 무효화하는 절대 방패가 아님

## 8. 보패의 분류

### 기능성 제작품

- 정교한 기관·약물·재료
- 반복 제작 가능하지만 비싸고 전문적

### 역사 유물

- 특정 인물·사건·사찰·가문과 연결
- 사회적 권위·진위·소유권이 핵심

### 기환 매개체

- 장소·죽음·수련·감정이 오래 누적
- 조건부 잔흔·감각·행동 편향

### 사기·오인 보패

- 기관·독·자석·빛·소리
- 가짜 계보·과장된 전설
- 실제 효과가 일부 있어 완전 사기로 판정하기 어려울 수 있음

## 9. 보패의 희귀도

- 일반 강호인이 평생 진짜 기환 보패를 보지 못할 수 있음
- 유명 문파·세가도 소수만 보관
- 시장에 자주 나오는 것은 가품·기능성 도구·역사 유물
- `보패`라는 광고가 진짜 기환성을 보증하지 않음

대량 생산 보패 산업은 금지한다.

## 10. 사용권과 소유권

- 제작자 소유
- 사문 공동 재산
- 직위 부속
- 개인 기탁
- 사찰 성물
- 관부 압수·증거물
- 피해 공동체 보관

소유자가 사용 조건을 아는 것은 아니며, 사용자가 소유권을 갖는 것도 아니다.

## 11. 수리·복제·위조

수리 시 위험:

- 원재료 교체
- 필획·인장 손상
- 잔흔 매개 부분 파괴
- 후대 사용 흔적 추가

복제품:

- 의례 기능은 재현될 수 있음
- 역사 권위·잔흔·계승권은 자동 복제되지 않음

위조:

- 오래된 종이·먹
- 가짜 인장
- 매장·연기·약품으로 노화
- 실제 약물·기관을 기환처럼 연출

## 12. 감정 절차

1. 출처·계보
2. 재료·제작법
3. 물리 기능
4. 사용 기록
5. 독·약·기관 검사
6. 독립 사용자 반응
7. 기환 가능성
8. 소유·보관·피해 기록

기환 반응 하나만으로 진품을 확정하지 않는다.

## 13. 종교·문파별 관점

- 도문: 부호·의례·호흡·방위
- 불문: 계율·발원·염송·공동체 보호
- 민간법사: 지역 신앙·장례·질병·가정 의례
- 관부: 위조 문서·사기·치안·사교 위험
- 상단: 진위·가격·담보·운송
- 하오문: 가품·도난품·비공식 사용자 정보

## 14. 팔대신병과의 차이

팔대신병은:

- 정확히 여덟 자리의 역사적 책임
- 복수 증거 채널
- 권리 분쟁
- 187년의 수리·계승
- 현재 강호 정치

를 가진다.

일반 보패가 같은 수준의 세계 정치 중심이 되지 않는다.

## 15. 대표 사례 설계 제한

설정집 v2.0에는:

- 부적 4~6종 기능 유형
- 법구 4종
- 보패 6~8개 대표 사례

정도만 둔다.

수백 개 아이템 도감을 만들지 않는다.

## 16. 악마의 변호인 위험

- 부적을 전투 주문 스크롤로 만들지 않는다.
- 보패를 게임 장비 등급으로 만들지 않는다.
- 종교 전문가가 모든 기환을 설명하지 않는다.
- 가품이 항상 무효이거나 진품이 항상 유용하지 않다.
- 피·시신 재료를 쓴다고 자동 사교가 아니다.
- 법구의 사회적 기능을 초자연 효과보다 가볍게 보지 않는다.

## 17. 교차 감사 질문

- 실제·심리·약물·기관·사기 중 무엇이 작동하는가
- 사용 조건과 실패가 있는가
- 누가 만들고 보관·수리하는가
- 사회적 권위와 기환 효과를 구분했는가
- 팔대신병의 희소성을 침해하지 않는가
- 사용자가 효과를 정확히 이해하는가
