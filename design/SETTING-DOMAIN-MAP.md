# 《팔대신병록》 전체 설정 분야 지도 v1.0

상태: `review`
상위 규칙: [`operations/DESIGN-HARNESS.md`](../operations/DESIGN-HARNESS.md)
총괄: PM 오케스트레이터

이 문서는 작품에 필요할 수 있는 설정 분야를 먼저 모두 펼쳐 놓고, 설계 도중 새로운 분야가 반복해서 끼어드는 일을 막기 위한 범위 지도다.

분류:

- `core-now`: 현재 Gate에서 반드시 구조를 확정
- `framework-now`: 지금 경계·용어·자리만 확정하고 세부는 후속 단계
- `later`: 선행 설정이 완성된 뒤 상세 설계
- `backlog`: 사용 가능성이 있으나 아직 필요성 미확정
- `excluded`: 작품 헌법상 의도적으로 제외

# 완전성 판정

현재 지도는 전통 무협, 역사정치, 보물 미스터리, 저강도 기환 작품에 필요한 상위 설정 분야를 모두 포괄하는 작업 기준이다.

이후 새 아이디어가 발견되면 먼저 D01~D27 중 어디에 속하는지 판정한다.

- 기존 분야의 세부라면 새 분야를 열지 않는다.
- 둘 이상의 분야를 바꾸면 `L2` 변경으로 등록한다.
- 작품 헌법을 바꾸면 `L3`로 선행 Gate를 재개방한다.

다음 단계는 전문가별 독립 누락 감사와 Gate 2 범위 동결이다.
