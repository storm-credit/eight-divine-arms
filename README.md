# eight-divine-arms

《팔대신병록》 세계관·서사 설계 저장소.

이 프로젝트는 본문 집필보다 전체 설계를 먼저 완료한다.

## 현재 단계

`Gate 0 — 프로젝트 경계`

현재 작업:

- 프로젝트 헌법 검토
- 오케스트레이션 운영 규약 적용
- 미해결 질문 장부 관리

세력·팔대신병·인물의 상세 설정은 상위 골격 승인 후 진행한다.
