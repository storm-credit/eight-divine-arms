# Eight Divine Arms

한국어 프로젝트명: **팔대신병록**

명나라 가정 연간을 배경으로 한 전통 무협·보물 쟁탈·미스터리 판타지 프로젝트.

이 저장소는 소설 원고보다 먼저 세계관, 세력, 역사, 팔대신병, 인물, 복선, 연표를 설계하고 감사하기 위한 공식 작업 공간이다.

## 운영 원칙

- 아이디어와 공식 설정을 구분한다.
- 상태는 `brainstorm`, `draft`, `review`, `provisional-canon`, `canon`, `archive`로 관리한다.
- 실제 소설 문장은 세계관과 서사 설계가 끝난 뒤 작성한다.
- 주요 반전은 기존 사실을 재해석해야 하며, 임의의 새 사실로 해결하지 않는다.
- 역사적 대사건의 결과는 무림이나 팔대신병 하나만으로 설명하지 않는다.
