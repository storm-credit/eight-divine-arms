# eight-divine-arms

《팔대신병록》 세계관·서사 설계 저장소.

이 프로젝트는 본문 집필보다 전체 설계를 먼저 완료한다.

## 현재 단계

`Gate 1 — 역사·지리·세계 작동 원리`

현재 작업:

- Gate 0 프로젝트 경계 임시 통과
- 1555년 절강 연해 시작안 검증
- 일곱 권력축과 강호·관부 균형 설계
- 권역·경제·물류·강호 관습 설계
- 이동 시간 장부 운영

세력·팔대신병·인물의 상세 설정은 Gate 1 감사 후 진행한다.
