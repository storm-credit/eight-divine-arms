# 《팔대신병록》 미디어 생성 규칙

상태: `provisional-canon`
상위 정책: [`book_writing.md`](book_writing.md)
미디어 허브: [`미디어_index.md`](미디어_index.md)

## 1. 기본 원칙

미디어는 설정을 표현하지만 설정을 임의로 확정하지 않는다.

생성 전 확인:

- 관련 설정 문서 상태
- 인물·세력·신병의 정본 여부
- 시대·지역·복식·병기의 역사 검증 수준
- 금지된 초자연 표현

## 2. 생성 파이프라인

`Draft → Validate → Human Edit → Finalize`

### Draft

복수의 시각·음향·홍보 후보를 만든다.

### Validate

- 세계관 일치
- 시대착오
- 인물 관계와 성격
- 신병 기능과 형태
- 지역 문화의 단순화
- 기존 시안과 중복

### Human Edit

사용자가 분위기·구도·색감·인상을 선택하고 수정한다.

### Finalize

승인된 시안에 설정 문서·버전·사용 범위를 연결한다.

## 3. 이미지 규칙

- 명나라 배경을 청나라 변발·복식으로 혼동하지 않는다.
- 무협 판타지 장식이 역사적 생활감을 완전히 덮지 않는다.
- 모든 문파가 동일한 도복 색상만 다른 형태가 되지 않는다.
- 신병은 크고 빛나는 무기라는 이유만으로 신성해 보이게 하지 않는다.
- 지역의 지형·생업·물류가 의상과 장비에 반영되어야 한다.

## 4. 인물 규칙

인물 시안에는 다음을 기록한다.

- 시점의 나이
- 소속과 신분
- 지역
- 경제적 수준
- 주무기와 휴대 방식
- 부상·흉터·생활 흔적
- 공개 이미지와 실제 성격의 차이

## 5. 신병 규칙

Gate 2 승인 전에는 최종 디자인을 생성하지 않는다.

승인 뒤에도 각 신병은 다음을 시각적으로 구분한다.

- 재료와 제작 시대
- 실제 사용 방식
- 운송과 보관 난도
- 기능의 조건과 대가
- 위조품과 구별되는 특징

무조건적인 발광·룬 문자·서양식 마법 효과는 사용하지 않는다.

## 6. 지도·도표 규칙

- 지도는 실제 지리와 작품용 권역을 구분한다.
- 이동 시간은 `audit/TRAVEL-TIME-LEDGER.md`와 연동한다.
- 세력 영향권은 국경선처럼 고정 표시하지 않는다.
- 공개 영향력·실제 영향력·비밀 거점을 구분한다.

## 7. 파일 관리

각 미디어 산출물에는 다음 메타데이터를 남긴다.

- 제목
- 상태
- 관련 설정 문서
- 생성 목적
- 버전
- 승인 여부
- 폐기 또는 교체 이유

## 8. 현재 허용 범위

- 명나라 가정 연간의 생활·항구·수로 무드보드
- 절강 연해·강남·중원·서남의 지역 분위기 연구
- 문파·세가·상단·표국의 기능별 시각 레퍼런스
- 확정 전 추상적 팔대신병 분위기 연구

현재 정식 캐릭터·신병·문파 디자인은 잠금 상태다.
