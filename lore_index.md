- 설정집 v2.0: `canon`
- 인물·관계 Gate: 설계·교차감사 완료
- 인물 Human Edit: 사용자 승인 대기
- 플롯 Gate: 잠금
- 원고: 잠금
- 기존 시놉시스·6부·60장·장면 카드: `deferred-reference`

## 설정 정본 승인 기준

1. 저강도 확정형 기환
2. 익숙한 명문 35~45% + 고유 기능 세력 55~65%
3. 팔대신병 현재 공통명 + 지역 이형명·원기록명
4. 쇄조도를 반쯤 알려진 재난으로 운용
5. 역사 골격 엄격 + 장르 통칭 가독성 우선

## 정식 작업 용어

- 진원표국
- 검표수
- 인계표사
- 표사
- 표두

- 쇄조도
- 쇄조도 이관 참변
- 쇄조 밀약
- 쇄조합증
