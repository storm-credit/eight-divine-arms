# 후속 상세화 원칙

지금부터 새 상위 설정을 늘리지 않는다.

장별 개요에 실제로 필요한:

- 지역 인물
- 사건 피해자
- 세부 지명·경로
- 전투 장소
- 문서 형식
- 개별 초식·별호

만 해당 장의 기능에 맞춰 추가한다.

정본 승격은 사용자 Human Edit 뒤 수행한다.
