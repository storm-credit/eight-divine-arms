# 설정 정본 기준

- 저강도 확정형 기환
- 익숙한 명문 35~45% + 고유 기능 세력 55~65%
- 팔대신병 공통명 + 지역 이형명·원기록명
- 쇄조도는 반쯤 알려진 재난
- 역사 골격 엄격 + 장르 통칭 가독성 우선

정식 작업 용어:
- 진원표국
- 검표수
- 인계표사
- 표사
- 표두
- 금의위 남방 회수별조
- 쇄조도
- 쇄조도 이관 참변
- 쇄조 밀약
- 쇄조합증

2. 5막·8구간 직조형
