# WP-304 전문가 검증 — 현재 팔대신병 여덟 지위

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 무공·병기 검토

- 여덟 병기는 장창·선상 도·갈고리 장병·철척 단병·봉·의술 단병·각궁·직검으로 구별된다.
- 형태는 O1~O8의 생업·환경·해문도 행동에서 나온다.
- 각 병기는 일반 병기로서 실제 거리·회수·방호·포박·사격 기능을 가져야 한다.
- 기환 효과가 없어도 명병으로 가치가 있어야 한다.
- 동일 숙련자가 모든 신병을 최적으로 사용할 수 없다.

판정: `pass`

## 2. 물리 보존·수리 검토

- 금속 날·창두·갈고리·척·검신·장식·쇠부품은 원본 또는 수리 원본으로 보존 가능하다.
- 자루·손잡이·시위·가죽·옻·목재는 반복 교체된다.
- 교체 부품은 후대 사용자 흔적을 더 강하게 받을 수 있다.
- 수리 원본의 정통성은 핵심 구조·재료·계보·반응을 함께 본다.

현재 권장 상태:
- 원본 또는 수리 원본 6자리
- 인정된 계승품 1자리
- 이중 계승 분쟁 1자리
- 완전 공석 0자리

판정: `pass`

## 3. 기억 채널 감사

각 신병의 강한 채널은 최대 두 개다.

- 개문창: 동작·의도
- 절류도: 동작·감정
- 철조구: 신체 저항감·공간 감각
- 묵형척: 촉각·언어 파편
- 연생장: 호흡 리듬·집단 감정
- 청맥인: 신체 감각·통증 감정
- 해명궁: 소리·거리 의도
- 봉서검: 동작 순서·언어 파편

얼굴·이름·정확한 날짜는 모든 신병에서 취약하다.

판정: `pass`

## 4. 기능 중복 감사

### 개문창과 연생장

- 개문창: 장애 돌파·선 유지·명령과 거부
- 연생장: 집단 호흡·비살상 보호·공황 안정

판정: 분리 가능.

### 철조구와 절류도

- 철조구: 걸기·당기기·구조물과 통로
- 절류도: 절단·회수·불안정 발판과 선박

판정: 분리 가능.

### 묵형척과 봉서검

- 묵형척: 물건·문서의 손질·압력·장부 흔적
- 봉서검: 전달·봉인·명령 순서와 지연

판정: 분리 가능.

### 청맥인과 연생장

- 청맥인: 개인 신체·독·맥박·정밀 제압
- 연생장: 다수의 호흡·대열·공황

판정: 분리 가능.

### 해명궁

- 유일한 장거리·바람·신호 중심 병기

판정: 독립.

## 5. 기환 상한 검토

허용:
- 특정 동작·거리·구조·호흡·재료의 잔흔 감지
- 과거 행동의 신체적 단서
- 좁은 조건에서 안정·강조·교차 반응

금지:
- 창으로 성문을 자동 파괴
- 도로 물과 불을 지배
- 갈고리로 지맥을 끌어냄
- 척으로 모든 거짓말 판별
- 봉으로 다수의 정신을 지배
- 의술 병기로 모든 병을 치료
- 활로 보이지 않는 적을 자동 명중
- 검으로 모든 봉인과 문서를 해독

판정: `pass`

## 6. 대가·침식 감사

각 신병은 사용자의 신체와 심리에 다른 위험을 준다.

- 개문창: 관절·척추 손상, 자리를 지켜야 한다는 강박
- 절류도: 현기증·균형 상실, 관계와 계류를 끊으려는 충동
- 철조구: 어깨·손목 손상, 존재하지 않는 무게와 통로 감각
- 묵형척: 손가락 마비, 이름·수량·진위에 대한 집착
- 연생장: 타인의 공황 흡수, 자신을 마지막까지 남겨야 한다는 강박
- 청맥인: 통증 전이·부정맥·환자 우선순위 죄책감
- 해명궁: 이명·감각 과부하·먼 표적과 신호에 대한 집착
- 봉서검: 명령과 자기 의지 혼동·봉인과 배신에 대한 편집증

판정: `pass`

## 7. 교차 반응 검토

강한 역사적 짝:
- 개문창–연생장: 봉쇄문과 민간 탈출
- 절류도–해명궁: 외항·구조 줄·불붙은 선박
- 철조구–청맥인: 화재 구조·의무소·시설과 부상
- 묵형척–봉서검: 문서고·명령·인장·장부

원칙:
- 짝은 정답 열쇠가 아니라 강한 공통 사건 영역
- 다른 조합도 약하거나 위험한 반응 가능
- 두 병기를 모으면 자동으로 완전 장면이 나오지 않음

판정: `pass`

## 8. 이중 계승 분쟁 검토

해명궁 자리를 우선 분쟁 후보로 둔다.

후보 1:
- 원본 금속 부품·뿔 조각을 계승한 복원궁
- 물증과 보관 계보가 강함
- 실제 잔흔과 사용 적합성은 불안정

후보 2:
- O7의 제자 계보와 신호·궁술·책임을 이어 온 후계궁
- 기능과 공명은 강함
- 원본 물질 증거가 약함

장점:
- 원본 물질과 역할·책임의 정통성 충돌
- 외부 무림·중원 감정가·현재 사용자의 이해관계

주의:
- 다른 자리까지 분쟁을 남발하지 않음

판정: `pass`

## 9. 악마의 변호인

### 위험 A — 이름과 기능이 지나치게 직설적

조치:
- 현재 명칭은 작업명
- Gate 3 인물·지역·전설 언어를 반영해 최종 이름 재작성

### 위험 B — 모든 병기가 수사 도구가 됨

조치:
- 실전 기능과 대가를 먼저 둠
- 일부 신병은 증거보다 방호·이동·신체 침식이 중심

### 위험 C — 여덟을 모두 본편에서 동일 비중으로 다룸

조치:
- Gate 4에서 중심 신병 4~5개, 보조·간접 신병 3~4개로 비중 차등
- 모든 신병에 동일한 발견·전투·회상 구조를 반복하지 않음

### 위험 D — 유물 명칭이 한국식 판타지 한자어로 과장

조치:
- 명나라·강호 기록·지역 구전의 명명 층을 분리
- 최종 명칭은 역사·언어 검토 뒤 확정
