# WP-210 전문가 검증 — 세력 분류와 대표 의석

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 무협 장르 검토

- 구파일방과 오대세가는 독자 진입점으로 유지한다.
- 명단 밖 지역 문파·특수 종문·유력 세가가 존재할 수 있다.
- 의석은 전투력만이 아니라 전통·보증·공적·지역 대표성을 반영한다.

판정: 대표 의석형 통과.

## 2. 강호 제도 검토

무림맹 참여 상태:
- 핵심 의석
- 일반 회원 세력
- 제휴·보증 세력
- 독립·비회원
- 정지·제명·적대

무림맹의 효력:
- 회원망의 보증·중재·공적·거래에 강함
- 외부 무림·국가기관·독립 교단에 자동 강제력 없음

판정: WP-201과 일치.

## 3. 정사마 분류 검토

### 정파

- 공개 계보와 사문 보증
- 무림맹 또는 지역 공통 법도 참여
- 민간 피해와 배상에 대한 공개 책임

정파가 도덕적으로 선하다는 뜻은 아니다.

### 사파

- 정파의 계보·법도·보증망 밖 또는 일부 거부
- 비공식 경제·지역 통행권·흑도 질서와 연결 가능
- 자동 범죄자나 악인이 아님

### 천마신교

- 중원 정사마 분류 밖의 독자 종교·행정·무공 공동체
- 사도련의 상위 지휘자나 사파 총본산으로 처리하지 않음

### 혈교·금단 결사

- 혈액·육체·시신·강제 침식 등 금단 수법 때문에 광범위한 적대 대상
- 천마신교와 별개
- 모든 사건의 배후로 사용하지 않음

### 외부 무림

- 중원 분류를 그대로 적용하지 않음

판정: 통과.

## 4. 의석 교체 검토

교체 사유:
- 계승 단절
- 장기간 공석
- 중대한 법도 위반
- 보증·배상 능력 붕괴
- 지역 대표성 상실
- 명백한 반역·대량 민간 피해
- 다른 세력의 장기적 공적과 회원 지지

교체는 단일 비무로 결정하지 않는다.

정지와 공석이 교체보다 먼저 올 수 있다.

판정: 통과.

## 5. 숫자 조직 감사

- 구파일방과 오대세가는 역사적 명칭과 의석 구조이므로 유지 가능
- 천하십대고수·십이명문 등 새 고정 숫자 집단은 별도 심사
- 숫자 밖 세력이 약하다는 뜻이 아님
- 실제 명단은 기능 슬롯 검토 뒤 결정

판정: 통과.

## 6. 악마의 변호인

### 위험 A — 무림맹이 모든 강호를 지배

조치:
- 회원·제휴·비회원 구분
- 지역 관행과 독립 세력의 권한 유지
- 관부·신교·외부 무림에 자동 강제력 금지

### 위험 B — 의석 정치가 현대 의회처럼 보임

조치:
- 선거·임기보다 전통·보증·맹회 합의·사실상 인정 중심
- 공식 규정과 관습이 일치하지 않을 수 있음

### 위험 C — 정파와 사파가 다시 선악 이분법

조치:
- 세력별 민간 피해·책임·경제·내부 파벌을 별도 기록
- 명칭보다 실제 행동을 평가
