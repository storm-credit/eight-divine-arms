## 7. 악마의 변호인 최종 반론

### 반론 1: 설정이 너무 많아 본문을 압도한다

판정:
- 설정집은 넓게 유지
- 한 사건·권역에서 사용하는 A세력·기환·신병 수를 제한
- 원고 단계에서 시야와 정보량을 통제

설정 삭제보다 사용 밀도 통제가 적절하다.

### 반론 2: 절차와 권리가 무협의 속도감을 죽인다

판정:
- 절차는 싸움과 선택의 원인·대가를 만든다.
- 본문에서 모든 절차를 설명하지 않고 갈등으로 드러낸다.
- 설정집 단계에서는 필요함.

### 반론 3: 기환 강도가 너무 낮아 팔대신병의 경이가 약하다

판정:
- 경이는 압도적 화력보다 잔흔·역사·권리·교차 진실에서 발생
- 실전 병기로서의 개성과 대가가 존재
- 장르 정체성에 부합.

### 반론 4: 역사 고증과 장르 용어가 충돌한다

판정:
- 사용자 결정으로 장르 용어 유지
- 작품 내부 통칭과 실제 공식 문서 표현을 필요 시 구분
- 실제 관직·지명은 사용 전 국소 감사

### 반론 5: 구파일방·오대세가 최종 명단이 없으므로 미완이다

판정:
- 제도·의석·교체·대표성 규칙은 완료
- 전체 명단은 작품 사용 밀도와 취향을 고정하는 선택이므로 Human Edit 또는 인물·세력 상세화 단계
- 상위 설정 누락 아님.

## 9. 정본 승격 전 사용자 Human Edit 항목

사용자가 취향으로 조정할 수 있는 상위 항목:

1. 기환 체감 강도
2. 익숙한 문파·세가의 최종 비중
3. 팔대신병 작업명과 이형명
4. 쇄조도 원사건을 설정 정본에 얼마나 전면 배치할지
5. 설정집의 역사 고증과 장르 편의 균형

수정이 있어도 전면 재설계가 아니라 관련 모듈만 재개방한다.
