# WP-202 전문가 검증 — 무공 존재론과 전투 상한

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 전통 무협 검토

통과 조건:
- 내공·경공·점혈·호신강기 등 장르 문법을 유지
- 단일 수치 레벨이 모든 전투를 결정하지 않음
- 사승·수련·상성의 가치 유지

판정:
- 층위 혼합형 통과
- 현대식 절대 경지표 반려

## 2. 역사·군사 검토

핵심 위험:
- 개인 고수가 군진·화살·화기·장기 포위를 무효화하는 구조
- 검·도를 모든 병장기보다 우월하게 묘사
- 병기의 휴대·보수·탄약·보급 무시

판정:
- 장병과 단병은 거리와 대형에 따라 상호 보완
- 방패·사격·투척·그물·화기·지형은 고수에게도 유효
- 고수는 짧은 국면을 뒤집을 수 있으나 전장을 지속적으로 지배하지 못함

## 3. 신체·의술 검토

무공이 강화할 수 있는 것:
- 순간 출력
- 균형과 낙법
- 호흡과 피로 관리
- 통증 억제
- 반응과 감각 집중
- 힘의 전달과 충격 분산

무공이 제거하지 못하는 것:
- 출혈
- 골절과 장기 손상
- 감염
- 수면 부족
- 탈수와 기아
- 독성 물질
- 반복 충격의 누적

판정:
- 무공 사용 비용은 WP-203에서 부상 유형별로 확정

## 4. 기환 경계 검토

허용:
- 접촉 또는 병기 매개형 내기
- 순간적인 감각 확대와 기세 압박
- 숙련자의 호흡·의념이 동작 효율에 미치는 영향
- 팔대신병이 기존 한계를 짧게 넓히는 현상

제한 또는 반려:
- 상시 비행
- 무제한 원거리 기탄
- 맨몸으로 총포·화살을 지속적으로 무효화
- 경지만 높으면 독·매복·군진이 무의미
- 기억을 본 것만으로 완전한 무공 복제

## 5. 경지 언어 검토

### 외부 평판

- 하수·숙수·고수
- 일류·절정급
- 종사·대종사급 존칭

성격:
- 전적·평판·공적·정보 정확도에 따른 사회적 평가
- 지역과 평가기관에 따라 달라짐

### 기예 숙련

- 초규문경
- 등당
- 입실
- 순숙 또는 노화순청
- 화경·등봉조극 등 극찬

성격:
- 검술·신법·의술 등 특정 분야의 숙련
- 전투 전체 출력 등급이 아님

### 종문 내부 현상

- 주천
- 삼화취정
- 오기조원
- 불문·마도·외부 무림의 독자 용어

성격:
- 해당 계보의 수련 이론과 경험을 설명
- 서로 자동 환산하지 않음

판정:
- 다층 언어 체계 통과

## 6. 병장기 검토

채택 구조:
- 공통 신체 원리
- 병장기 계통별 전문성
- 전술 역할
- 지역·생업·조직 특화
- 개인의 체격·상처·경력에 따른 변형

금지:
- 무기별 절대 강약표
- 비주류 무기를 기괴한 필살기 소품으로만 사용
- 상위 고수는 아무 무기나 즉시 완전 숙련

## 7. 악마의 변호인

### 위험 A — 규칙이 많아 전투가 교범화

조치:
- 설정집에는 프로필을 기록하되 본문에서는 목표·지형·실수·비용으로 보여줌

### 위험 B — ‘제한적’이라는 말로 예외 남발

조치:
- 모든 초상 현상은 조건·지속·대가·실패를 기록
- 신병 효과도 동일 감사 대상

### 위험 C — 경지 언어가 결국 숨은 레벨표가 됨

조치:
- 인물 프로필에서 단일 종합 등급 사용 금지
- 최소 세 개 능력축과 약점을 병기

### 위험 D — 최고수의 위엄 부족

조치:
- 최고수는 불가능을 하는 사람이 아니라, 가능한 동작을 더 빠르고 정확하며 적은 낭비로 연결하는 사람
- 준비된 짧은 교전에서는 압도적일 수 있으나 장기전 비용은 남음
