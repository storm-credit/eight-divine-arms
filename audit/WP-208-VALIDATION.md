# WP-208 전문가 검증 — 신병의 기억·감정·동작 흔적

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 미스터리 검토

- 신병은 과거를 완전 재생하지 않는다.
- 강한 보존 채널이 병기마다 다르다.
- 결정적 사실은 여러 신병·문서·물증을 교차해야 한다.
- 한 번 접근한 기억도 사용자의 언어와 기대 때문에 다르게 서술될 수 있다.

판정: 다층 잔흔형 통과.

## 2. 무공 검토

신병이 보존할 수 있는 것:
- 손의 압력과 위치
- 발의 순서와 중심 이동
- 호흡의 시점
- 충격과 저항
- 공격을 멈추거나 바꾼 순간의 감각

보존하지 못하는 것:
- 사용자의 근력·유연성·경맥 상태
- 수십 년의 상황 판단
- 상대의 보이지 않은 움직임
- 비급 전체와 이론의 완전한 설명

판정: 동작 단서 제공은 허용, 완전 복제는 반려.

## 3. 의술·정신 검토

접근 비용:
- 두통·구토·현기증
- 근육과 관절의 잘못된 모방
- 불면·악몽
- 감정 전염과 출처 혼란
- 자기 기억의 공백 또는 재배열
- 신병 사용에 대한 의존

대응:
- 접근 시간 제한
- 기록과 동료의 현실 확인
- 수면·식사·치료
- 병기와 거리 두기
- 의례·진법을 통한 집중과 차단

판정: 통과.

## 4. 물질·수리 검토

- 손잡이·날·자루·시위·장식 등 흔적 보존 부위가 다를 수 있다.
- 파손과 수리는 일부 흔적을 지우거나 접근 순서를 바꾼다.
- 새 부품은 새 사용자의 흔적을 더 쉽게 받는다.
- 계승품은 원본 재료에 남은 일부 흔적만 이어받는다.

판정: 통과.

## 5. 왜곡 검토

왜곡 원인:
- 후대 사용자 흔적의 중첩
- 반복 접근으로 기억을 특정 해석에 맞춤
- 관측자의 공포·욕망·죄책감
- 독·부상·수면 부족
- 수리·파손·재료 교체
- 의도적 의례와 가짜 촉발 장치
- 익숙하지 않은 언어와 지역 감각

판정: 통과.

## 6. 악마의 변호인

### 위험 A — 필요한 정보를 편리하게 보여줌

조치:
- 개별 신병마다 보존 채널·사각지대·촉발 조건을 사전에 장부화
- 플롯 작성 뒤 기억 기능을 소급 추가하지 않음

### 위험 B — 기억이 너무 주관적이라 무의미

조치:
- 반복 가능한 물리 감각 또는 행동 결과를 최소 하나 포함
- 여러 접근자의 공통 요소를 기록

### 위험 C — 죽은 인격과 대화하는 것처럼 변함

조치:
- 질문에 답하는 안정적 인격 금지
- 언어 파편은 당시 반복된 실제 말 또는 사용자의 재구성일 수 있음

### 위험 D — 주인공만 특별히 완전 접근

조치:
- 접근 적합성은 혈통이 아니라 훈련·상처·경험·사용 관계로 설명
- 다른 인물도 다른 채널에 더 잘 접근 가능
