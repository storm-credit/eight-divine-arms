# 《팔대신병록》 전문가 합동 설정 완전성 감사 v1.0

상태: `review-complete`
기준 문서: [`design/SETTING-DOMAIN-MAP.md`](../design/SETTING-DOMAIN-MAP.md)
총괄: PM 오케스트레이터

목적:

- 기존 설정에서 빠진 상위 분야를 찾는다.
- 필요한 설정은 현재 지도에 편입한다.
- 이름과 사례만 다른 중복 설정은 합친다.
- 지금 설계할 것과 후속 상세화를 구분한다.
- 설정이 완성될 때까지 중간 추가가 반복되는 구조를 막는다.

판정 등급:

- `ADD-NOW`: 현재 Gate에 필수 편입
- `FRAME-NOW`: 지금 상위 규칙만 필요
- `LATER`: 후속 Gate에서 상세화
- `MERGE`: 기존 분야에 통합
- `EXCLUDE`: 의도적 제외

# Round 1 — 독립 전문가 감사

## A01. 전통 무협 장르 에이전트

### 발견한 공백

1. 무공의 이름은 많지만 **내공·외공·경공·점혈·호신강기 등의 공통 작동 규칙**이 아직 없음.
2. 문파·세가·무관·산장·방회·단맥전승의 제도적 차이가 부족함.
3. 사승·파문·폐공·비무·복수·중재의 법도가 상위 구조에 없음.
4. 후기지수·무림대회·서열록을 인물 명단 문제로만 보면 안 되고 명성 제도로 먼저 설계해야 함.
5. 병기와 맨손, 암기와 독, 협공과 진법의 상성을 설명할 전투 원칙 필요.

### 판정

- 무공 상위 체계: `ADD-NOW`
- 강호 신분·법도: `ADD-NOW`
- 명성·대회·서열 제도: `FRAME-NOW`
- 실제 후기지수·고수 명단: `LATER`

### 편입 위치

## 검사 2. 기환 요소 × 의술·물류·경제

질문:

영수·영초·보패·강시가 존재할 경우 어떤 새 산업과 문제를 만드는가?

발견:

- 영초는 채집권·보존·가공·위조·밀렵 문제를 만듦.
- 영수는 사육 가능한 가축이 아니라 희귀 개체로 제한해야 함.
- 보패는 대량 생산이 불가능해야 팔대신병과 경제가 붕괴하지 않음.
- 강시는 운송 가능한 전투병기가 되면 장르가 바뀌므로 극히 제한해야 함.

판정:

새 상위 분야 추가 없음. D11~D15·D20에 통합.

## 검사 3. 국가·세력 × 기록·미스터리

질문:

진실이 밝혀졌을 때 누가 무엇을 증거로 인정하는가?

발견:

- 무림맹의 진위 판정과 관부 법정의 증거 기준이 달라야 함.
- 황실 기관 내부에서도 문서의 효력과 비밀 명령의 정당성이 다름.
- 신병 기억은 정치적으로 강력하지만 법적 증거로 자동 인정되지 않아야 함.
- 세력은 진실보다 소유권·관할·명예·배상을 두고 싸울 수 있음.

판정:

새 상위 분야 추가 없음. D04·D05·D17·D22에 통합.

## 검사 4. 인물·서사 × 생활·제도

질문:

설정이 인물의 선택과 장면으로 전환될 수 있는가?

발견:

- 신진기수 제도는 명예뿐 아니라 혼인·사승·후원·표행 계약에 영향.
- 부상은 무공뿐 아니라 생계·가문 계승·혼인·여행을 바꿈.
- 문파 파문은 숙소·수입·신원·보호망을 잃는 사회적 처벌이어야 함.
- 진실 공개는 죽은 사람의 명예보다 살아 있는 후손의 생존에 영향을 줌.

판정:

새 상위 분야 추가 없음. D05·D11·D18·D19·D24에 통합.

구파일방·오대세가·팔대신병·십대고수·십이명문 등 숫자 조직이 겹치면 게임 세계처럼 보임.

조치:

- 현재 문파·세가·신병 이름은 후보 자료
- 기능 슬롯과 시스템 승인 뒤 최종 명단 결정

# 완전성 결론

전문가 12개 관점과 교차 검사 4회를 기준으로, 현재 D01~D27 지도 밖에 **당장 별도의 상위 분야로 추가해야 할 설정은 발견되지 않았다.**

- 필요한 상위 설계 분야의 목록은 충분히 포괄적이다.
- 앞으로 떠오르는 아이디어는 대부분 D01~D27의 하위 항목으로 분류할 수 있다.
- 새 상위 분야를 열려면 기존 지도에 수용할 수 없는 이유와 두 분야 이상의 구조 변경을 증명해야 한다.
