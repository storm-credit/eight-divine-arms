# 《팔대신병록》 조기 플롯·원고 참고안 격리 감사

상태: `review-complete / pass`
총괄: PM 오케스트레이터

## 1. 목적

설정집 완성 전에 만들어진 주인공·적대자·6부·60장·장면 카드가 설정 정본을 역으로 구속하지 않도록 문서 위계와 사용 경로를 점검한다.

## 2. 격리 대상

- 심소연 주인공 기초안
- 핵심 동료·경쟁자·적대자안
- 사문경 중심 적대 구조
- 6부 대서사
- 전체 시놉시스
- 60장 개요
- 장면 카드
- 제1부 초고 시도
- 특정 결말 실행안

## 3. 유지 가능한 세계 설정

조기 플롯 문서에서 설정집에 독립적으로 남길 수 있는 것은 다음뿐이다.

- 1368년 쇄조도 이관 참변의 역사 골격
- 최초 여덟 당사자의 기능적 역할 구조
- 팔대신병 여덟 지위와 잔흔·권리 상위 규칙
- 1555년 명 가정 연간의 현재점 후보
- 해상·상업·표국·기록·국가 관할의 세계 구조

이 항목은 통합 설정집과 S모듈에서 다시 정의됐으므로 조기 플롯 파일을 직접 참조할 필요가 없다.

## 4. 정본에서 제외하는 요소

- 심소연이 반드시 주인공이라는 결정
- 진원표국이 반드시 주인공 소속이라는 결정
- 철조구가 반드시 첫 신병이라는 결정
- 사문경이 반드시 최종 적대자라는 결정
- 청계 화재가 반드시 도입 사건이라는 결정
- 여섯 부의 지역·신병 순서
- 쇄조합증의 구체적인 결말 방식
- 고학선 실종·방화 책임·범인 구조
- 인물별 연령·관계·부상·신뢰 변화

## 5. 문서 위계 점검

교정 완료:

- `book_writing.md`가 설정집 v2.0을 최우선 기준으로 지정
- `lore_index.md`가 통합 설정집과 최종 감사를 전면 배치
- `원고_index.md`가 원고·장면·플롯 잠금을 명시
- `PM-PROGRESS-DASHBOARD.md`가 Human Edit 대기로 표시
- `SETTING-BIBLE-v2.0-MASTER.md`가 특정 주인공·6부·60장 구조를 참고안으로 강등

## 6. 사용 규칙

후속 인물·플롯 Gate에서 기존 자료를 사용할 때:

1. 기존안을 자동 복원하지 않는다.
2. 최소 세 가지 새 구조와 함께 비교한다.
3. 설정집 v2.0에 맞는지 다시 감사한다.
4. 기존안의 장점만 분리 채택할 수 있다.
5. 사용자 승인 전 정본화하지 않는다.

## 7. 위험 점검

### 위험 A — 익숙한 기존 인물에 끌려감

대응:
- 인물 설계 Gate에서 역할·계층·권리 충돌부터 새로 설계
- 심소연은 후보 P안 중 하나로만 취급

### 위험 B — 여덟 신병이 기존 6부 순서에 고정

대응:
- 사용 신병 수·등장 순서·중심 비중을 플롯 단계에서 재선정

### 위험 C — 쇄조합증이 유일 결말로 고정

대응:
- 결말 후보 최소 4안 재비교
- 설정집의 공개·봉인·파기 권리 체계만 유지

### 위험 D — 조기 장면 카드가 설정을 과도하게 구체화

대응:
- 장면 카드의 고유 사건·인물은 증거가 아니라 참고 예시
- 설정 수정 근거로 인용 금지
