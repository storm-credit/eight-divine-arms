# 1. 감사 결론

Gate 4의 선택 결말·6부 플롯·주적 계획·복선 구조는 Gate 0~3의 작업 기준과 치명적으로 충돌하지 않는다.

판정:

- 치명적 충돌: 0
- 중간 위험: 8
- 경미한 동기화: 11
- Gate 5 진입 가능

---

# 2. 선행 설정 감사

## 강호·국가 관계

결말의 `분산 합증`은 국가 밖의 새 초법정부를 만들지 않는다.

- 관부는 공식 사건·압수·구금 기록을 수정
- 무림맹은 계승·공적·판정을 재심
- 상단·표국은 채무·인계 장부를 정정
- 의가·사찰·공동체는 독립 부속 기록을 보관

판정: 통과.

## 무공·전투 상한

- 소연은 최종 결투에서 절대고수를 쓰러뜨리지 않음
- D3는 구조·지형 제어에 제한 사용
- 군대·화기·봉쇄·협공이 고수를 계속 제한
- 부상 누적과 역할 교대 존재

판정: 통과.

## 기환 상한

- 신병 합증이 완전 영상·부활·절대무공을 만들지 않음
- 잔흔은 감각·동작·감정 파편과 사각지대를 유지
- D7 두 후보의 반응도 완전체 합성으로 이어지지 않음

판정: 통과.

## 증거 체계

- 중대한 결론은 문서·물증·증언·신병 잔흔 중 최소 두 계열로 교차
- 사문경 자료도 완전 거짓이 아님
- 현재 범죄와 원사건 책임을 분리

판정: 통과.

# 3. 플롯 구조 감사

## 위험 M1 — 여덟 신병 수집 퀘스트화

상태: 완화됨.

- 직접 중심은 D3·D4·D6·D7·D8
- D2는 현 사용자가 보유한 반복 증언
- D1·D5는 이동하지 않고 기록·대표로 참여
- 주인공 일행은 신병 소유 집단이 아님

추가 Gate 5 조건:

본문 홍보·부제·장별 목표에서 `다음 신병 찾기` 표현을 반복하지 않는다.

## 위험 M2 — D7 때문에 팔대신병이 실제로 아홉 물건처럼 보임

상태: 관리 필요.

보정:

- `해명궁 자리의 두 현존 증거`로 표현
- 복원궁·전승궁을 두 개의 별도 신병 번호로 세지 않음
- 하나로 합쳐 완전체를 만들지 않음
- 독자가 혼동하지 않도록 최초 설명 시 권리 분리와 물건 수를 명시

## 위험 M3 — 6부가 주제별 교과서처럼 보임

상태: 중간 위험.

보정:

- 각 부에서 이전 부의 현재 범죄·추적·부상·관계 갈등을 지속
- `장부의 부`, `의술의 부` 같은 설명형 본문 구분을 피함
- 한 장면에 제도 설명과 감정 결산을 동시에 몰지 않음
- 지역 인물의 현재 욕망을 원사건 설명보다 우선

## 위험 M4 — 이동 경로와 기간

상태: 작업 기준상 가능, 역사 감사 필요.

- 약 14개월
- 절강→금릉→중원 경계→서남→장강 하행→해문도
- 가장 긴 이동에 6~8주 확보

Gate 5에서 실제 지명·수로·계절을 검증한다.

## 위험 M5 — 최종부 증거·신병·세력 과밀

상태: 중간 위험.

보정:

- 최종부 이전에 각 증거의 의미를 대부분 확정
- 최종부는 새 설정 해설보다 인계·보호·압수 충돌에 집중
- D1·D2·D5는 짧은 독립 증언으로 사용
- 모든 대표에게 긴 연설을 주지 않음
- 공개본·제한본·봉인본을 시각적 물건과 운송 행동으로 보여 줌

# 4. 인물 감사

## 심소연 필수성

통과.

- D3 현장 상태와 인계 연속성
- 표국·관부·상단 사이의 실무 언어
- 과거 기록 조작의 거울
- 최종 단일 인계서 거부

그녀가 빠지면 결말의 증거 연속성과 주제 선택이 성립하지 않는다.

## 동료 기능 도구화 위험

상태: 완화됨.

- 진소무: 현재 신원 동의와 보호본
- 두약란: 공적 사인·환자 비밀 분리
- 유청하: 무림맹 판정 재심
- 나해성: 지역 보관과 외부 감정
- 고정문: 국가 공식 사본과 기관 책임

Gate 5 조건:

각 인물은 자기 전문 판단 때문에 한 번 이상 사건을 악화시키거나 비용을 치른다.

## 고학선 면죄부 위험

상태: 완화됨.

- 방화범이 아님
- 구조 행동 존재
- 봉인 변경·침묵·책임 전가 인정
- 생환해도 소연을 면죄하지 않음

## 사문경의 사상적 균형

통과 조건부.

- 교차 감정과 위험 기억 제한의 필요성은 유효
- 참가자 선정·편집·허가권 독점이 패배 원인
- 최종 자료 일부가 합증에 기여
- 단순 광인·위선자로 축소 금지

---

# 5. 현재 범죄 감사

## 위험 M6 — 사문경이 결국 모든 범죄의 원인처럼 보임

상태: 분리 완료.

- 주계평: 직접 방화·살상·밀무 은폐
- 사문경: 비밀 회수 의뢰·사후 보호·구금·증거 편집
- 국가 강경파: 비밀 구금·일괄 압수
- 상단 중앙: 조직 신용 보호와 내부 감사 지연
- 고학선·소연: 보호 목적의 절차 위반과 책임 전가

## 위험 M7 — 청계·해문도 화재 반복

상태: 의도적 거울, 차별화 필요.

차이:

- 청계: 정보 부족·단일 현장·사후 분산
- 해문도: 사전 분산·공개 이의·복수 구조 책임
- 원사건: 명령 충돌과 통제 붕괴
- 현재 최종 화재: 주계평의 개인 증거 제거

Gate 5 장면 규칙에서 동일한 붕괴·구조 동선을 반복하지 않는다.

# 6. 결말 감사

## 위험 M8 — 현대적 위원회·개인정보 제도처럼 보임

상태: 중간 위험.

보정:

- 새 전국기관 창설 금지
- 기존 관부 판결·무림맹 재심·상단 장부 정정·사찰 명부·사문 서약으로 표현
- `정보 권리` 같은 현대 추상어보다 봉인·인장·증인·인계·보증·당사자 동의를 장면으로 사용
- 공개본·부속본·봉인본 명칭은 작품 용어 감사 뒤 다듬음

## 감정 만족

통과.

- 주인공과 스승 생존 가능
- 과오의 실제 비용 존재
- 사문경 처형으로 단순 마무리하지 않음
- 피해 인정과 현재 보호를 함께 달성
- 세계가 완전히 고쳐지지 않음

## 비극 의존

통과.

- 핵심 동료 전멸 없음
- 주인공 희생 없음
- 대량 사망은 과거 사건과 현재 범죄의 피해이며 결말 감동을 위한 추가 희생으로 남발하지 않음

# 7. 복선 감사

- 결정적 단서가 최종부에 처음 등장하지 않음
- 각 가짜 가설은 합리적 근거와 남는 일부 진실을 가짐
- 완전 영상·완전 고백 없음
- 신병 잔흔마다 사각지대 존재
- 소연의 과거가 주제와 현재 범죄에 구조적으로 연결

판정: 통과.

# 8. Gate 5 이전 필수 동기화

1. 실제 지명·이동·계절 감사
2. 인물 성명·조직명·직업명 역사·언어 검토
3. 부상·회복·침식 연속성 장부
4. 장면별 직접 세력·인물 과밀 제한
5. 최종 합증 용어를 명대·무협 질감에 맞게 조정
6. 신병 이름과 형태의 용어 감사
7. 각 부 전투 목표·지형·비용 모델
8. 설명을 행동과 갈등으로 변환하는 문체 규칙
