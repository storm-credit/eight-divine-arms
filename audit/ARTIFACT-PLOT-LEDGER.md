# 물류 제한

- 신병 이동 시 물건·기록·사용자·보관자가 한 권리를 모두 갖지 않는다.
- 장거리 운송은 최소 두 종류의 증인과 상태 기록을 남긴다.
- D3·D4·D6·D7·D8을 한 수레·한 배·한 창고에 동시에 두지 않는다.
- 최종부 화재 전에도 공개본·제한본·봉인본은 이미 다른 장소에 분리한다.
- 신병 탈취가 곧 소유권·계승권·증거 독점을 뜻하지 않는다.

# 파손·사용 비용

- D3: 소연 어깨·평형 감각 악화, 구조 사용 중심
- D4: 반복 접근 시 이름·수량 강박과 기억 혼선
- D6: 사용자에게 환자 고통·맥박 감각 중첩, 장시간 사용 금지
- D7 복원궁: 재료 연속성은 높지만 장력·공명 불안정
- D7 전승궁: 기능 적합성은 높지만 현재 물질 연속성 약함
- D8: 상충 명령·복종 압박·판정 강박 위험

# 결말 규칙

- 신병은 파괴·중앙 수집되지 않는다.
- D7의 두 물건을 합쳐 새 완전체를 만들지 않는다.
- 각 보관자는 정기 감정 기관의 하급 지부가 되지 않고 독립 권리를 유지한다.
- 비상 합증은 신병 전부를 물리적으로 소집하는 의무가 아니다.
