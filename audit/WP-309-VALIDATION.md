# WP-309 전문가 검증 — 핵심 세력 이름·대표 인물

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 유명 세력 과잉 감사

유명 장르 앵커:
- 소림사
- 무당파
- 개방
- 남궁세가
- 천마신교

작품 고유·기능 중심:
- 해남파의 해상 역할 강화
- 산동악가
- 금의위 남방 회수별조
- 하오문
- 창해상련
- 진원표국
- 만독문
- 무림맹과 사도련의 제도 설계

판정:
- 유명 세력 5개가 기준점을 제공하고 9개 기능 세력이 사건을 움직이는 균형으로 통과.

## 2. 핵심 인물 소속 감사

- 심소연 → 진원표국
- 진소무 → 하오문
- 두약란 → 만독문
- 유청하 → 무당파
- 나해성 → 해남파
- 고정문 → 금의위 남방 회수별조
- 사문경 → 전 천검각 계열, 현재 독립 합의망
- 위철산 → 사도련과 계약하는 해문 후손 보호망

한 핵심 인물이 한 세력을 완전히 대표하지 않도록 내부 반대자를 둔다.

판정: `pass`

## 3. 대표 인물 구조 감사

A등급 세력마다 최소 다음을 둔다.
- 외부 대표 또는 지도자
- 실무 대표
- 내부 반대자 또는 다른 파벌

모든 대표를 장문 인물 바이블로 만들지 않고 Gate 4에 직접 쓰이는 사람만 상세화한다.

판정: `pass`

## 4. 원사건·신병 연결 감사

- 무림맹·소림·무당은 직접 시조 계승이 아니라 후대 판정·기록·구휼·의례 경로로 연결
- 해남파는 D2 보관·사용
- 남궁세가는 D8·계승 판정 정치와 연결
- 산동악가는 D1 보관
- 금의위 별조는 D7 복원궁·국가 기록·D3 압수
- 창해상련은 D4 보관·해상 장부
- 진원표국은 D3 인계 증언과 현재 이동
- 만독문은 D6 보관·의술 기록
- 천마신교는 D7 전승궁과 외부 기록
- 하오문은 D3 수리 탁본·생존자 경로

현재 세력이 최초 여덟과 일대일 대응하지 않는다.

판정: `pass`

## 5. 역사·명칭 감사

주의 대상:
- 금의위의 실제 편제와 작품용 별조 명칭 구분
- `산동악가`의 악비 후손 주장을 사실로 확정하지 않음
- `하오문`, `천마신교`, `만독문`은 장르 관습적 명칭으로 역사 실재를 주장하지 않음
- 창해상련·진원표국의 중국어식 자호와 지역 지점명은 후속 조정

처리:
- 가상 조직에는 설정집에서 `fictional` 또는 작업명 표시
- 실제 역사 기관과 가상 파견조를 구분

판정: 조건부 통과.

## 6. 고정 숫자 감사

- A등급 14개는 제작용 내부 분류이며 본문 집합 명칭이 아님
- 구파일방·오대세가 외 새 고정 숫자 정치 블록을 만들지 않음
- 실제 반복 전면 세력은 Gate 4에서 부별 최대 5개

판정: `pass`

## 7. 대표 인물 작업안 검토

### 무림맹
- 맹주 `조문헌(趙文憲)` — 합의와 의석 안정을 중시
- 기록총관 `방여진(方如眞)` — 재심·증거 교차를 주장

### 소림사
- 장로 `혜정(慧靖)` — 구휼·민간 피해 우선
- 계율원 `혜각(慧覺)` — 옛 기록 공개에 신중

### 무당파
- 장로 `청허(淸虛)` — 유청하의 스승, 공개 절차와 산문 명예 중시
- 유청하 — 젊은 외부 실무 대표

### 개방
- 장로 `노백천(魯百川)` — 민생·실종자 정보
- 분타주 `문칠매(文七妹)` — 무림맹보다 지역 생존 우선

### 해남파
- 장문 `목연파(穆延波)` — 지역 권리·선단 생존
- 나해성 — D2 사용자, 강경 실무파

### 남궁세가
- 가주 `남궁정윤(南宮正允)` — 계승·의석 안정
- 방계 대표 `남궁소미(南宮小眉)` — 기록 재심·상업 개혁

### 산동악가
- 가주 `악승도(岳承道)` — 군문·가문 책임 균형
- 후계 경쟁자 `악지명(岳知明)` — 국가 차출과 군사 활용 적극

### 사도련
- 련주 `연무백(燕無白)` — 계약 질서와 회원 자치
- 강경파 `위철산` — 파기·탈환

### 천마신교
- 교주 `소경월(蕭鏡月)` — 외교·교리 균형
- 호법 `탁발륜(拓跋倫)` — D7 전승궁과 중원 복권 적극

### 금의위 남방 회수별조
- 별조 지휘 `권도형(權道衡)` — 즉시 압수와 기관 공적
- 실무자 `고정문` — 증거·절차 중시

### 하오문
- 총연락책 `화삼랑(花三娘)` — 정보 중립과 생업 보호
- 진소무 — 지역 신분·경로 실무자

### 창해상련
- 련주 `임만주(林萬舟)` — 신용·항로 안정
- 장부 책임자 `심옥령(沈玉玲)` — D4와 옛 채무 공개에 신중

### 진원표국
- 총표두 `진대원(陳大元)` — 표국 신용과 전국 지점
- 태주 지점 표두 `고학선(高鶴先)` — 심소연의 스승·과거 호송 책임

### 만독문
- 문주 `두백산(杜百山)` — 지역 자치·지식 통제
- 두약란 — 외부 진료·검증파

이름은 작업 기준이며 음운·한자·지역 중복을 후속 감사한다.

판정: 기능과 파벌 분리가 가능.

## 9. 악마의 변호인

### 위험 A — 대표 인물 이름이 너무 많음

조치:
- 본편 1부에서 이름을 노출하는 대표는 최대 8명
- 나머지는 직함·문서·후속 부에서 소개

### 위험 B — 익숙한 문파가 자동 선역

조치:
- 소림·무당·개방·남궁 내부에 서로 다른 이해와 과오 배치
- 작품 고유 세력도 선악 단일화 금지

### 위험 C — 모든 A등급 대표가 젊은 주인공과 직접 만남

조치:
- 실무자·문서·대리인·후원으로 간접 개입
- 수장 회담은 드물고 비용이 큼
