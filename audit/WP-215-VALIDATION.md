# WP-215 전문가 검증 — 생활·신분·외부 무림 최소 골격

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 사회·신분 검토

강호 소속은 기존 사회 신분을 지우지 않는다.

- 사대부·관료 가문
- 군호·퇴역 군인
- 상인·전장·표국
- 장인·의원·교두
- 농민·산민·뱃사람
- 객잔·유흥·장례·시장 노동
- 종교 공동체
- 노비·하층 신분·유랑민

각자는 다른 문서·보증·이동·법적 위험을 가진다.

판정: 통과.

## 2. 여성 인물 검토

가능한 경로:
- 문파·도관·사찰·보타암형 종교 조직
- 세가·상단·표국의 가업
- 의원·산파·약재·독문
- 뱃사람·상인·공연·장례·객잔 생업
- 과부·상속자·분가·양녀
- 군문·지역 공동체의 방어

제약:
- 혼인·상속·평판·여행 안전
- 남성 보증을 요구하는 지역 관행
- 숙박과 복식·위장
- 임신·출산·양육과 가족 의무

예외는 조직·재산·기술·동료·지역 관행으로 설명한다.

판정: 통과.

## 3. 가족·계승 검토

구분:
- 혈연
- 양자·양녀
- 의부·의자
- 의형제·의자매
- 사제·사손
- 혼인·첩·과부·분가
- 후원자·보증인

사제 관계는 무공 계승에는 강하지만 재산 상속과 법적 가족 관계를 자동 대체하지 않는다.

판정: 통과.

## 4. 아이·노인·장애·부상 검토

- 아이는 제자·시동·상단 가족·난민·인질·후계자로 등장 가능
- 조기 수련은 신체 성장과 교육·착취 문제를 가짐
- 노인은 약한 인물이 아니라 지식·관계·권리를 가질 수 있으나 체력과 돌봄 문제가 존재
- 장애·후유증은 무공·병기·생업의 적응을 만들 수 있음
- 부상자는 장기 회복·비용·가족과 조직 의존을 가짐

판정: 통과.

## 5. 생활 검토

후속 인물과 장면에 필요한 최소 항목:
- 옷과 신발의 계절·신분·직업 차이
- 식사·물·연료·말 먹이
- 객잔·사찰·표국 지점·선박·야영 숙박
- 목욕·세탁·상처 위생
- 짐·병기·문서·현금·어음 보관
- 화장실·월경·질병·벌레·악천후
- 통역·안내인·현지 중개

세부 도감은 장면 필요 시 작성한다.

판정: 통과.

## 6. 구휼·착취 검토

구휼 주체:
- 사찰·도관
- 종족·가문
- 향신·관부
- 상단·방회
- 개방·의원
- 지역 공동체

착취 주체:
- 인신매매
- 고리대와 채무 종속
- 유곽·도박·보호비
- 난민 노동과 아동 착취
- 사병·산채·부패 관원

피해자를 배경 장식이나 영웅 각성 도구로만 쓰지 않는다.

판정: 통과.

## 7. 외부 무림 검토

원칙:
- 일본계 해상 집단·몽골·여진·티베트·서역·남방 집단을 단일 문화로 묶지 않음
- 무사·상인·선원·통역·승려·용병·포로·혼인 이주민을 구분
- 중원식 정사마 분류를 자동 적용하지 않음
- 외부 인물에게도 가족·경제·정치·종교·언어가 있음
- 기괴한 무공과 보물의 공급처로만 사용하지 않음

판정: 통과.

## 8. 통역·혼인·교역 검토

- 통역은 단어 변환뿐 아니라 관습·계약·명예를 번역
- 혼인과 인질은 외교·보증·생존 장치
- 혼혈·다언어 가정과 항구 공동체 존재 가능
- 조공·밀무·합법 교역·해적 행위를 구분

판정: 통과.

## 9. 악마의 변호인

### 위험 A — 생활 설정 과잉

조치:
- 장면의 선택·갈등·감정에 영향을 주는 항목만 상세화
- 음식·복식 도감 자체를 목표로 하지 않음

### 위험 B — 제약을 여성과 하층민에게만 적용

조치:
- 남성 명문 자제도 혼인·가문·군역·체면·상속 제약을 가짐
- 모든 인물의 이동 비용과 보증을 기록

### 위험 C — 외부 문화가 장식

조치:
- 핵심 외부 인물은 고향·언어·생계·관계·정치 목표를 가짐
- 문화적 특징보다 인물과 조직의 구체적 선택을 우선

## 10. 정합성 판정

치명적 충돌 없음.

개별 생활·복식·역사 항목은 Gate 3~5에서 장면 단위로 검증한다.
