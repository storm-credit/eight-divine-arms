# WP-211 전문가 검증 — 세력 권력·경제 프로필

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 역할 중복 감사

### 개방과 하오문

- 개방: 공개된 민생·거리·구휼·정파 연결 정보
- 하오문: 시장·부두·유흥·장례·밀항·위조 신분과 비공식 거래

판정: 분리 유지.

### 상단과 표국

- 상단: 자본·창고·선박·채무·거래
- 표국: 사람과 화물의 인계·호송·배상·노선 협상

판정: 주인공·초기 사건과 연결될 경우 각각 A등급 유지. 그렇지 않으면 한쪽을 B+로 축소 가능.

### 불문·도문 대종문

- 불문: 구휼·계율·속가·도덕적 중재
- 도문: 내공·심신·검리·의례 해석

판정: 기능 분리 가능.

### 검가와 군문 무가

- 검가: 세가 정치·혼인·계승·정통성
- 군문 무가: 집단전·창궁기마·군진·병참

판정: 분리 유지.

### 만독문형 세력과 의가·당문

- 서남 독·약 세력: 자연독·독충·생태·현지 약재·통행권
- 당문: 정제독·암기·기관
- 의가·약왕장: 치료·재활·복합 처방

판정: A등급은 서남 세력 하나, 나머지는 B 전문 풀.

## 2. 경제·생존 감사

모든 A등급 세력은 최소 다음을 가진다.

- 독립 수입원 두 개
- 다른 세력에 대한 필수 의존 한 개 이상
- 비용이 큰 공적 또는 의무
- 내부에서 자원 배분을 다투는 파벌

자급자족·전지전능 세력은 금지한다.

판정: 통과.

## 3. 권력축 감사

세력 프로필은 다음 일곱 축을 사용한다.

- 정통·명분
- 지역 지배·보증
- 경제·물류
- 정보·중개
- 국가 강제력
- 전문 지식
- 비공식 폭력

축은 강·중·약과 의존 관계로 기록하며 단일 총점은 만들지 않는다.

판정: 통과.

## 4. 지역 다양성 감사

전국 A등급만으로 각 권역을 덮지 않는다.

- 동남 해안: 해상 문파·선단·관군·밀무망
- 강남: 상단·전장·세가·관부
- 중원: 대종문·무림맹·교통 요지
- 서남: 독·약·광산·산길·토착 보증
- 북방: 군진·마시장·무가·변경 공동체
- 서북: 천마신교·대상로·요새·외부 무림

각 부에서 B등급 지역 세력 2~4개를 전면화한다.

판정: 통과.

## 5. 악마의 변호인

### 위험 A — 14개도 많음

조치:
- 한 부에 직접 개입하는 A등급 세력 최대 5개
- 한 신병을 직접 추적하는 주요 세력 최대 5개
- 나머지는 배경 압력·간접 지원·후속 반응으로 처리

### 위험 B — 전통 문파가 약해 보임

조치:
- 불문·도문·개방·검가를 기준점으로 유지
- 차별점은 해상 정파·군문 무가·상단·표국·하오문이 사건을 움직이는 데 둠

### 위험 C — 모든 세력이 팔대신병을 원함

조치:
- 세력별 목표를 병기·기록·증인·권리·파기·민간 피해 중 하나로 구분
- 직접 소유를 원하는 세력은 일부로 제한

### 위험 D — 경제가 설정 설명으로만 남음

조치:
- 채무·보관비·배상·인력·약재·노선이 실제 선택과 갈등을 바꾸게 함

## 6. A등급 승격 기준

다음 중 최소 세 항목을 만족한다.

- 여러 부에 반복 등장
- 핵심 인물의 관계와 선택 변화
- 신병·원사건의 핵심 증거·권리와 연결
- 전국 또는 핵심 권역 판세 변화
- 다른 세력 둘 이상을 연결·차단
- 결말 뒤 권력 변화가 명확

## 7. 정합성 판정

치명적 충돌 없음.

최종 이름은 Gate 3 인물·플롯과 교차 후 확정한다.
