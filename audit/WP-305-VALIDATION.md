# WP-305 전문가 검증 — 현재 소유·보관·계승 지도

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 분산 구조 검토

권장 분포:
- 동남 해안: D2 절류도, D3 철조구
- 강남·남직례: D4 묵형척, D7 복원궁 후보
- 중원: D5 연생장, D8 봉서검
- 북방·산동: D1 개문창
- 서남: D6 청맥인
- 서북: D7 전승궁 후보

장점:
- 1555년 절강 시작과 전국 이동 연결
- 한 권역에 과도한 신병 집중 방지
- 해상·상업·종문·군문·외부 무림의 서로 다른 보관 관행

주의:
- 모든 신병을 실제로 방문·회수할 필요는 없음
- Gate 4에서 중심 신병 4~5개만 직접 플롯화

판정: `pass`

## 2. 권리 분리 검토

각 신병에서 다음 중 최소 세 주체가 다를 수 있다.
- 물리 보관자
- 법적·재산 소유 주장자
- 사용 적합자
- 계승 주장자
- 감정 기관
- 기록 보유자
- 공개·파기 이해관계자

어느 세력도 신병·혈통·기록·사용·공개 권리를 모두 독점하지 않는다.

판정: `pass`

- D6 청맥인

## 4. 세력 집중 감사

- 무림맹은 D8 감정·보증에 참여하지만 물리 소유자가 아님.
- 황실 회수조는 D7 복원궁을 관리하거나 압수할 수 있으나 다른 신병은 정보 수준이 다름.
- 대종문은 신병을 대량 보유하지 않음.
- 상단은 D4를 보관하지만 기록·계승을 독점하지 않음.
- 천마신교는 D7 전승궁 후보만 직접 연결.

판정: `pass`

## 5. 증거 교차 분산 검토

- O1 명령: D8 보관 기관과 황실 기관에 분산
- O2 선박 기록: 상단 장부와 해상 구전으로 분산
- O3 수리 기록: 지방 창고와 하오문 탁본으로 분산
- O4 장부: D4 보관처와 D5 구휼 명부에 교차
- O5 생존자 명부: D5 사찰과 해안 가족에 분산
- O6 검시 기록: D6 계보와 관부 검시 사본으로 분산
- O7 신호 기록: 두 D7 후보와 항구 통역 가문에 분산
- O8 인장·전달 기록: D8 기관과 무림맹 중앙에 분산

한 병기와 같은 장소에 그 병기의 모든 설명이 있지 않다.

판정: `pass`

## 6. 이동·물류 검토

- D1·D4·D5·D6·D8은 장기 보관 상태에서 사건 때문에 이동할 수 있음.
- D2는 실사용·해상 이동 상태.
- D3는 다른 산업 도구로 오인되어 현지에 정착.
- D7 후보 둘은 정치적 이유로 이동 제한.

모든 신병을 동시에 호송하지 않는다.

판정: `pass`

## 7. 첫 사건 연결 검토

- D2 보관자의 기록 또는 잔흔이 D3의 정체를 부분 확인
- 황실 회수조·해상 정파·상단·표국·하오문 중 최대 다섯 세력이 서로 다른 목표로 개입

## 8. 악마의 변호인

### 위험 A — 지도 순회 수집물

조치:
- 직접 발견·회수하는 중심 신병은 4~5개
- 나머지는 소유자·기억·합증·정치 반응으로 간접 참여

### 위험 B — 각 권역에 정확히 한 개씩 배치한 게임 지도

조치:
- 동남·중원에 두 자리, D7은 두 권역에 걸친 분쟁
- 보관 위치와 기원·계승 지역이 다를 수 있음
- 지역 B세력과 증거는 신병 수와 무관하게 배치

### 위험 C — 비밀인데 모두 팔대신병으로 알아봄

조치:
- 공개 명칭·감정 수준·전설이 다름
- D3·D6은 처음부터 신병으로 인식되지 않음
- 일부 세력은 가짜 신병을 더 중요하게 추적

### 위험 D — 현재 세력과 최초 당사자 일대일 계승

조치:
- 물건·기록·혈통·기술·책임을 서로 다른 세력에 분산
