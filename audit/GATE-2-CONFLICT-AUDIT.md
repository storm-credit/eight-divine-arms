# 3. 치명적 충돌 검사

## C-01. 저강도 기환과 무공의 상한

검사:
- 제한적 내기가 범용 마법으로 확대되는가?
- 기환이 무공과 군대·화기·독을 무효화하는가?

결과:
- 치명적 충돌 없음.

근거:
- 내기는 접촉·병기·발성·지면 반력의 순간 전달로 제한됨.
- 경공은 비행이 아니며 호신강기는 절대 방어가 아님.
- 모든 기환은 매개체·촉발·대가·실패·소실을 요구함.
- 개인 고수는 짧은 국면을 바꿀 수 있으나 준비된 군대와 장기 포위를 지속 무효화하지 못함.

판정: `pass`

## C-02. 무공과 부상·치료

검사:
- 내공과 영약이 전투 결과를 즉시 제거하는가?
- 신병 사용 대가가 의원 한 명에게 해소되는가?

결과:
- 치명적 충돌 없음.

근거:
- 구조 손상·기능 장애·기혈 장애·독·감염·정신 침식을 분리함.
- 운기·침구는 구조 손상을 즉시 복구하지 못함.
- 신병 기억 침식은 완전 삭제보다 관리와 구분이 중심임.

판정: `pass`

## C-03. 일반 병기·보패·팔대신병의 격차

검사:
- 보패와 법구가 팔대신병의 소형판이 되는가?
- 신병이 모든 전투에서 최강이어야 하는가?

결과:
- 치명적 충돌 없음.

근거:
- 희귀 물건은 단일 위력 등급이 아닌 다축 분류를 사용함.
- 보패는 좁은 기능과 조건을 가진 휴대 법구임.
- 팔대신병의 특별함은 장기간 구조화된 흔적·공동 원사건·권리·정치적 영향에 있음.

판정: `pass`

## C-04. 팔대신병 정의와 숫자 여덟

검사:
- 우주론·사방신·팔괘가 기원에 뒤늦게 개입하는가?
- 하나의 완전병기 분할·부활 구조로 되돌아가는가?

결과:
- 치명적 충돌 없음.

근거:
- 여덟은 원사건 뒤 남은 실제 당사자 수라는 역사적 결과임.
- 사방신·팔괘는 후대 상징과 해석에만 사용함.
- 완전병기·완전 부활·자동 절대무공을 명시적으로 반려함.

판정: `pass`

## C-05. 신병 기억과 미스터리

검사:
- 기억이 완전한 영상·인격·정답 장치가 되는가?
- 기억이 지나치게 주관적이라 증거 가치가 사라지는가?

근거:
- 신체·동작·감정·감각·의도의 다층 잔흔으로 제한함.
- 각 병기의 강한 채널은 최대 두 개이며 사각지대를 사전 장부화함.
- 반복 가능한 물리 감각과 여러 증거의 교차 검증을 요구함.

판정: `pass`

## C-06. 신병 권리와 강호·국가 제도

검사:
- 물건을 빼앗은 사람이 모든 권리를 얻는가?
- 무림맹 또는 관부가 모든 분쟁을 단독 해결하는가?

결과:
- 치명적 충돌 없음.

근거:
- 점유·소유·보관·사용·계승·열람·공개·파기 권리를 분리함.
- 관부·무림맹·문파·상단·표국·유족의 판단 포럼을 구분함.
- 강호는 국가 밖의 독립 법체계가 아니라 국가 사회 안의 보증망임.

판정: `pass`

## C-07. 신병 집합과 결말 조기 고정

검사:
- 여덟을 모아야만 이야기를 끝낼 수 있는가?
- 집합이 결말을 자동 선택하는가?

결과:
- 치명적 충돌 없음.

근거:
- 합증은 진위·기억·권리·협약 위반을 넓게 검증하는 절차임.
- 합증은 군대와 세력의 복종 또는 자동 처리를 만들지 않음.
- 여덟 합증은 가장 완전한 검증이지만 유일한 물리 열쇠가 아님.
- 실제 공개·봉인·분산·파기 선택은 Gate 4로 남김.

판정: `pass`

## C-08. 세력 수와 보물 쟁탈 반복

검사:
- 너무 많은 세력이 매 신병마다 집결하는가?
- 모든 세력이 물리적 소유만 추구하는가?

결과:
- 치명적 충돌 없음.

근거:
- 전국 반복 A등급은 14개 기능 슬롯으로 제한함.
- 한 부의 직접 A등급 세력은 최대 5개임.
- 한 신병 사건은 물리 추적 3개 + 현장 기능 2개로 제한함.
- 세력 목표를 확보·보관·검증·기록·보호·공개·은폐·무력화·중개로 분리함.

판정: `pass`

- 새로운 고정 숫자 집단은 별도 심사를 요구함.

판정: `pass`

# 4. 중간 위험과 보정 조건

## R-01. Gate 3에서 원사건이 협약 구조에 끌려갈 위험

심각도: `medium`

문제:
- 원사건을 아직 설계하지 않았는데 여덟 생존자와 책임 분산 구조가 먼저 확정됨.
- 원사건이 이 구조를 설명하기 위한 인공적 사건이 될 수 있음.

보정:
- Gate 3에서 최소 네 개 원사건 후보를 독립 생성함.
- 각 후보는 여덟 생존자가 자연스럽게 남는 인과를 제시함.
- 정확히 여덟을 만들기 위한 편의적 사망·선별 금지.

## R-02. 권리 체계가 법률극처럼 보일 위험

심각도: `medium`

문제:
- 권리와 증거 구분은 견고하지만 본문에서 행정 절차로 설명될 수 있음.

보정:
- 한 부에서 전면화하는 권리 갈등은 하나만 둠.
- 권리는 인물의 채무·유품·가족·보증·배신으로 표현함.
- Gate 5 장면 규칙에서 설명량 제한.

## R-03. 공명이라는 용어의 만능화

심각도: `medium`

문제:
- 무공·신병·부적·장소·강시를 모두 공명으로 설명하면서 원인이 단조로워질 수 있음.

보정:
- 공명은 상위 감사 언어로만 사용함.
- 본편과 개별 설정에서는 구체적 매개체·감각·재료·실패를 우선 묘사함.
- 개별 현상마다 WP-204 아홉 필수 항목을 작성함.

## R-04. 세력 14개와 팔대신병 8개가 숫자 구조로 보일 위험

심각도: `medium`

문제:
- 제작용 슬롯 수가 독자에게 노출되면 게임판처럼 보일 수 있음.

보정:
- A등급 14개는 내부 제작 분류이며 본문에서 집합 명칭으로 말하지 않음.
- 한 부에 3개 중심 + 2개 보조만 전면화함.
- 팔대신병 외 새로운 고정 숫자 조직을 최소화함.

## R-05. 최초 병기와 현재 계승품의 복잡도

심각도: `medium`

문제:
- 원본·수리 원본·계승품·이중 주장·공석이 동시에 등장하면 독자가 혼란스러움.

보정:
- 본편 시점 공석·이중 주장은 합쳐 최대 두 자리.
- 초반부에는 안정된 진품 후보를 먼저 제시함.
- 각 부 결산에서 최소 하나의 진위·기능·권리 사실을 확정함.

## R-06. 역사·용어 세부 검증 미완

심각도: `medium`

미확정:
- 정확한 원사건 연도
- 1555년의 시작 월·계절
- 일부 병장기·법제·문서·해상 용어
- 포르투갈계·일본계·지역 집단의 구체적 비중
- 명대의 실제 종교·상업·군사 용어

보정:
- Gate 3의 원사건·현재 사건 설계마다 역사 검증 패킷을 병행함.
- 검증 전 고유 용어는 `working-term`으로 표시함.

## R-07. 주인공이 아직 시스템 밖에 있음

심각도: `expected / Gate 3`

문제:
- 신병 접근 적합성과 권리 구조는 있지만 주인공이 중심이 되는 이유가 없음.

보정:
- 혈통 선택을 금지한 상태에서 경험·직업·상처·기술·보증망 후보를 최소 네 안 비교함.
- 주인공만 모든 신병 채널에 가장 잘 접근하는 구조 금지.

# 7. PM 최종 판정

## 판정

`Gate 2 — resolved-provisional`

## 근거

- 15개 작업 패킷 완료
- 치명적 상호 모순 없음
- 중간 위험 7개에 보정 조건 존재
- Gate 3가 요구하는 선행 구조 충족
- 파생 설정이 상위 공통법 아래 배치됨
- 후속 상세화와 의도적 제외가 명확함

## 제한

Gate 2의 `provisional-canon`은 사용자 최종 정본 승인이 아니라 후속 설계용 작업 기준이다.

Gate 3에서 원사건·인물·개별 신병을 적용하다 구조적 모순이 발견되면 관련 WP만 재개방한다.
