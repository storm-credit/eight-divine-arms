# 《팔대신병록》 미해결 질문 장부

상태: `writing-readiness-review`
관리: 정합성 감사 에이전트 / PM 오케스트레이터

Gate 0~4의 핵심 질문은 작업 기준으로 해결됐다.

남은 항목은:

- `blocking`: 시놉시스·장별 개요를 시작할 수 없게 하는 문제
- `outline-check`: 장별 개요 전에 검증할 문제
- `later`: 해당 인물·지역·장면을 상세화할 때 결정
- `resolved-provisional`: 작업 기준 해결

으로 분류한다.

# 7. 장별 개요 전 검증

## O-501 실제 지리 노선

- 상태: outline-check
- 태주부 연해 허구 선창 위치
- 금릉까지 수로
- D8 중립기관 위치
- 장강 상류·서남 약재로
- 서남에서 해문도 귀환 항로

## O-502 계절·수운

- 상태: outline-check
- 음력 월별 우기·풍향·태풍
- 겨울 중원 이동
- 장강 수위·하행 속도

## O-503 금의위 차견 직급·문서

- 상태: outline-check
- 고정문·권도형의 품계·지휘 인원
- 지방관·위소·순안과의 권한 충돌
- `남방 회수차견`의 본문 표현

## O-504 상업·호송 용어

- 상태: outline-check
- 진원행·검화수·인계호송인·행관·호송두 최종 한자
- 창해상련의 본문명
- 교할·험수·압운 등 문서 상황별 용어

## O-505 결말 문서 용어

- 상태: outline-check
- 해문합증
- 공개본·제한본·분장봉인본을 명대·무협 질감에 맞게 표현

이 다섯 항목은 각 부의 장소와 문서 장면을 상세화할 때 검증하며, 전체 시놉시스 작성을 막지 않는다.

# 8. 후속 상세화

- 구파일방·오대세가 전체 명단
- 실제 후기지수·십대고수 목록
- 무림대회 회차·참가자
- 개별 영수·영초·부적·보패·강시 사건
- 비중 낮은 세력 대표와 내부 인물
- 가족·생일·별호·복식
- 문파별 초식명
- 로맨스 여부

해당 장면이나 인물의 서사 기능이 확정될 때만 상세화한다.
