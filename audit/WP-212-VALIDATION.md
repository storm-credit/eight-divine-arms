# WP-212 전문가 검증 — 세력–신병 관계 행렬

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 목표 중복 감사

물리적 소유를 상시 최우선으로 둘 수 있는 슬롯:
- 세가권 대표 검가
- 군문 무가
- 황실 비밀 회수조
- 사도련 또는 내부 강경파

그 외 세력은 보관·검증·기록·증인·공개·무력화·중개를 우선할 수 있다.

판정: 직접 소유 추구 최대 4개 원칙 통과.

## 2. 세력별 행동 가능성 검토

- 무림맹: 판정과 중재를 위해 진위·권리 자료 필요
- 불문 대종문: 피해와 봉인·책임을 우선
- 도문 대종문: 공명·침식의 검증과 안정
- 개방: 실종자·유족·민간 피해·소문 정정
- 해상 정파: 지역 소유·선박·해상 피해와 압수 반대
- 검가: 계승·명예·사용권
- 군문 무가: 군사 위험·방어·국가 동원
- 사도련: 정파 독점 해체와 회원 권리
- 천마신교: 중원 기록에 지워진 증언과 역사적 복권
- 황실 회수조: 반역·군기·비밀 문서와 압수
- 하오문: 증인·운송자·비공식 경로와 정보 가격
- 상단: 계약·채무·창고·압수 손실
- 표국: 인계·호송·배상·신용 회복
- 서남 독약 세력: 사용 대가·독성·약재·지역 자원 보호

판정: 기능 분리 가능.

## 3. 공개 목표와 실제 목표 검토

모든 A등급 세력은 다음을 분리한다.

- 공개 명분
- 실제 조직 이익
- 내부 파벌의 다른 목표
- 넘지 않겠다고 선언한 금지선
- 실제 위기에서 금지선을 깰 조건

판정: 인물·플롯 단계의 필수 프로필로 이관.

## 4. 동맹·충돌 검토

자연 동맹 후보:
- 개방–표국: 실종자·호송·증인 보호
- 해상 정파–상단: 항로·화물·밀무 피해
- 도문–서남 독약 세력: 침식과 독·약 감별
- 무림맹–불문 대종문: 중재와 민간 피해 수습
- 천마신교–하오문: 중원 공식 기록 밖의 경로

구조적 충돌 후보:
- 무림맹–황실 회수조: 계승 판정과 국가 압수
- 해상 정파–황실 회수조: 지역 피해와 중앙 관할
- 검가–군문 무가: 정통 검계와 군사 실용
- 개방–하오문: 정보 공개와 거래 비밀
- 상단–표국: 의뢰인의 거짓과 배상 책임
- 사도련–무림맹: 법도와 권리 독점
- 천마신교–중원 정파: 역사 명칭과 책임

판정: 통과.

## 5. 신병별 개입 상한 검토

한 신병의 한 사건에서:
- 직접 물리 추적 세력 최대 3개
- 현장 중개·보호·수사 세력 최대 2개
- 간접 정치 반응은 후속 장면에서 처리

장편 전체에서 같은 다섯 세력이 모든 신병을 반복 추적하지 않는다.

판정: 통과.

## 6. 악마의 변호인

### 위험 A — 행렬이 너무 기능적

조치:
- 세력의 목표를 대표 인물의 개인 손실·채무·명예와 연결
- Gate 3에서 각 세력에 내부 반대자 배치

### 위험 B — 전투가 줄어 보물 쟁탈 맛 약화

조치:
- 증인·문서·보관·호송도 물리 충돌의 이유가 됨
- 소유 추구 세력은 적게 두되 강하게 압박

### 위험 C — 목표 변화가 편의적

조치:
- 정보 획득·지도자 교체·피해 발생·채무 변화 중 하나를 원인으로 기록

## 7. 정합성 판정

치명적 충돌 없음.

개별 신병과 최종 세력명이 확정되면 관계 행렬을 다시 채운다.

## 8. PM 판정

목표 유형 행렬 + 단계 변화 구조를 `provisional-canon`으로 승격 가능.
