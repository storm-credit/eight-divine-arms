# WP-213 전문가 검증 — 기환 하위 현상 경계

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 기환 공통법 검토

모든 실제 하위 현상은 다음을 가져야 한다.

- 발생 원인
- 매개체
- 촉발 조건
- 관측 조건
- 효과 범위
- 지속 시간
- 대가
- 실패
- 소실 조건

판정: 통과.

## 2. 부적·법구·보패 검토

- 부적: 기록·경고·집중·재료 배치·봉인의 준비형 도구
- 법구: 의례·감정·치료·경계·기록에 쓰는 기능 도구
- 보패: 희귀하고 휴대 가능한 좁은 기능의 법구·호신구
- 즉석 만능 주문과 대량 생산 금지

판정: 통과.

## 3. 진법·풍수·점복 검토

- 진법: 인원·지형·도구·신호·훈련의 배치 기술
- 풍수: 물·바람·지형·거주·매장·이동과 장소 흔적을 해석하는 지식
- 점복: 징후·정보·심리·확률을 구조화하는 판단 도구
- 공간 왜곡·확정 미래 예언·무준비 마법 장벽 금지

판정: 통과.

## 4. 영물·영수·영초 검토

- 영물: 오래된 물건·생물·장소 중 특이 흔적을 가진 사례를 부르는 느슨한 통칭
- 영수: 희귀 환경에 적응해 비정상적 감각·습성·수명을 보이는 개체
- 영초: 희귀 생육 조건과 약성을 가진 식물
- 종족형 마수·사냥 산업·내단 획득 성장 체계 금지

판정: 통과.

## 5. 신수·사방신 검토

- 용·봉황·기린·사방신은 왕권·방위·계절·문양·진법·가문 신화의 상징으로 우선 사용
- 실재 생물 종족으로 확정하지 않음
- 극히 드문 목격담과 전승은 사실·오인·상징을 열어 둠
- 직접 강림과 계약 소환 금지

판정: 통과.

## 6. 강령·강시·시귀 검토

- 강령: 완전한 영혼 소환이 아니라 잔념·감각·행동 흔적 접근
- 강시: 시신이 움직이는 현상을 부르는 포괄어이며 독·경직·침술·기관·최면·잔향·금술 가능성
- 시귀: 시신·유품·장소에 붙은 잔존 의식처럼 체험되는 현상 또는 강호 용어
- 안정적 인격 대화·독립 귀신 사회·대규모 강시 군단 금지

판정: 통과.

## 7. 종교·사회 검토

- 의례는 실제 효과가 없어도 장례·책임·공포 관리·공동체 회복 기능을 가짐
- 종교기관은 토지·구휼·교육·장례의 사회적 역할을 가짐
- 실제 기환 판정 권위를 독점하지 않음
- 사교 판정은 정치적 탄압과 실제 금단 행위를 구분

판정: 통과.

## 8. 경제·희귀도 검토

- 영초·영수·보패는 채집·보존·위조·밀렵·거래 문제를 만듦
- 흔한 상품이 되면 기환 상한과 팔대신병 희소성이 붕괴
- 개별 도감은 플롯에서 실제 사용되는 항목만 작성

판정: 통과.

## 9. 악마의 변호인

### 위험 A — 이름만 제한하고 실제로는 마법 남발

조치:
- 개별 효과 장부와 등장 횟수 관리
- 한 부의 핵심 기환 현상은 팔대신병 포함 최대 2종

### 위험 B — 모든 괴이가 혼합 원인이라 결론 회피

조치:
- 사건 결산에서 최소 하나의 원인과 물리 결과를 확정

### 위험 C — 전문 종문이 만능 해결사

조치:
- 각 종문은 자기 해석·지역·재료·접근 조건에 제한
- 두 분야 이상 증거 교차

## 10. 정합성 판정

치명적 충돌 없음.

개별 항목은 플롯 사용이 확정될 때만 상세화한다.
