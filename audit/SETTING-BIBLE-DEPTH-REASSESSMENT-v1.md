# 《팔대신병록》 설정집 상세도 재평가 v1

상태: `retained-audit / scope-overreach-corrected`
총괄: PM 오케스트레이터
후속 교정: `operations/PROJECT-DESIGN-SCOPE-LOCK-v1.md`

## 1. 유효한 지적

기존 `SETTING-BIBLE-v2.0-MASTER.md`는 상위 규칙·금지선·운영 절차를 통합한 핵심 정본이다.

다음 영역은 백과사전식 설정집 관점에서는 덜 채워져 있다.

- 세력과 문파의 개별 프로필
- 지역별 도시·생업·의복·식사·신앙 사례
- 문파·병장기·전승별 무공 사례
- 부적·보패·영초·영수·강시·진법 도감
- 가격·거리·계약·호칭의 구체 샘플

이 관찰 자체는 유효하다.

## 2. 잘못된 후속 판정

그러나 위 빈칸을 이유로 `S8~S12 백과사전형 상세화`를 작품 설계의 필수 본선으로 둔 결정은 범위 과잉이었다.

프로젝트 목표는 상세 설정 백과사전이 아니라 장편소설의 완성형 설계 패키지다.

따라서 다음 판정을 철회한다.

- 상세 설정집 100%가 플롯·Outline의 선행 조건이라는 판정
- 구파일방·오대세가 전원 완전 프로필이 필수라는 판정
- B급 세력 최소 12개 상세 카드가 필수라는 판정
- 도시·생활·기환 도감을 전면 작성해야 한다는 판정

## 3. 현재 올바른 완료 기준

설정은 다음을 설명하면 설계상 충분하다.

1. 사건이 왜 가능한가
2. 누가 이익·손해를 보는가
3. 인물이 어떤 선택을 할 수 있는가
4. 힘·돈·거리·정보·권리의 한계가 무엇인가
5. 실패의 비용이 무엇인가
6. 플롯과 다른 설정에 모순이 없는가

## 4. 실제 상세화 규칙

개별 세력·장소·무공·기환·생활 요소는 다음 중 하나일 때만 상세화한다.

- Outline에서 반복 사용
- 핵심 인물의 선택을 변화
- 단서·반전·전투 결과에 영향
- 이동·치료·계약·권리 판정을 변화
- 정본 모순을 해결

그 밖의 아이디어는 백로그로 유지한다.

## 5. 기존 깊이 판정의 보존 가치

### 깊은 편

- 팔대신병 개별 실물·수리·잔흔·권리
- 병장기 계통과 전술 상성
- 무공 상한·경지 언어
- 기록·증거·관할·권리
- 기환 공통 상한과 판별 원칙

### 필요 시 국소 보강할 편

- 권역의 실제 사용 장소
- 직접 등장 세력의 내부 파벌·재정·무공
- 사건에 쓰이는 의술·기환 사례
- 구체 거리·가격·문서·호칭

### 후속 창작 슬롯

- 작품에 등장하지 않는 세력·장소·초식·영물·대회 명단

## 6. S8~S12 처리

기존 제안은 백로그 묶음으로 재분류한다.

- S8 세력·문파·세가 도감 → Outline 사용 세력 상세화 체크리스트
- S9 무공·병장기 도감 → 등장 인물·세력 무공 카드
- S10 기환·생태 도감 → 실제 사건용 사례 카드
- S11 지역·생활 도감 → 사용 장소·생활 압력 카드
- S12 색인 → 최종 설계 패키지 색인·감사
