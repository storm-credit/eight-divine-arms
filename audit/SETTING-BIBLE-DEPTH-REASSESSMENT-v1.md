# 《팔대신병록》 설정집 상세도 재평가 v1

상태: `review-complete / depth-gap-confirmed`
총괄: PM 오케스트레이터
발동 사유: 사용자 지적 — 구조 완성과 상세 설정집 완성을 혼동한 PM 보고 교정

## 1. 결론

기존 `SETTING-BIBLE-v2.0-MASTER.md`는 상위 규칙·금지선·운영 절차를 통합한 **핵심 정본**으로는 유효하다.

그러나 다음 의미의 설정집은 아직 완성되지 않았다.

- 세력과 문파의 개별 프로필이 충분히 채워진 설정집
- 지역마다 도시·생업·의복·식사·신앙·교통·관습이 체감되는 설정집
- 문파·병장기·전승별 무공 사례가 축적된 설정집
- 부적·보패·영초·영수·강시·진법의 대표 사례 도감
- 가격·거리·계약·법·호칭을 실제 사용 가능한 표와 사례로 정리한 설정집

따라서 이전의 `설정집 100% 완료` 보고는 **구조 완전성만을 전체 완성도로 잘못 표현한 과대평가**다.

## 2. 완성도의 세 층

### A. 구조·헌법·상한

- 상태: 100%
- 포함: 작품 정체성, 세계 작동 원리, 금지선, 권리·비용·실패, 분야 간 경계
- 유효 문서: 설정집 v2.0 MASTER와 S1~S7 상위 모듈

### B. 운영 절차·대표 구조

- 상태: 약 75~85%
- 포함: 국가·강호·상업 관할, 표국·상단·무림맹 운영, 수련·병장기·의술·기환 판별, 팔대신병 개별 카드
- 부족: 지역별 실제 사례, 조직별 변형, 반복 사용 가능한 표와 샘플

### C. 백과사전형 세부·세계의 살

- 상태: 약 40~55%
- 부족:
  - 구파일방·오대세가 최종 명단과 개별 완전 프로필
  - B/C 세력 이름·내부 파벌·대표 인물·수입·거점
  - 문파별 대표 무공·병장기·훈련·상성
  - 실제 고수록·후기지수록·무림대회 사례
  - 영초·영수·부적·보패·진법·강시 대표 도감
  - 도시·항구·객잔·시장·사찰·도관·선창·관도 등 장소 카드
  - 음식·의복·주거·혼인·장례·계절 행사·생업 일과
  - 가격·임금·거리·여비·의료·배상 구체 예시
  - 지역 방언·호칭·문서·계약·비급 샘플
  - 팔대신병 지역 이형명·원기록명·수리 세대 일부

## 3. 수정된 PM 판정

- 설정 핵심 정본: 완료
- 설정집 상세화: 미완
- 사용자가 기대한 상세 설정집 전체: **약 60~70%**
- 인물·플롯 Gate: 상세화가 끝날 때까지 다시 보류
- 원고: 잠금 유지

## 4. 기존 문서의 실제 밀도

### 충분히 깊은 편

- 팔대신병 개별 실물·수리·잔흔·권리
- 병장기 계통과 전술 상성
- 무공 상한·경지 언어
- 기록·증거·관할·권리
- 기환 공통 상한과 판별 원칙

### 중간 깊이

- 권역 설정
- 국가·법·군사
- 경제·교통·생활
- 조직 운영
- 의술·독·종교·범죄

이 문서들은 정의와 운영 원리는 있으나, 실제 고유명·장소·사건 사례·지역 변형이 부족하다.

### 얕거나 의도적으로 비어 있는 편

- 구파일방·오대세가 전체 구성
- 문파별 무공과 전승
- 고수·후기지수·대회 명단
- 영초·영수·보패·부적·진법 도감
- 구체 생활문화와 장소 도감
- 실제 가격·거리·문서 샘플

## 5. 상세화 완료 기준

각 핵심 분야가 다음 다섯 층을 가져야 한다.

1. 원칙
2. 실제 운영 절차
3. 대표 고유명 사례
4. 지역·조직별 변형
5. 다른 분야와 충돌하는 사례와 판정

기존 설정집은 1~2층이 강하고, 일부만 3층까지 있다. 상세 설정집 완료를 위해 3~5층을 보강한다.

## 6. S8~S12 상세화 계획

### S8 세력·문파·세가 상세화

- 구파일방 최종 명단과 의석 역사
- 오대세가 최종 명단과 교체 기준
- A급 세력 완전 프로필
- B급 세력 최소 12개 상세 카드
- 각 세력의 재정·거점·내부 파벌·계승·무공·외교·약점

### S9 무공·병장기·전승 도감

- 핵심 문파·세가·표국·군문·해상 세력별 대표 무공
- 병장기별 수련 단계와 실패·부상
- 비주류 무기 실제 전승
- 내공·경공·점혈·진법·협공 대표 사례
- 기술명은 장식이 아니라 전술·철학·대가와 연결

### S10 기환·의술·생태 대표 도감

- 부적·법구·보패·진법 대표 사례
- 영초·영수·희귀 생태
- 강시·시귀·강령 오인 사례
- 독·약·검시·재활 대표 사례
- 실제·오인·사기 비교표

### S11 지역·생활·경제·장소 도감

- 여섯 권역별 핵심 도시·항구·수로·산문·시장·객잔·사찰·선창
- 계층별 의식주·하루 일과·혼인·장례·교육
- 가격·임금·여비·치료·배상 예시표
- 이동 거리·계절·재난·검문 사례
- 지역별 호칭·방언·문서 샘플

### S12 설정집 편집·색인·최종 상세 감사

- 용어집·인명·지명·세력·무공·물품 색인
- 중복·공백·과밀 감사
- 스포일러 없는 독자용 설정집과 작가용 비밀 장부 분리
- 설정 상세도 최종 재평가

## 7. 과잉 설정 방지

상세화는 모든 것을 무한히 채우는 작업이 아니다.

- 작품에서 반복 사용 가능한 분야는 깊게
- 단발성 배경은 대표 사례와 생성 규칙만
- 모든 현·촌·초식·약재를 나열하지 않음
- 한 세력의 이름보다 재정·사승·지역·약점·갈등을 우선
- 도감 항목은 서로 다른 기능과 문화가 있을 때만 추가

## 8. 현재 결정

- 설정집 v2.0의 핵심 정본은 폐기하지 않는다.
- `설정집 전체 완료` 상태는 철회한다.
- 상태를 `core-canon / detail-expansion-active`로 교정한다.
- 플롯 Human Edit와 Outline 진행은 상세 설정집 완료 뒤로 보류한다.
- 다음 작업은 S8 세력·문파·세가 상세화다.
