## 2. 용어 감사

최신 사용자 결정에 따라 이후 원고는 다음 용어를 사용한다.

- 진원표국
- 검표수
- 인계표사
- 표사
- 표두
- 금의위 남방 회수별조

`진원행·검화수·인계호송인·호송인·행관·남방 회수차견`은 폐기된 검토안이다.

지명은 실제 태주 해문위와의 혼동을 피하기 위해 다음 교정을 유지한다.

- 쇄조도
- 쇄조도 이관 참변
- 쇄조 밀약
- 쇄조합증

## 3. 신병 배치 감사

- 직접 중심 신병: D3·D4·D6·D7·D8
- 간접 증언 신병: D1·D2·D5
- 여덟 신병의 물리적 집합: 없음
- 수집 퀘스트 위험: 통과
- D3 주인공 자동 소유 위험: 통과 — 청계 지역 공동 보관
- D7 진품 하나 선택 위험: 통과 — 이중 계승 유지

## 4. 주인공 감사

심소연은:

- 혈통 선택자가 아니다.
- 모든 신병의 사용자가 아니다.
- 절정고수로 성장하지 않는다.
- 기록·인계·현장 상태·책임 구조를 설계하는 능력으로 성장한다.
- 과거 기록 조작을 중반에 공개한다.
- 최종부에서 징계·배상·직무 제한을 수용한다.

판정: 영웅 특권과 도덕적 면죄 위험 통과.

## 5. 적대 구조 감사

- 주계평: 청계 방화·매수·밀무 직접 책임
- 사문경: 비밀 회수·구금·자료 제외·중앙화 공범 책임
- 국가 강경파: 불법 구금·일괄 압수
- 전면 파기파: 현재 보호 명분으로 권리 증거까지 파괴할 위험

사문경이 모르는 정보:

- D6 사인 구조
- 전승궁 실제 반응
- 위철산의 제한 파기 전환
- 소연 일행의 최종 분산 인계 경로

판정: 전지적 흑막 위험 통과.

## 7. 단서·가짜 가설 감사

주요 초기 가설과 재해석:

1. D3는 복운호 화물이다 → 청계 설비로 수십 년 재사용
2. 한 명령이 위조됐다 → 정당한 두 명령이 충돌
3. 쇄조도 사망은 독·자해다 → 봉쇄·연기·압박·익사·구조 지연
4. 전승궁은 가짜다 → 기능·책임 계승
5. 팔대신병은 모두 모여야 한다 → 분산 교차 증언 가능
6. 고학선은 방화범이다 → 봉인 조작·침묵 책임은 있으나 방화범이 아님
7. 사문경이 모든 사건을 지시했다 → 독립 행위자와 제도 압력이 결합

판정: 새 문서 한 장으로 모든 것을 설명하는 위험 통과.

## 8. 장면 기능 감사

- 한 장면의 새 핵심 개념 최대 두 개 원칙 유지
- 전투는 확보·구조·퇴로·증인 보호·문서 보전 목표를 가짐
- 적 전멸형 전투 없음
- 합증은 토론만이 아니라 보관·봉쇄·화재·인계 행동과 결합
- 각 부 종료 시 신병·관계·부상·증거 상태 변화 존재

## 9. 집필 중 비차단 확인 항목

1. 실제 태주 수로·포구 거리
2. 절강에서 금릉까지 이동 기간
3. 겨울 중원 경계 이동의 기후 세부
4. 장강 상류·서남 대상로의 구체 지명
5. 금의위 남방 회수별조의 허구 조직 설명 밀도
6. 표국 제도의 장르적 용어 사용과 실제 명대 사회의 조화
7. 쇄조합증 문서·봉인·인장 명칭

이 항목은 각 장 초고 검증 때 확정하며 전체 초고를 막지 않는다.
