# WP-207 전문가 검증 — 팔대신병 정의와 왜 여덟인가

상태: `review-complete`
총괄: PM 오케스트레이터

## 1. 주제·콘셉트 검토

통합안은 다음 질문을 만든다.

- 진실을 보존한 원본과 현재 역할을 수행하는 계승품 중 무엇이 정통인가?
- 소유자는 과거의 책임까지 물려받는가?
- 역사적 진실을 공개할 권리는 물건을 가진 사람에게 있는가?

판정: 작품 핵심 주제와 강하게 연결.

## 2. 미스터리 검토

장점:
- 여덟 기억이 같은 사건을 다르게 보존
- 원본·계승품·위조품의 증거 차이
- 현재 여덟 목록 자체가 미스터리

제한:
- 현재 공석 또는 이중 주장은 최대 두 자리
- 모든 신병이 정체를 숨기는 구조 금지
- 독자는 부별로 최소 하나의 진위와 기능을 확정적으로 알게 됨

판정: 통과.

## 3. 무협·병기 검토

- 최초 여덟은 모두 실제 전투에서 사용 가능한 병기 또는 무장 도구다.
- 순수 문서·두루마리·보패를 신병 자리에 넣지 않는다.
- 살상용이 아닌 방호·포박·의료·지휘 성격의 무장 도구는 가능하다.
- 병기의 개별 형식은 원사건의 역할보다 사용자의 삶·지역·전법에서 나온다.

판정: 통과.

## 4. 역사·정치 검토

- 숫자 여덟은 우주론적 필연이 아니라 원사건 뒤 실제로 협약에 남은 여덟 당사자의 수다.
- 협약은 국가 전체의 공식 헌법이 아니라 비밀에 가까운 제한적 합의다.
- 황실·무림맹 어느 한쪽이 모든 유물을 통제하지 못하도록 소유와 보관이 분산됐다.
- 왕조 교체·숙청·전쟁·해상 이동·가문 몰락으로 원본 일부가 사라질 수 있다.

판정: 통과.

## 5. 계승 검토

후계 신병 지위의 후보 증거:
- 원본의 실질 재료 일부
- 원래 보관 계보의 기록
- 역할과 기능의 연속성
- 독립된 다른 신병의 교차 반응
- 원사건 관련 문서·증언
- 장기간 공개된 사용과 책임 이행

최소 세 종류가 필요하며, 그중 하나는 물증 또는 다른 신병과의 반응이어야 한다.

후계품은:
- 원본의 모든 기억을 받지 않음
- 새 사용자와 사건의 흔적을 추가
- 기능과 대가가 변형될 수 있음

판정: 통과.

## 6. 악마의 변호인

### 위험 A — 결국 여덟 역할 수집 퀘스트

조치:
- 역할명은 내부 감사용이며 본편 전면에 게임 슬롯처럼 제시하지 않음
- 세력은 신병 자체보다 기록·증인·권리·파기 여부를 추적할 수 있음

### 위험 B — 협약이 현대 제도처럼 보임

조치:
- 보편적 권력분립이 아니라 서로 불신한 생존자들의 임시 책임 분산
- 문서보다 인적 보증·유품·맹세·인질·분산 보관이 중요

### 위험 C — 계승품으로 무엇이든 신병화

조치:
- 인정 조건 최소 세 종류
- 공석과 이중 주장 수 제한
- PM과 정합성 감사 승인 없이 새 계승품 추가 금지

### 위험 D — 팔대신병 모두가 원사건 영상기

조치:
- 각 병기는 감각·동작·감정 보존 비율이 다름
- 일부는 기억보다 기능·반응·결손이 중심
