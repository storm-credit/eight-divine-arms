# 이동 타당성 원칙

- 가장 빠른 길을 항상 사용하지 않는다. 관부 검문·부상·증거 은닉·계절·호송 계약이 경로를 바꾼다.
- 절강→금릉은 주로 수로와 연해·운하 연결을 활용한다.
- 금릉→중원 경계는 육로·수로 혼합이다.
- 중원 경계→서남은 장강 수계와 약재로를 이용하며 가장 긴 이동 구간이다.
- 서남→해문도는 장강 하행과 연해 선박을 나눠 사용한다.
- 증거·신병·증인은 같은 배나 수레에 모두 싣지 않는다.

# 계절·부상 의무

- 여름 연해: 습기·태풍·부패·화재·해상 봉쇄
- 가을 강남: 수운·상업·창고·세무 활동
- 겨울 중원 이동: 속도 저하·숙박·식량·부상 악화
- 봄 서남: 약재 채집·우기 전 산길·감염 위험
- 다음 여름 귀환: 폭풍 위험이 최종 봉쇄·구조에 영향을 줌

소연의 어깨·평형 후유증은:

- 1부 D3 사용 뒤 악화
- 2부 선상 이동에서 반복
- 3부 치료와 휴식으로 부분 회복
- 4부 겨울·구출전에서 재발
- 5부 장거리 호송에서 다른 인물에게 전투 역할을 넘기게 함
- 6부 D3를 장시간 휘두르는 결투가 아니라 구조 장치로 제한 사용
